Before Launching the usecase

Before launching this usecase,we have to perform following checks :

  • XPGClientService is started
  • Database is started
  • 3DEXPERIENCE Platform (V6 server) is running
  • 3DEXPERIENCE Gateway is running
  • XPDM gateway is running

We can retrieve the information about above mentioned components from Gateway's Administration Console. If every point mentioned on console is in green state then we can say gateway is successfully up.If everything is working correctly then we can launch the application and invoke this usecase.

XPDM Gateway Administration Console is shown in Fig.2. Left side of Fig.2 has status table. This table consist of components and it's state.This table contains status informaton of Database, File Server and Configured Sites.

Area present in right side of status table displays information about selected element.In Fig.2 welcome message is displayed here.

Fig.2. XPDM Gateway Administration Console

3DEXPERIENCE Gateway Administration Console is shown in Fig.3. Left side of Fig.3 has status table.This table consist of components and it's state.It has status information about components : Database, File Server, 3DEXPERIENCE Platform, XPGClientServices and Configured Sites.

Area present in right side of status table displays information about selected element.In Fig.3 welcome message is displayed here.

Fig.3. 3DEXPERIENCE Gateway Administration Console

Both Gateway Administration console contain "Event Activity Dashboard", this contains information about events at gateway.Here events are listed depending on state of processing. This is shown in Fig.4.

This dashboard is present on console page below status table. Please refer to Product Documentation for details.

Fig.4.Event Activity Dashboard

User have to launch the MyX-CAD application and MyX-PDM. Refer to How to launch MyX-CAD?[1] and How to launch MyX-PDM?[2] for details.

After successful launching both applications, user need to be connected with MyX-PDM. Refer to How to use MyX-PDM ?[3]. To execute different scenario related to X-PDM Please refer to X-PDM Addins[4].
