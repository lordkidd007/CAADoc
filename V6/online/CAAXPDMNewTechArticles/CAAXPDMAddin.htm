X-PDM Addins
Use Case

Abstract

This usecase gives you basic information about X-PDM addins illustrated by using Sample CAA data. This addins are used in integrating MyX-PDM with the 3DEXPERIENCE  Platform which allows to exchange the information between MyX-PDM and 3DEXPERIENCE  Platform.

Here it is important to understand this application is only sample application, user can also create his own PDM or use existing PDM or create his own application and replace this application.

Basic Objective

The basic objective of this article is that user will be able to understand how X-PDM addins work in MyX-CAD application.

MyX-PDM Menu

X-PDM addins are available through MyX-CAD Application. Launch MyX-CAD application from MyX-CAD Launcher. Refer to How to Launch a MyX-CAD [1] for details.

 These addins are integrated under MyX-PDM menu.This is shown in [Fig.1].

Fig.1 : My-XPDM Menu

It is necessary to start MyX-PDM before performing any operation. Start MyX-PDM by using MyX-PDM Launcher. Refer to How to Launch a MyX-PDM [2] for details.

As we know, before performing any operation it is necessary to connect to MyX-PDM. Once user will be connected successfully following window will appear. For significance of different commands refer to  How to Use MyX-PDM? [3]. Once user connects to MyX-PDM,addin menu will be visible.This is shown in Fig.2

This article explains about "Send To 3DEXPERIENCE Platform " menu. (Note : Processing of events at XPDM which are received from 3DEXPERIENCE Platform will be done automatically)

This menu is treated as X-PDM addins. By using this menus user will be able to exchange the data between 3DEXPERIENCE Platform and MyX-PDM.

Fig.2: My-XPDM Menu

X-PDM Addins

When this addins get loaded user will be able to perform this two functionalities :

Send to 3DEXPERIENCE Platform:

This functionality allows the user to save the working file in V6 database. V6 is the remote server where we can store the data. This data can be retrieved by any CAD application and used for further processing.

When we click on this command following window appears [Fig.3]. Send to 3DEXPERIENCE Platform window has following components :

Name of destination directory : Enter the name of directory where you want to save the file in MyX-PDM database. This directory will be treated as local shadow directory.It needs to be different as that of XPDM shadow directory.

XPDM Gateway URL : URL of the XPDM gateway which is connected with 3DEXPERIENCE gateway.

XPDM Shadow Directory : Path of XPDM Gateways shadow directory, while sending the data to V6 this directory will be treated as source. User can select directory by using "Browse" button.

Organization : Name of organization in which you are working

Project : Name of Project in which you are working

Fig.3 : Send To 3DEXPERIENCE Platform

Enter all the parameters and click on "Send to 3DEXPERIENCE Platform" button. Currently opened file along with some supported files will be saved to V6 database.

On successful sending it shows following information window  Fig.4.

Fig.3 : Success

For details about this functionality refer to Send Sample to 3DEXPERIENCE  Platform [4].

Received from 3DEXPERIENCE  Platform:

This functionality allows the user to receive the V6 data sent by any client application and gives the notification about this in MyX-CAD application.

When user click on "Receive From 3DEXPERIENCE  Platform" following window will appear [Fig.5].

Name of Directory for receive : Enter the name of directory where you want to save the file after receiving it from V6. This directory is folder present at local machine.

Remainig components are same as mentioned in "Send To 3DEXPERIENCE  Platform" command.

Fig.5 : Receive From 3DEXPERIENCE  Platform

Enter all the parameters and click on "Receive from 3DEXPERIENCE  Platform" button,event data will be retrieved from XPDM gateway. This data consist of XPDMXML file and attachment files, will be stored in mentioned directory. Once it is succeesfully received following window will appear Fig.6

Fig.6 : Success

For details about this functionality refer to Receive From 3DEXPERIENCE  Platform[5] for details.
