Receive Assembly Data from 3DEXPERIECNE Platform
Use-Case

Abstract 

This article illustrates about processing of remote events in MyX-PDM.Here, 3DEXPERIENCE Platform is exporting assembly data. We are processing this event by using event exchange services.  This scenario is called as export scenario.

When assembly data is exported processing steps of remote event are same as that of component data export. This steps are detailed in Receive Sample from 3DEXPERIECNE Platform.
