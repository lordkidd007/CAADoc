This use case can be divided in seven steps:

  1. Retrieves the current Editor
  2. Retrieves the Simulation Reference from the Editor
  3. Retrieves the Simulated Model
  4. Retrieving the Specifications Collection
  5. Retrieving the Scenario Representation under the Scenarios Category
  6. Retrieving the Root Feature of the Rep Scenario
  7. Retrieving the Results Collection
  8. Retrieving the Result Representation under the Results Category
  9. Retrieving the Root Feature of the Rep Result

  1. Retrieves the current Editor

    ... 
            Dim oSimulationEditor As Editor
            Set oSimulationEditor = CATIA.ActiveEditor
...

  2. Retrieves the Simulation Reference from the Editor

    ... 
             Dim oSimulationRoot As SimulationReference
             Set oSimulationRoot = oSimulationEditor.ActiveObject
...

  3. Retrieves the Simulated Model

    In this step, we retrieve the Simulated Model.

    ...
            Dim oPLMModel  As AnyObject
            Set oPLMModel = oSimulationRoot.Model
...

  4. 4- Retrieving the Specifications Collection

    In this step, we retrieve the Simulation Category Scenario.

    ...					
            Dim oSpecifications  As  SimulationSpecifications
            Set oSpecifications = oSimulationRoot.Specifications
...

  5. 5- Retrieving the Scenario Representation under the Scenarios Category

    ...
             Dim oSimulationRepSpec  As SimulationRepresentation
             Set oSimulationRepSpec = oSpecifications.Item(1)
...

  6. 6- Retrieving the Root Feature of the Rep Scenario (not implemented)

    ...			
             Dim oRoot  As AnyObject 
             Set oRoot = oSimulationRepSpec.Root 
...

     

  7. 7- Retrieving the Results Collection

    ...			
            Dim oResults   As  SimulationResults
            Set oResults  = oSimulationRoot.Results
...

  8. 8- Retrieving the Result Representation under the Results Category

    ...
            Dim oSimulationRepRes  As SimulationRepresentation
            Set oSimulationRepRes = oResults.Item(1)	
...

  9. 9- Retrieving the Root Feature of the Rep Result (not implemented)

    ...
             Dim oRoot  As AnyObject 
             Set oRoot = oSimulationRepRes.Root 	
...
