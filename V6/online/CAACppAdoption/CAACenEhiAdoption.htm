Electrical Harness Installation Changes
Technical Article

Abstract

This article shows how to migrate easily your V5 code to V6. Part One describes the mapping between the V5 APIs and the V6 APIs. The mapped V6 APIs are listed directly in the table if the mapping is one-to-one. If the mapping is more complex, a link is available to Part Two which describes the mapping methodology.

Introduction

The naming in the ElecHarnessItf APIs has been entirely reworked. Here is a mapping of harness installation vocabulary between V5 and V6:

V5 Naming V6 Naming
Branchable Branch
Bundle Segment Segment
Bundle Segment Extremity Segment End
Geometrical Bundle Geometrical Harness
Multibranchable Harness Bundle

Part Two: Mappings Methodologies

This chapter is aimed to explain how to migrate V5 APIS whose V6 mapping is not as simple as one-to-one.

Compute a Harness Bundle

V5

...
CATIEhiGeoBundle *pIEhiGBN = NULL;
rc = pIUnkGBN->QueryInterface(IID_CATIEhiGeoBundle, (void **)&pIEhiGBN);
...
CATIProduct *pIPrdGBN = NULL;
rc = pIUnkGBN->QueryInterface(IID_CATIProduct, (void **)&pIPrdGBN);
...
CATListValCATBaseUnknown_var* pListChildren = pIPrdGBN->GetAllChildren();
...
CATIProduct_var spChild = NULL_var;
for (int mbns=1; mbns<=pListChildren->Size(); mbns++)
{
    spChild = (*pListChildren)[mbns];
    CATIEhiMultiBranchable *pIEhiMultiBranchable = NULL;
    rc = spChild->QueryInterface(IID_CATIEhiMultiBranchable, (void **)&pIEhiMultiBranchable);
    ...
    pIEhiGBN->ComputeMultiBranchable(pIEhiMultiBranchable);
}
...

V6

...
CATIEhiGeoHarness *pIEhiGBN = NULL;
rc = pIUnkGBN->QueryInterface(IID_CATIEhiGeoHarness, (void **)&pIEhiGBN);
...
CATIPLMNavOccurrence *pIPrdGBN = NULL;
rc = pIUnkGBN->QueryInterface(IID_CATIPLMNavOccurrence, (void **)&pIPrdGBN);
...
CATListPtrCATIPLMNavOccurrence ListChildren = pIPrdGBN->GetAllChildren();
...
CATIProduct_var spChild = NULL_var;
for (int mbns=1; mbns<=pListChildren.Size(); mbns++)
{
    spChild = ListChildren[mbns];
    CATIEhiHarnessBundle *pIEhiHarnessBundle = NULL;
    rc = spChild->QueryInterface(IID_CATIEhiHarnessBundle, (void **)&pIEhiHarnessBundle);
    ...
    // Update the harness bundle
}
...

List Segments Under a Harness Bundle

V5

...
CATIEhiMultiBranchable *pIEhiMBNS = NULL;
rc = pIUnkMBNS->QueryInterface(IID_CATIEhiMultiBranchable, (void **)&pIEhiMBNS);
...
CATListValCATBaseUnknown_var *pListBundleSegmentsUnderMBNS = NULL;
rc = pIEhiMBNS->ListBundleSegments(&pListBundleSegmentsUnderMBNS);
...
int NbBundleSegmentsUnderMBNS = pListBundleSegmentsUnderMBNS->Size();
...

V6

...
CATIEhiHarnessBundle *pIEhiMBNS = NULL;
rc = pIUnkMBNS->QueryInterface(IID_CATIEhiHarnessBundle, (void **)&pIEhiMBNS);
...
// retrieve the list of branches under the harness bundle
CATListValCATBaseUnknown_var *pListBranchesUnderMBNS = NULL;
rc = pIEhiMBNS->ListBranches(&pListBranchesUnderMBNS);
...
CATListValCATBaseUnknown_var *pListSegmentsUnderMBNS = NULL;
pListSegmentsUnderMBNS = new CATListValCATBaseUnknown_var;
CATBaseUnknown_var spUnkBranch;
int NbBranchesUnderMBNS = pListBranchesUnderMBNS->Size();
for (int branch=1; branch<=NbBranchesUnderMBNS; branch++)
{
    // retrieve the segments under each branch
    CATIEhiBranch *pIEhiBranch = NULL;
    spUnkBranch = (*pListBranchesUnderMBNS)[branch];
    rc = spUnkBranch->QueryInterface(IID_CATIEhiBranch, (void **)&pIEhiBranch);
    ...
    CATListValCATBaseUnknown_var *pListSegmentsUnderBranch = NULL;
    rc = pIEhiBranch->ListSegments(&pListSegmentsUnderBranch);
    ...
    rc = pListSegmentsUnderMBNS->Append(pListSegmentsUnderBranch);
}
...
int NbSegmentsUnderMBNS = pListSegmentsUnderMBNS->Size();
...
