Framework Mapping

This table summarizes the mapping between the V5 frameworks and the V6 frameworks. The columns are as follows:

  • Frameworks: lists the V5 frameworks
  • Available: indicates whether the frameworks still expose classes, interfaces, global functions, etc. with V6
  • Moved Headers:
    • Mapping: gives a link to a file listing headers moved to other frameworks, along with the names of the modules to add in the Imakefile.mk file of your modules
    • Frameworks: lists the frameworks where the moved headers are located
  • Replaced APIs: gives a link to a file listing replaced classes, interfaces, methods, or global functions and how to replace them in your applications.
