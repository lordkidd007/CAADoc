Generative Shape Design Changes
Technical Article

Abstract

Although many APIs remain the same in Generative Shape Design, this article provides information for the few modified APIs:

  • CATIGSMBoundary

CATIGSMBoundary

Continues to be supported for former V5 Boundary feature (GetSupport only useful for BorderEdge).

In V6, use following algorithm:

  1. Look if GetSupport returns an object
  2. If not, need to look for Method of CATIMfBRep (Boundary object adheres to CATIMfBRep).
