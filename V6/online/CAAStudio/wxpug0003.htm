3DS - Tasks List

The 3DS Tasks view shows the task currently being executed, the tasks waiting for execution, and the 20 last tasks completed.
Before you begin:

Open the appropriate workspace.

  1. On the VIEW menu, point to 3DS Views, and click 3DS Tasks List.

    The 3DS - Tasks List view opens.

This view shows:

  • In To do tasks, the current task being executed, and the possible tasks waiting for execution.
  • In History, the last 20 tasks completed. For each task, you can see:
    • if the tasks succeeded, or if it failed.
    • The task description.
    • The task command.
    • The task elapsed time.

You can access the trace log for each completed task. You can also restart any of them.

  • To see the trace log of a task:

    1. In the 3DS - Tasks List view, right-click a task.

    2. In the contextual menu, click Show trace view.

      A pop-up window opens, displaying the trace log.

    3. Click Close to close the pop-up window.

  • To restart a task:

    1. In the 3DS - Tasks List view, right-click a task.

    2. In the contextual menu, click Restart.

      The task restarts.
