Adding a Forward Declaration

The Add forward Declaration command enables you to add a forward declaration to the selected class or interface in the currently opened header or source file.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Open a header or a source file from the Solution Explorer.

You can insert a forward declaration of a class or interface already used in the edited file, or by giving the name of the class or interface.

  • To insert a forward declaration of a class or interface already used in the edited file:

    1. In the currently edited header or source file, select a class or interface name.

    2. On the EDIT menu, point to 3DS Edit, and click Add forward Declaration.

      A forward declaration of the selected class or interface is added at the top of the file, among the other forward declarations.

  • To insert a forward declaration of a class or interface by giving its name:

    1. On the EDIT menu, point to 3DS Edit, and click Add forward Declaration.

      A dialog box opens, prompting you to enter a class or interface name.

    2. Type the name of class or interface, such as CATBaseUnknown.h.

      A forward declaration to the typed class or interface is added at the top of the file, among the other forward declarations.
