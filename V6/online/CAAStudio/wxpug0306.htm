Generating Commands

This task shows you how to generate the source files of a C++ command class.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.

  1. In the Solution Explorer, right click a module.

  2. In the contextual menu, point to Add, and click New Item.

    The Add New Item dialog box opens.

  3. In Installed, click the  arrow in front of 3DS Templates to expand its contents, if not yet expanded.

  4. Click C++.

    The C++ installed templates are displayed in the central pane.

  5. In the central pane, click 3DS Command.

  6. In the Name box, type the name you want to assign to the command class. Refer to C++ Naming Rules.

    Warning: This name is also used to create the header and source files. Bear in mind that:
    • All the characters are allowed except those listed below: 
      * ? \ / < > " ' | :
    • With Windows, full path names must not exceed 260 characters.

  7. Click Add.

    The 3DS - New Command dialog box opens.

  8. If the command is intended to be exclusive, check Is exclusive.

    An exclusive command does not coexist with other commands and requests the command stack to be cleaned before being loaded. Otherwise, when this command is loaded, the active command is deactivated and kept in the command stack, and reactivated in its current state when the loaded command completes. 

  9. If the command is intended to be called with arguments, check With arguments.

  10. Click Finish.

    The wizard creates the command class header file where you have requested it to be, and the cpp file in the module src folder. Both files are opened in the Code Editor window.
