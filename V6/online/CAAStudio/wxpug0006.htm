Run Time Prompt

The Run Time Prompt command opens a command prompt set up with the workspace run time environment.
Before you begin:

Open the appropriate workspace.

  1. On the WINDOW menu, point to 3DS Windows, and click Run Time Prompt.

    A command prompt opens with the workspace root folder as the current one.

    The workspace run time environment is set up, as if you had typed the mkrun command with appropriate options to match your workspace run time environment.

    This command prompt is dedicated to run time commands only. Applications can be started directly from this command prompt.

    Warnings:

    • Applications will not run correctly if the run time environment itself is not correctly set, that is, if the task Set runtime environment automatically started when opening the workspace has failed, since in this case runtime variables are not retrieved. See the 3DS - Tasks List view to make sure this task run successfully.

    • Compilation tools must not be launched from this window: it will fail because the environment is not set for build operations. See Build Time Prompt to start compilation tools.

    • This command is not available is the workspace is opened with the option Disable run time environment computing. See Opening a Workspace.
