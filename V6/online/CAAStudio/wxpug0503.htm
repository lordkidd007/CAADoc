Refreshing the Run Time View

Once you have built or rebuilt your code, you need to refresh the run time view. This applies to the full workspace.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Update the framework IdentityCard file.
  • Create and build the application.

Before being able to run the written application, and after you have built it, you need to refresh its run time view with the new built objects (exe or dll) and the newly created or modified resources.

  1. In the 3DS - Workspace Explorer, right click the workspace name.

  2. Click Refresh Runtime View.

The Output windows is refreshed with the mkRtv log.

If errors occurred, of if warnings or messages are output, they are displayed in the Error List window. Click Error List to display it.

To correct the errors, double click the error row. The file in error opens.
