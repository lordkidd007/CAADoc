Swap Source Header

The Swap Source Header command command enables you to open the header file corresponding to a given opened cpp source file, and the reverse.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Open a source file from the Solution Explorer.
  • To use keyboard shortcuts, such as CTRL+Q, these shortcuts must have been enabled. See Accessing and Customizing Settings.

  1. On the EDIT menu, point to 3DS Navigate, and click Swap Source Header, or press CTRL+Q.

    The header file corresponding to this header file is displayed.

  2. From this header file, on the Edit menu, point to 3DS Navigate, and click Swap Source Header.

    The original source file is displayed again.
