Source Generation Overview

The source generation wizards enable you to create the source files of objects matching the 3DEXPERIENCE patterns, such as C++ classes, Object Modeler interfaces and their implementation in components.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.

The source generation wizards are available from the Solution Explorer contextual menu.

  1. In the Solution Explorer, right click a module.

  2. In the contextual menu, point to Add, and click New Item.

    The Add New Item dialog box opens.

  3. In Installed, click the  arrow in front of 3DS Templates to expand its contents, if not yet expanded.

  4. Click C++.

    The C++ installed templates are displayed in the Templates pane.

Eight templates are available:

  • In C++:
    Template Description
    3DS Command A C++ class deriving from the CATCommand class.
    CATStateCommand A C++ state command deriving from the CATStateCommand class.
    Class A C++ class.
    Interface An Object Modeler interface deriving from CATBaseUnknown.
    Component An Object Modeler component deriving from CATBaseUnknown to implement an interface.
    3DS Adapter An Object Modeler adapter.
  • In Tests:
    Template Description
    Xml test case An XML test case.

The template in IDD Templates is available if you have installed and have a license of Native Apps Dialog Designer.

Notes: When creating source files from these templates, bear in mind that:
  • All the characters are allowed except those listed below: 
    * ? \ / < > " ' | :
  • With Windows, full path names must not exceed 260 characters.
