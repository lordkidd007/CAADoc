Shortcuts

Native Apps Interactive Development Environment lets you use shortcuts to access the following commands. See Accessing and Customizing Settings to enable them.
Shortcut Command Command Description
Ctrl+T Go To Header. See Go To Header.
Ctrl+Q Swap Source Header. See Swap Source Header.
F7 Link All. See Compiling.
? In a dialog box, contextually opens the Microsoft Help Viewer with this dialog box documentation. NA
Alt+F1 Open the 3DS Help Viewer. In a source code editor with a class or method selected, contextually opens the class or method documentation in the 3DS Help Viewer. NA
Ctrl+Shift+X Explore from Object. See Explore from Object.
Ctrl+Shift+H In the source editor, adds an include line. NA
Ctrl+Shift+V In the source editor, cycles through the Clipboard Ring tab, and pastes the current displayed item when you release the keys. NA
