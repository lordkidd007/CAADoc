Intellisense

Intellisense is Microsoft Visual Studio feature that provides code development aids to edit source code quickly and accurately.

Before you begin:
  • Make sure that Intellisense is enabled (see the Microsoft documentation).
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Build the workspace.

Intellisense applied to Native Apps Interactive Development Environment provides:

  1. Quick Info: a pop-up text box displays a description of an identifier, such as a class name or method, when you hover the mouse above the identifier.

    For example, hovering the mouse above the StreamData method displays a tooltip showing the method signature.

  2. Statement completion: a process that helps you finish a language keyword or a class, method, or function name that you have started typing.

    For example, typing a method name opens a window that highlights the first possible method of the list with respect to the characters typed. Pressing Enter completes the method name in the editor with the one highlighted.

    Pressing CTRL+SPACEBAR completes the method name if there is no ambiguity, or displays the window as above.

  3. Parameter Info Tooltips: a window that display parameter information about the method being typed. A method tip typically opens whenever you type a parameter list token, such as an open parenthesis.

    For example, typing the opening parenthesis following a method name displays a tooltip showing the method signature, and highlighting the argument to type.

Warning: Intellisense applies to the workspace prerequisites, and to the part of the workspace that is projected to the Solution Explorer.

To generate Intellisense:

  1. Compile your workspace. In the 3DS - Workspace Explorer, right click the workspace name, point to Build, and click Build with mkmk.

    The 3DS - Build options dialog box opens.

  2. Check Force mkdata update (-u) to build the whole workspace.

  3. Click Build.

  4. On the VIEW menu, point to 3DS Views, and click 3DS Task List.

    The 3DS - Tasks List view opens. Make sure that the workspace is built successfully.

  5. In the 3DS Workspace Explorer, right click Generate Intellisense.

    Intellisense is generated for your workspace and its prerequisites.

Important: If you add prerequisite frameworks, or if you add new classes in your frameworks, run again the Intellisense generation as described above from step 1.
Tip: If Intellisense has worked correctly and becomes unavailable, see When IntelliSense Is Unavailable from MSDN.
