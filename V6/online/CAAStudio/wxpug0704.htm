Using the 3DS Object Browser

This article describe the usual 3DS Object Browser scenarios.

  • Listing the Interfaces Implemented by a Given Feature
  • Listing the Features Implementing a Given Interface
  • Listing All the Features Implementing Interfaces in a Given Library
Before you begin:
  • Open Visual Studio.
  • Open a workspace with its prerequisites set.
  • Open the 3DS Object Browser from the Help menu.

Listing the Interfaces Implemented by a Given Feature

  1. In the text box, type the feature name.

  2. Either click or press Enter.

    The table in the left pane is filtered to display only rows containing this feature, or features the name of which include this feature name, in the first cell.

  3. Double click this feature in any rows.

    This feature is displayed along with the interfaces it implements, the frameworks containing the dictionary files, and the libraries containing the implementation run time code, either as a tree or as a table.

Listing the Features Implementing a Given Interface

  1. In the text box, type the interface name.

  2. Either click or press Enter.

    The table in the left pane is filtered to display only rows containing this interface, or interfaces the name of which include this interface name, in the second cell.

  3. Double click this interface in any rows.

    This interface is displayed along with the features implementing it, the frameworks containing the dictionary files, and the libraries containing the implementation run time code, either as a tree or as a table.

  4. Click   to display the tree view, or click to display the table view.

Listing All the Features Implementing Interfaces in a Given Library

  1. In the text box, type the library name.

  2. Either click or press Enter.

    The table in the left pane is filtered to display only rows containing this library, or libraries the name of which include this library name, in the third cell.

  3. Double click this library in any rows.

    This library is displayed along with the features implementing it, the interfaces they implement in this library, and the frameworks containing the dictionary files, either as a tree or as a table.
