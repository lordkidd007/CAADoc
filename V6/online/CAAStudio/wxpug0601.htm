Checking Sources

Checking sources implies that you have a license for Native Apps Source Checker.
Before you begin:
  • Make sure that you have a license for Native Apps Source Checker.
  • You can run Native Apps Source Checker
    • Against a whole workspace, framework, or module.
    • Against a single file.
  • Open a workspace with at least a framework and a module containing C++ sources.

To run Native Apps Source Checker against:

  • A whole workspace, framework, or module:

    1. In the 3DS - Workspace Explorer, right click the workspace name, a framework, or a module, and click Check Sources.

      The Native Apps Source Checker mkscc command is run against the workspace, framework, or module.

    2. In the Output window, consult the command traces.

  • A single source file:

    1. In the Solution Explorer, double click a file to open it in a window.

    2. On the EDIT menu, point to 3DS Edit, and click Check Source.

      The Native Apps Source Checker mkscc command is run against the current opened file.

    3. In the Output window, consult the command traces.
