Managing Prerequisites

The prerequisite frameworks of your workspace are the frameworks from which you use classes, interfaces, or global functions.

  • Adding a Prerequisite Workspace
  • Removing a Prerequisite Workspace
  • Changing Prerequisite Concatenation Order
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Update the framework IdentityCard file.

Adding a Prerequisite Workspace

  1. In the 3DS - Workspace Explorer, right click the workspace name, and click Prerequisites.

    The 3DS - Manage prerequisites dialog box opens.

  2. Click .

    A field appears prompting new to select or type a folder.

  3. Click .

    The Browse for Folder dialog box opens.

  4. Locate the workspace prerequisite root folder, that is, the folder where you installed the brand CD ROM (usually C:\Program Files\Dassault Systemes\B417), and click it.

  5. If you want to copy the prerequisite framework from this workspace, check Copy prerequisite file.

    The Target directory box becomes active.

  6. In this case, click Browse to select a folder when the frameworks will be copied, or type the path name of this folder.

  7. Click OK.

    The declared workspace is added to you workspace prerequisites, possibly copied to the folder you have selected, and the identity cards of your frameworks are rebuilt, taking this workspace into account. The start program defined in is also copied to your workspace. See Accessing and Customizing Settings.

    Each class, interface, or global function you use from these prerequisite workspaces are retrieved from the first workspace in which it is found in the order they appear in the dialog box.

Removing a Prerequisite Workspace

  1. In the 3DS - Workspace Explorer, right click the workspace name, and click Prerequisites.

    The 3DS - Manage prerequisites dialog box opens.

  2. Click the root folder of the workspace to remove from your workspace prerequisites.

  3. Click .

    The workspace is removed from your workspace prerequisites.

Changing Prerequisite Concatenation Order

  1. In the 3DS - Workspace Explorer, right click the workspace name, and click Prerequisites.

    The 3DS - Manage prerequisites dialog box opens.

  2. Click the root folder of the workspace to move.

  3. Click to move it upwards, or to move it downwards.
