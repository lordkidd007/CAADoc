This section deals with Native Apps Interactive Development Environment, the Visual Studio 2010 customization with design tools to create V6 applications. It brings workspace, framework, and module management views, and commands in addition to those of Visual Studio solutions and projects, enabling you to see and handle actual V6 development workspaces as such within Visual Studio.

It also lets you use the Native Apps Builder to build your V6 applications, and to the Tool Configuration Key to manage several API releases.

Providing they are installed and licensed, it also gives access to:

  • The Native Apps Dialog Designer to design and implement your dialog windows.
  • The Native Apps Source Checker to run C++ source checks.
  • The Native Apps Unit Tester to create and run test objects.
