Adding an Include

The Add Include command enables you to add a #include preprocessing directive to the header file of the selected class or interface in the currently opened header or source file.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Open a header or a source file from the Solution Explorer.
  • To use keyboard shortcuts, such as CTRL+Shift+H, these shortcuts must have been enabled. See Accessing and Customizing Settings.

You can insert an include directive of the header file of a class or interface already used in the edited file, or of a header file by giving its name, or the name of the class or interface it defines.

  • To insert an include directive of the header file of a class or interface already used in the edited file:

    1. In the currently edited header or source file, select a class or interface name.

    2. On the EDIT menu, point to 3DS Edit, and click Add Include, or press CTRL+Shift+H.

      A #include preprocessing directive to the header file of the selected class or interface is added at the top of the file, among the other #include preprocessing directives.

  • To insert an include directive of a header file by giving its name, or the name of the class or interface it defines:

    1. On the EDIT menu, point to 3DS Edit, and click Add Include, or press CTRL+Shift+H.

      A dialog box opens, prompting you to enter a header name.

    2. Type the name of header file, such as CATBaseUnknown.h, or the class or interface of which you want to include the header file, such as CATBaseUnknown.

      A #include preprocessing directive of the typed file, or of the one of the typed class or interface is added at the top of the file, among the other #include preprocessing directives.
