Go To Header

The Go To Header command enables you to retrieve and display the header file where an identifier selected in a source file is defined.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Open a source file from the Solution Explorer.
  • To use keyboard shortcuts, such as CTRL+T, these shortcuts must have been enabled. See Accessing and Customizing Settings.

When selecting a word inside the editor and calling this functionality (Edit menu or CTRL+T), the tool will try to find the corresponding header file. If the header is not in the workspace or the prerequisites, the command will fail. If the word does not match any header name, the command will fail too.

  1. In the source file editor, select an identifier such as a class, method, or function name.

  2. On the EDIT menu, point to 3DS Navigate, and click Go To Header, or press CTRL+T.

    The header file containing this identifier opens.

Note: The command fails if the selected identifier does not match a valid entity, or if the header belongs to a framework that is outside of the prerequisites.
