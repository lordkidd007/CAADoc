Installing Native Apps Interactive Development Environment Help

This task explains you how to install the Native Apps Interactive Development Environment help.

You can access this help using the View Help command of the Help menu, or open it contextually by clicking the question mark (?) command of the 3DEXPERIENCE Open Extensions for Native Apps dialog boxes. See 3DS Contextual Help. The API documentation is still available using the 3DS Help Viewer of the Help menu, or using the Alt+F1 shortcut from a class name selected in a source editor. See 3DS Help Viewer.

This task also explains you how to uninstall the Native Apps Interactive Development Environment help.

  • Installing Native Apps Interactive Development Environment Help
  • Uninstalling Native Apps Interactive Development Environment Help
Before you begin: Note that:
  • You should have first installed the Native Apps Development Toolset CD ROM.
  • You should also have set up Native Apps Development Toolset and the Native Apps Development Toolset licenses.
  • You should also have installed Native Apps Interactive Development Environment from its VSIX. See Installing Native Apps Interactive Development Environment.
Warning You should be part of the HelpLibraryUpdaters group in the Local Users and Groups. To check this, in the Control Panel, click User Accounts, click Change account type, then click Manage User Accounts. In the User Accounts dialog box, click the Advanced tab, and click Advanced. In the lusrmgr dialog box, click Groups, double click HelpLibraryUpdaters, and check whether you are part of the group. If not, click Add to add you to this group.

Installing Native Apps Interactive Development Environment Help

To install Native Apps Interactive Development Environment help:

  1. Launch Visual Studio with administrator privileges.

  2. On the HELP menu, click Add and Remove Help Content.

    The Microsoft Help Viewer 2.0 window opens.

  3. In Installation source, check Disk.

  4. Either type or select from an explorer the path of the help file.

    This path is by default C:\Program Files (x86)\Dassault Systemes\B417\intel_a\resources\VisualStudio\HelpContent2012\wxpug.msha.

  5. Click Add.

  6. Click Update.

    The following prompt opens.

  7. Click Yes.

    The Native Apps Interactive Development Environment help is now installed.

The help is available for display:

  • To display the Native Apps Interactive Development Environment help:

    1. Launch Visual Studio.

    2. On the HELP menu, point to Set Help Preference, and click Launch in Help Viewer.

    3. On the HELP menu, click View Help.

      The Microsoft Help Viewer window opens. The Contents tab is normally displayed. Otherwise, click it on the left side.

    4. Click the arrow in front of Native Apps Development Toolset.

    5. Click Native Apps Interactive Development Environment.

      The Native Apps Interactive Development Environment help opens.

  • To contextually open the help, click ? from the 3DS plug-in dialog boxes which opens the help on the page describing the current dialog box.

Uninstalling Native Apps Interactive Development Environment Help

To uninstall the Native Apps Interactive Development Environment help:

  1. Launch Visual Studio.

  2. On the HELP menu, click Add and Remove Help Content.

    The Microsoft Viewer 2.0 window opens.

  3. Click the Manage Content tab.

    The available helps are displayed.

  4. Click Remove beside Native Apps Development Toolset.

    Click Update.

  5. The Native Apps Interactive Development Environment help is uninstalled.
