List Question Answers

Role

Returns a list of the answers belonging to an iQuestion. It Does not return validated answers.

Request

 Method  Path 
GET api/question/listanswers

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
iQuestions_id yes A positive number. Identifier of the iQuestion to list answers from.
limit no A number greater than one. The maximum number of elements returned by the list.
orderBy no A string. Possible values are:
  • rate: to order following the number of votes.
  • qna_update: to order following the answer dates.
 Ordering result for the answers inside.
page no A number greater than one. Optional parameter to be used with "limit". For paginated result (see Paginated Result).

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:

 Key   Description   Value   Applicability 
NbRatings Number of votes for this answer. A positive number. yes
_acl Access Control List (ACL) of the answer for the current user.
