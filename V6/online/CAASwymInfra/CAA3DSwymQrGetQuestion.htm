Get Question

Role

Returns informations about an existing iQuestion.

Request

 Method  Path 
GET api/question/get

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
iQuestions_id yes A positive number. Identifier of the iQuestion to get.
