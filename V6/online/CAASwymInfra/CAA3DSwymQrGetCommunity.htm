Get Community

Role

Returns informations about an existing community.

Request

 Method  Path 
POST api/community/get

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
id yes A positive number. Identifier of the community to get.
Container for all post parameters.
request_id yes 0 For internal use.
