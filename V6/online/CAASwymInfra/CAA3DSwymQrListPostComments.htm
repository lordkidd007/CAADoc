List Post Comments

Role

Returns a list of the comments belonging to a post. Returns also deleted comments (see moderation_status in response).

Request

 Method  Path 
POST api/post/listcomments

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
community_post_id yes A positive number. The identifier inside the community of the post to get comments from.
limit no A number greater than one. The maximum number of elements returned by the list.
page no A number greater than one. Optional parameter to be used with "limit". For paginated result (see Paginated Result).
Container for all post parameters.
request_id yes 0 For internal use.

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:

 Key   Description   Value   Applicability 
_acl Array of Access Control List (ACL) per action on blog post comment for the current user.

The value is a JSON composed as follows:

 Key   Description   Value   Applicability 
delete Comment deletion ACL.

The value is a JSON composed as follows:
