Rate Answer

Role

Votes or unvotes for an answer. Express the agreement with tan answer by voting for it.

Request

 Method  Path 
GET api/answer/addrate

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
qna_answer_id yes A positive number. Answer identifier
ratevalue yes A number. Possible values are:
  • 0: unvote for the answer (if any previous voting has been made).
  • 1: vote for the answer.
Voting value.

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is itself a JSON composed as follows:

 Key   Description   Value   Applicability 
id Answer identifier A positive number. yes
qna_id Answer identifier. A positive number. yes
rate Rate value. 1 yes
