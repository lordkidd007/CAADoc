Add Question

Role

Adds an iQuestion into an existing community. Returns the added iQuestion.

Request

 Method  Path 
POST api/question/add

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
community_id yes A positive number. Identifier of the community where the iQuestion will be added.
published yes A boolean:
  • 0: draft.
  • 1: published.
Should always be 0 has long as the iQuestion has not been edited with additional information. 		Status of the iQuestion.
question yes A string. Format is HTML-like (see Format HTML-like). iQuestion message.
title yes A string. Title of the iQuestion.
Container for all POST parameters.
request_id yes 0 For internal use.
