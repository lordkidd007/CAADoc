Log in

Role

Logs user in 3DSwym if a Service Ticket is given as input parameter. Redirects to 3DPasport login URL otherwise.

Request

 Method  Path 
GET api/passport/login

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
ticket no A string. A service ticket mandatory to log in. This parameter is a query parameter (see GET Method).

Response

The response is an HTML page which content may vary depending on input parameters and request result.
