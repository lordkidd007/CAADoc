Version

Role

Returns the API Version.

This API must be called just after a log in to the 3DSwYm server, and before any other call to this server.

Request

 Method  Path 
GET api/index/tk

Parameters

None.

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is itself a JSON composed as follows:

 Key   Description   Value   Applicability 
tk no
ServerVersion server version. A string. yes
ProductName A string. no
ServerToken A CSRF token (see CSRF token). A string. yes
