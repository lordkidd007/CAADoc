List Current User Communities

Role

Returns a list of the communities associated with the current user. Default filter returns all communities the user is member of.

Request

 Method  Path 
GET api/community/listmycommunities

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
creation_granted_for no A string. Possibles values are:
  • post: return only communities where the user can create post.
  • media: return only communities where the user can create media.
  • iquestion: return only communities where the user can create question.
  • ideation: return only communities where the user can create ideation.
  • wikitree: return only communities where the user can create wiki.
 Filter result by user insertion rights policy.
limit no A number between 1 and 128. Default value if not indicated is 10. The maximum number of elements returned by the list.
page no A number greater than one. Default value if not indicated is 1. Optional parameter to be used with "limit". For paginated result (see Paginated Result).

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:
