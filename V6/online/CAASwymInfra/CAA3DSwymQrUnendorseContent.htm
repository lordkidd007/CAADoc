Unendorse Content

Role

Unendorses content. Remove previously setted endorse flag on content.

Request

 Method  Path 
POST api/user/unendorsecontent

Parameters

None.

POST Body

The value is a JSON composed as follows:

 Name   Require   Value   Description 
params yes The value is a JSON composed as follows:
 Name   Require   Value   Description 
model yes A string. Possible values are:
  • Community: for a community.
  • CommunityPost: for a blog post.
  • Media: for a media.
  • QnaQuestion: for an iQuestion.
  • Dsi_DFSTree_WikiTree: for a Wiki page.
  • Dsi_Ideation_Ideation: for Ideation.
 Type of the data to unendorse.
model_id yes A positive number. Identifier of the data to unendorse.
Container for all POST parameters.
request_id yes 0 For internal use.

Response

The response is a JSON composed as follows:

 Key   Description   Value   Applicability 
monitoring no
nb_result no
request_id no
result no
success Status of the action. A boolean:
  • false: the action failed.
  • true: the action succeeded.
yes
