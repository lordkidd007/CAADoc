List Posts

Role

Returns a list of the posts belonging to a community.

Request

 Method  Path 
GET api/post/list

Parameters

The GET parameters (see GET Method) are:

 Name   Require   Value   Description 
community_id yes A positive number. Identifier of the community from which post will be listed.
limit no A number greater than one. Maximum number of post to return.
page no A number greater than one. Default value if not indicated is 1. Optional parameter to be used with "limit". For paginated result (see Paginated Result).

Response

The response is a standard 3DSwYm JSON (see JSON as Output) where the key result is an array. Each element of the array is a JSON composed as follows:

 Key   Description   Value   Applicability 
_community Information on the community the blog post has been published into.

The value is a JSON composed as follows:
