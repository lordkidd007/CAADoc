Report Parameters

There are the parameters that you set with the SetReportParameters method of the CATPLMBatch class. These parameters are batch independent.

Parameter Name What is it? Possible Values
PLMBATCH_Report_Directory Defines the directory were reports and traces will be generated.

By default, the report directory is the PLMBATCH_Default_ReportDir value. When not exported, the report directory is set to the current directory (directory where the .exe application has been submitted).

Complete path of the directory.
PLMBATCH_Report_GlobalReportName Defines the name of the htm global report where traces will be generated.

The default value is batch dependent.

htm extension is automatically added at the end of the global report name when needed.

When running the application, the global report is written under the report directory.

 Name
PLMBATCH_Report_ObjectReportName Defines the format of the Object report where traces will be generated. When processing objects, `PLMBatch Data Access` platform creates a report per processed objects. By default, this object report is an htm report, named with the name of the processed object plus .htm extension. Certain specific batches support different formats for object reports (htm or txt). Refer to the documentation of the application to know which formats are supported. 1: txt format

2: htm format
