Check Representation Batch Parameters
Quick Reference
Parameter Name What Is It? Possible Value Multi List
CHECKREP_CleanMode Main action that has to be done on the selected objects Clean, Check (*) No
CHECKREP_Severity Severity to check

(1): Severity One: Cleaning action may lead to data deletion.

(2): Severity Two: Cleaning action may lead to data modification (without deletion).

(3): Severity Three: Unimportant error. Cleaning action without huge impact on data.

 1, 2, 3 Yes
CHECKREP_Message Defines which kind and detail information has to be written in the report. Long, Short, Long/Short (*) No
CHECKREP_ErrorSelected Error to process filter: defines a list of error, identified with their names, to process or not, depending on the <CHECKREP_ErrorSelectionMode> tag value. UAV_11, LIF_3, DOC_4, CST_4
See the complete list of possible error in the CheckRepresentation documentation error code table		 Yes
CHECKREP_DomainSelected Error to process filter: defines a list of error, identified with their Domain, to process or not, depending on the <CHECKREP_ErrorSelectionMode> tag value. Drafting, Sketcher, MechanicalDesign, ....
See the complete list of possible Domain in the CheckRepresentation documentation error code table		 Yes
CHECKREP_KeywordSelected Error to process filter: defines a list of error, identified with keyword, to process or not, depending on the <CHECKREP_ErrorSelectionMode> tag value. PCS, Specification, Update, LinksMngt ...
See the complete list of possible keyword in the CheckRepresentation documentation error code table		 Yes
CHECKREP_ErrorSelectionMode Define which filter has to be applied on the list of selected errors. NonePlus, AllBut (*) No

italic: default value

(*) all the value are not case sensitive: you can write "Clean", "CLeant", "NONEPlus", "LONG".

NonePlus

Only errors defined in the <CHECKREP_xxxxSelected> tag will be checked/cleaned. This tag is ignored when no specific errors to process filter is provided with the <CHECKREP_xxxxSelected> tag.

AllBut

All errors available on the site except the one defined in the <CHECKREP_xxxxSelected> tag will be checked/cleaned.
