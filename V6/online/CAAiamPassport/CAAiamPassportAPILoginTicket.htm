CAS Login Ticket

Role

Returns the login ticket (lt) required for a later 3DPassport CAS server authentication.

lt is used as CSRF token to prevent automatic replay of authentication request. It should be passed when authenticating on CAS server.

Request

Method Path
GET /login

Parameters

Name Required Value Description
action YES get_auth_params

Response

The response code can be:

  • 200

    Login ticket successfully returned as a string.

  • Error

Example

The web services has been launched while it is the first connection to the 3DPassport. There is no CATSTGC cookie yet, and no 3DPassport session. The response code is 200, and the response body is the login ticket.
