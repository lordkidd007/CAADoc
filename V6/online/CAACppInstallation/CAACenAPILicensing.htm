API Licensing
 

This article explains which API can be used to create an application aimed to be run on a target 3DEXPERIENCE platform installation.

How to Develop under Licensing Conditions

It is recommended to install the process targeted to support the CAA app runtime, then its API, select the corresponding license and develop directly using this configuration. If this is not possible, at least one such configuration should exist to allow testing of the app under real conditions.

When creating the architecture of the app, it is a good practice to cautiously establish the prerequisites of each framework. It may happen that few non authorized frameworks be installed with the product and this helps avoiding problems that otherwise could be discovered later in the development process.
