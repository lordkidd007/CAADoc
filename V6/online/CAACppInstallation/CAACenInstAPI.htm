Installing API
 

This task explains how to unload the API of 3DEXPERIENCE processes on a single computer running a supported Windows operating system.

Installation and uninstallation rely on Windows-compliant tools enabling anyone familiar with Windows procedures and concepts to install the software without assistance.

Note that the API installation uses the Dassault Systèmes V6 installer.
Before you begin:
  • First install the CD-ROM containing the run time part of the processes for which you want to install the API.
  • Before starting the installation, see the Program Directory to check you have all the software prerequisites.
  • Please close any running programs before starting the installation.
  • Note that you can undo each installation step using the Back push button on each dialog box.

Installing the Files

  1. Log on as an administrator.

    You must belong to the Administrators group, or have the privileges assigned to the Administrators group. Otherwise, you will not be able to start the installation.

  2. Insert the CD-ROM into the drive.

    If needed double-click on setup.exe. The setup program starts. A prompt asks whether you want to allow the Dassault Systemes installer to make changes to your computer.

  3. Click Yes in this prompt.

    The 3DEXPERIENCE CAA Installation dialog box opens.

  4. Click Next.

    The 3DEXPERIENCE CAA Choose Destination Location dialog box opens. By default, it displays as selected the folder where you have installed the run time part of the processes. You should keep this folder to install the API at the same place.

  5. Click Next.

    3DEXPERIENCE CAA Select Processes dialog box opens. It lists the processes installed during the run time part installation as selected.

  6. If you want to install the API of only several processes, click the ones you want to select from the list.

    Clicking a process unselect all the others. Then you can either select a process and use the SHIFT key to select another process and all the processes between these two processes, or the CTRL key to successively select several processes. Possibly use CTRL+A to select again all the processes.

  7. Click Next.

    The 3DEXPERIENCE CAA Start Installation dialog box is displayed.

  8. Review your choices before installing the code.

    The central area lists the current choices you set in the previous steps (depending on what you chose to install):

    • The installation directory.
    • The names of the configurations and processes to install.

    Possibly click Back to move backward to the process selection step.

  9. Click Install to start copying the files to your computer.

    The names of the files are displayed when they are installed along with their sizes. When the installation is finished, the 3DEXPERIENCE CAA Setup Complete dialog box opens to inform you that the setup procedure has finished installing the API on your computer, and listing the installed configurations and processes.

  10. Click Close. The installation is complete.

Finding the Encyclopedia

The Encyclopedia home page is available from the following locations:

  • The Developer Assistance shortcut in the Start menu, Dassault Systemes 3DEXPERIENCE R2105x, Tools.
  • The file <ROOT_INSTALLATION_PATH>\CAADoc\Doc\DSDoc.htm

where <ROOT_INSTALLATION_PATH> is the path of the folder where have installed the API, namely C:\Program Files\Dassault Systemes\B417 as proposed as a default by the installation procedure.

The Encyclopedia will provide you with all necessary information helping you starting with and using the installed brand API.
