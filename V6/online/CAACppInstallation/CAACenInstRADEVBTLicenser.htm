Setting up Native Apps Development Toolset Licenses
 

This task explains how to set up the licenses for the Native Apps Development Toolset.
Before you begin:
  • Before starting the license set up, refer to the Program Directory where additional useful information could be given.

Once the Native Apps Development Toolset is installed, you need to set the licenses you wish to reserve for each Native Apps Development tool, for each user, and for each operating system.

  1. Start the licensing manager tool by either:

    • On the Start menu, click All Programs, then click Dassault Systemes Development Toolset for Native Apps  3DEXPERIENCE R2015x, click Tools, and click 3DEXPERIENCE Open Development Toolset Licenses Manager.

    • Running the following command:

      C:\>"Program Files (x86)\Dassault Systemes\B417\intel_a\code\bin\CATVBTLicenser.exe"

      provided you have used the default installation folder, or:

      C:\><INSTALLATION_PATH>\intel_a\code\bin\CATVBTLicenser.exe

      where:

      <INSTALLATION_PATH> is the folder where you have installed the Native Apps Development Toolset CD-ROM.

  2. The 3DEXPERIENCE Open Development Toolset License Manager dialog box is displayed.

    The left-hand part shows you the configurations and products available with Native Apps Development Toolset. The right-hand part shows you the licenses available either on the licensing server cross-referenced with what you have installed.

  3. Check the licenses you need, click Apply.

  4. A prompt confirms that the licenses are set.

    Click OK. Then close the box by either clicking Exit in the File menu, or clicking X on the top right corner of the dialog box. A prompt may ask you to confirm again. In this case, click Yes.

You need then to activate the Visual Studio customization. See Installing Native Apps Development Toolset.
