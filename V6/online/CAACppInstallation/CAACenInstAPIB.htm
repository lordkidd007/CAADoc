Installing API in Text User Interface (TUI) Mode
 

This task explains how to unload 3DEXPERIENCE process API on a single computer running a supported Windows operating system in Text User Interface (TUI) mode.

Installation and uninstallation rely on Windows-compliant tools enabling anyone familiar with Windows procedures and concepts to install the software without assistance.
Before you begin:
  • First install the CD-ROM containing the run time part of the processes for which you want to install the API.
  • If you perform several installations of the run time part, keep at hand the paths of the folders where they are installed.
  • To prevent the installation from hanging due to concurrently running programs such as screen savers or virus scanning programs, we recommend that you first shut down any such programs.

Installing the Files

This section describes how to perform an installation in TUI mode.

In this mode, the machine does not need a graphics card. The installer prompts you to answer questions in text mode.

  1. Log on as an administrator.

    You must belong to the Administrators group, or have the privileges assigned to the Administrators group. Otherwise, you will not be able to start the installation.

  2. Open a Command Prompt window.

    The command must be launched from an elevated command prompt from Windows Vista onwards.

  3. Go to the folder to which you unloaded the run time media.

    This folder is by default C:\Program Files\Dassault Systemes\B417 as proposed by the installation procedure.

  4. Insert the API CD-ROM into the drive.

  5. Launch the installation by running StartTUI.exe from the API CD-ROM, mapped for example on the Z: drive.

Finding the Encyclopedia

The Encyclopedia home page is available from the following locations:

  • The Developer Assistance shortcut in the Start menu, Dassault Systemes 3DEXPERIENCE R2105x, Tools.
  • <ROOT_INSTALLATION_PATH>\CAADoc\Doc\DSDoc.htm

where <ROOT_INSTALLATION_PATH> is the path of the folder where have installed the API, namely C:\Program Files\Dassault Systemes\B417 as proposed as a default by the installation procedure.

The Encyclopedia will provide you with all necessary information helping you starting with and using the installed brand API.

Uninstalling

Use the Uninstall.bat command to uninstall both the run time code and the APIs.

  1. Using an explorer, go to the installation folder.

    This folder is by default C:\Program Files\Dassault Systemes\B417 as proposed by the installation procedure.

  2. Locate the Uninstall.bat file.

  3. Right click Uninstall.bat, and click Run as Administrator.

  4. Click OK in the confirmation prompt.

    The run time code and the APIs and the APIs are uninstalled.
