Uninstalling a Brand API
 

This task explains how to uninstall unused CAA Brand API.

When you stop supporting your applications for a given API V6 release level, the corresponding Brand API installation becomes of no use. Here is the way to uninstall it.
Before you begin: Note that:
  • This uninstalls all the API CD-ROMs you have installed for this release level.
  • You should uninstall the run time part of the same release level afterwards.

  1. On the Start menu, click Control Panel.

    The Control Panel dialog box opens.

  2. In the Name column of the right pane, double click Add or Remove Programs.

    The Add or Remove Programs opens.

  3. Locate Dassault Systemes Software Bxxx, and click it, where xxx is the release, such as B215 for V6R2014. You can get the correspondence between the release code level and its marketing name in Authorized API Identification, Usage, Deprecation, and Stability

    The display expands to show Dassault Systemes Software Bxxx data.

  4. Click Remove.

    A prompt opens to ask you whether you want to uninstall Dassault Systemes Software Bxxx.

  5. Click Yes.

  6. A prompt opens with a progress bar to show the uninstallation progress, and disappears once completed.

    The Dassault Systemes Software Bxxx is removed from your computer.
