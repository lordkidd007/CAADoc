Reflect Line
Use Case

Abstract

There are two kinds of reflect lines:

  1. The conical reflect lines:
    given a point O, conical reflect lines are the set of points P for which the tangent to the surface in each point P presents the same angle with OP.
  2. The cylindrical reflect lines:
    given a direction D, cylindrical reflect lines are the set of points P for which the tangent to the surface in each point P presents the same angle with D.

Operator to be Used

To generate reflect lines on a surface, use the CATICGMTopReflectLine operator in GMOperatorsInterfaces. This operator has to be created using the CATCGMCreateTopReflectLine global function.

Use Case Description

The CAAGMOperatorsReflectLine.m module in CAAGMOperatorsInterfaces.edu framework illustrates how create reflect lines on a surface. This use case creates its own input data.

If you are not already familiar with geometric modeler use cases, go to About Geometric Modeler Uses Cases.

Case 1: Conical Reflect Line with 140 Degree Angle

With the input data below
Fig.1 Reflect Line Inputs: surface, point
 

and the code below:

CATICGMTopReflectLine *pConicalReflectLineOp = CATCGMCreateTopReflectLine( pGeomFactory, &topdata, 
                  pSkin, pOriginBody, 140*CATDegreeToRadian);
...
pConicalReflectLineOp->Run();
CATBody * pConicalReflectLineBody = pConicalReflectLineOp->GetResult();
pConicalReflectLineOp->Release();
pConicalReflectLineOp = 0;

Case 2: Cylindrical Reflect Line with 40 Degree Angle with x Axis

With the same input data and the code below:
