Electrical Library Overview
Technical Article

Abstract

This article gives you an overview of the Electrical Library objects.

Objects and Interfaces

Thanks to the CATIElbReferenceFactory interface [1] implemented by any product reference, the Electrical Library modeler enables you to add a device extension to a product of the following kinds:

  • Backshell
  • Contact
  • Equipment
  • Filler Plug
  • Ground Stud
  • Mounting Equipment
  • Shell
  • Single Connector
  • Splice
  • Terminal Block
  • Terminal Strip.

You can manipulate a device reference using the CATIElbDeviceReference to add, remove, or list connection points, and get the device reference type or part number.

You can manipulate a device instance using the CATIElbDeviceInstance to connect it to, or disconnect it from another device instance, list the connected device instances, list the segment ends connected to it through a segment connection point, and get its name or its type.

The CATIElbConnectionPoint interface manages connection points of the following types:

  • Cavity
  • Cavity connection point
  • Connector connection point
  • Backshell connection point
  • Shell connection point
  • Segment connection point
  • Segment position point.

It enables you to retrieve the constraint elements, and get the name, the representation, the owner, and the type of the connection point.
