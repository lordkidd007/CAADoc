ElecImportFinalizerService Object

Represents a service to post-process the electrical data after an FBDI import.

Retrieving the ElecImportFinalizerService Object

Refer to the Service Object.

Using the ElecImportFinalizerService Object

Use the Finalize method to post-process electrical data after importing them using FBDI.

Dim oElecService As ElecImportFinalizerService
Set oElecService = CATIA.ActiveEditor.GetService("ElecImportFinalizer")

oElecService.Finalize
