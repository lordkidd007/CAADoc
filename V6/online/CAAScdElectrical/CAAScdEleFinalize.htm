Post Processing Electrical Data
 

This use case shows you how to post process the imported electrical data.

This macro retrieves the active object (Root) in the session to get the ElecImportFinalizerService service object and calls its method Finalize to post process electrical data.
Before you begin: Note that:
  • You should launch CATIA.

Where to find the macro: CAAScdEleFinalizeSource.htm

  1. Launch CATIA and import the Root_GBN.CATProduct file.

  2. Select the root product Root_GBN and launch the macro.

    The following figure show the imported data.

    Fig.1 Data Before Finalize

  3. Retrieve the ElecImportFinalizerService object

    ...
Dim oElecService As ElecImportFinalizerService
Set oElecService = CATIA.ActiveEditor.GetService("ElecImportFinalizer")
...

  4. Use the ElecImportFinalizerService object to post process the electrical data.

    ...
oElecService.Finalize
...

    The following figure shows the data after having called the Finalize method.

    Fig.2 Data After Finalize

In Short

This use case has shown how to use the Finalize method of the ElecImportFinalizerService object to post process electrical data.
