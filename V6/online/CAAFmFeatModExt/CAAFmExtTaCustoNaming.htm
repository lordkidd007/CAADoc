Naming Customization
Technical Article

Abstract

This article gives you some tips to implement the CATIAlias interface. It is strongly recommended to read previously the article "Understanding Feature Naming" [1].

  • When
  • Why
  • How
  • References

When

If you feature derives from scratch, otherwise usually DS modelers provide a default implementation for their derivable features. In case of doubt, please read the Modeler documentation in the CAA Encyclopedia.

Why

You may to implement this interface to provide a free access to the display name of your feature. Otherwise only its owner (the owner of the catalog defining the StartUp) could have an access to the display name, by using the GetDisplayName of the CATFmFeatureFacade class. This method requiring credentials [2].

You may also need to implement this interface to be able to have a customized name, one different of the GetDisplayName of the CATFmFeatureFacade class.

How

This section does not describe AN implementation, but some tips. You can always have a look to the provided example [3].

StartUp

If your implementation requires to save something, add an attribute on your StartUp.

Tips:

  • Avoid to set the update behavior as IN
  • Set private the attribute if it keeps the name to return; it avoids to have a name inherited by the reference: Each feature will have its own name. See the article about attribute [4]

SetAlias

This method can keep the name passed as input (not recommended), as the name can be used to compute an NLS name (recommended). 

Note: it is strongly not recommended to save a name that can be displayed in a NLS context.

This method can be also empty.

GetAlias

The usual implementation is:

  • The attribute is not valid (empty - unset - blank ) , you return the string of GetDisplayName .
  • Otherwise
    • You return the value of the attribute (not recommended since the name can be not NLS)
    • You compute a value according to your own logic.

Don't forget, that you can use GetDisplayName to retrieve the index value, and use the same number to your own output.
