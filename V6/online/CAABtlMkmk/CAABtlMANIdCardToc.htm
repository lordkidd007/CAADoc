In this section:
Note that the IdentityCard.h file format is deprecated starting with V6R2012x. You should convert your IdentityCard.h files to XML ones. The following commands help you to do this:
  • The mkICE command enables you to interactively edit an XML Identity Card, and open a .h Identity Card to save it as an XML one.
  • The mkCreateIC command enables you to create an empty Identity Card ready to be edited using mkICE.
  • The mkIc2Xml command enables you to convert a .h Identity Card to an XML one.
