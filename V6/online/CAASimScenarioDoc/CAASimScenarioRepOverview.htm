Simulation PLM Object representations persistent data model overview
Technical Article

Abstract

This article's main goal is to help partners understand the data model of the simulation PLM object and identify the APIs necessary to navigate it.

In order to better depict the relationships, the data model and navigation behavior is described through the following four sub-sections:

  • Simulation PLM object
  • FEM representation manager
  • Scenario manager
  • Results manager
