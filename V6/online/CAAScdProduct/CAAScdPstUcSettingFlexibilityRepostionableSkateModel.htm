Setting Flexibility/Repositionable for Skate Product Model Interactively
 

The use case mainly deals with describing steps for Setting Flexibility/Repositionable for Skate Product Model Interactively.

Before you begin: Note that:
  • Launch CATIA

Where to find the macro:

This scenario of creation of Skate Structure Model can be divided in 5 steps

  1. Open Axle Assembly Reference

  2. Set Flexibility/repositioning option for following Objects as shown in [Fig.1]

    • Axle Reference -> Flexibility

      • Left Wheel Inst -> Repositionable

      • Right Wheel Inst -> Repositionable

    Fig.1 Axle Assembly Flexibility

  3. Open Skate Reference

  4. Set Flexibility/repositioning option for following Objects as shown in [Fig.2]

    • Skate reference -> Flexibility

      • Front Axle instance -> Repositionable

    Fig.2 Flexibility

  5. Next we set the repositionable for Following Occurrence objects

    • RearAxleInst-LeftWheelInst -> Repositionable

    • RearAxleInst-RightWheelInst-> repositionable

    Fig.3 Setting Repositionable Occurrence Objects
