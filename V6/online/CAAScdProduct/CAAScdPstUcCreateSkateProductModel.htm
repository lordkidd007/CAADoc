Creating Skate Product Model Interactively
 

The use case mainly deals with describing steps for creating Skate model interactively.
Before you begin: Note that:
  • Launch CATIA

Where to find the macro:

This scenario of creation of Skate Structure Model can be divided in eight steps:

  1. Create New or Open Wheel Reference

  2. Create New or Open Axle Reference

  3. Instantiate Wheel instance under Axle Reference

    • Rename as "LeftWheelInst" for newly instantiated Wheel instance

  4. Create one more instance of Wheel below Axle Reference

    • Rename it as "RightWheelInst" for newly instantiated Wheel instance.

  5. Then Create or Open Skate Reference

  6. Instantiate Axle instance below Skate reference

    • Rename as "FrontAxleInst" for newly instantiated Axle instance

  7. Create one more instance of Axle below Skate Reference

    • Rename it as "RearAxleInst" for newly instantiated Axle instance

  8. Save into database

    Fig.1 The Skate Model

    The Skate model "Skate" as PLM_ExternalID of root and "---" version value as shown in [Fig.1]

  9. Before positioning the Skate model the next main step to see the object with actual 3dparts we follow the following steps, this step is not mandatory but for visualization of positions this step is added

    1. open Wheel reference (Created above) right click on it and add existing Wheel 3DPart

    2. Next open Axle Reference and add existing Skate 3Dpart below it

    3. Finally open Skate Reference and add/instantiate existing SkateBoard 3Dpart below it.
