Relations and Design Tables

This section deals with Relations and Design Tables.
In this section:
  • Creating and Managing Relations is an example of creating rules and checks.
  • Creating and Managing Formulas is an example of doing some basic operations with formulas.
  • Programming with Design Tables is an example of creating a design table from the represenation of an external file, creating associations, selecting a current configuration, and so on.
  • Accessing the Design Table Values is an example of reading the values defined in the design table cells, and retrieving the list of associations or the location of a parameter in a table.
