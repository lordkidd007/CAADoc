Parameters

This section deals with contextual help.
In this section:
  • About Parameters Naming helps you choose the correct method to manipulate the name of your applicative parameters.
  • Using Persistent Parameters is an example introducing the basics of creating persistent parameters, i.e., parameters that can be saved in a permanent form which persists after they are deleted from memory
  • Using Volatile Parameters is an example of using volatile parameters, i.e., parameters that won't persist after they are deleted from memory.
  • Managing Parameter Rights is an example of accessing rights on parameters. It warns you about the limitations of the methods provided.
  • Subscribing to Events is an example of using the events supported by parameters and subscribing to these events.
  • Creating a Parameter Editor and Creating a Parameter Editor(2) make up an example of creating a basic parameter editor in a CATIA window.
