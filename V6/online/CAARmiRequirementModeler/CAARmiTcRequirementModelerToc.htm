This section exposes the Requirement Modeler functionalities through CAA for CAA partners.

Requirements Central is a powerful change proposal management tool that allows you to create and maintain requirements created to propose changes to hardware and software products. With Requirements Central, you can capture and import requirements from an external specification, work with traceability reports related to requirement specifications, and manage decisions made about proposed requirements. So here you will see the Basic Scenarios that you may execute with batch files by using Requirement Modeler specific CAA APIs. Also you will get to know how generic APIs can be used for some basic tasks.
