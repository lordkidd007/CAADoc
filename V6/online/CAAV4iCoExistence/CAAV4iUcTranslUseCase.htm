What You Will Learn With This Use Case

This use case is intended to show you how to migrate geometrical element from a V4 data. This is a way to adapt the behavior of the migration.

The CAAV4iEduTranslV4ToV5 Use Case

CAAV4iEduTranslV4ToV5 is a use case of the CAAV4V5Interoperability.edu framework that illustrates CATIAV4Interfaces framework capabilities.

What Does CAAV4iEduTranslV4ToV5 Do

CAAV4iEduTranslV4ToV5 is the implementation of the CATIV4DataTranslator interface. This interface is used by CATIA during Copy Paste operation of a V4 Element into a Part. According as you implemente the interface, the V4 element will be pasted or not into the Part. You can take into account the type of the element, its name, etc., to know if an element has to be pasted. Currently the implementation is used only for spline curves and nets.

How to Launch CAAV4iEduTranslV4ToV5

First of all, set up the build time environment, compile CAAV4iEduTranslV4ToV5 with its prerequisites, set up the run time environment to take into account the use case dictionary.

Then, launch a CATIA session. Open a V4 model that contains Spline curves or NET surfaces. Copy the *MASTER into a Part. The sufaces and curves are converted.

Where to Find the CAAV4iEduTranslV4ToV5 Code

The CAAV4iEduTranslV4ToV5 use case is made of one file located in the CAAV4iEduTranslV4ToV5.m module of the CAAV4V5Interoperability.edu framework:

InstallRootFolder\CAADoc\CAAV4V5Interoperability.edu\CAAV4iEduTranslV4ToV5.m

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

Step-by-Step

It is possible to manage your intent in CAAV4iEduTranslV4ToV5 :

  1. Changing the V4 Paste Behavior

Changing the V4 Paste Behavior

  HRESULT CAADataTranslator::GetUserIntent(CATV4iV4Element * V4Elem, int & oIntent)
  {
    HRESULT rc = S_OK;
    ...
    oIntent = 1
    return rc;
  }

oIntent is set to:

  • 1: do the generic process. The V4 element is converted.
  • 0: The element is not converted.

In Short

This use case provides a way to decide if an element will be pasted or not.
