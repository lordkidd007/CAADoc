Launching ExchangeServices Usecase
Use-Case

Abstract

This article gives detailed information about how to execute exchange services usecases.

  • Launching CAAVIExchangeServicesInBatch Usecase
  • Launching CAAVIExchangeServicesWithSendAPI Usecase
  • References

Launching CAAVIExchangeServicesInBatch Usecase

CAAVIExchangeServicesInBatch usecase is a simple CAA Batch usecase. For execution of this usecase Please refer to Building and Launching a CAA Use Case [1].

To know more about CAA Batch usecase,Please refer to Creating a Basic PLM Batch[2].

Details about all the necessary parameters and command to be executed to run this usecase is explained in How to Launch section of Exchange Data using PLM Batch[3] .

Launching CAAVIExchangeServicesWithSendAPI Usecase

CAAVIExchangeServicesWithSendAPI usecase is illustrating C++ API SendToXPDM.This usecase can be used only from 3DEXPERIENCE Platform Client Machine.

CAAVIExchangeServicesWithSendAPI usecase is a simple CAA Batch usecase. For execution of this usecase Please refer to Building and Launching a CAA Use Case [1].

To know more about CAA Batch usecase,Please refer to Creating a Basic PLM Batch[2].

This usecase requires event bus connectivity.Events created by this usecase can be monitored by using gateway administration console. For more details about this refer to Description[4]. 

Details about all the necessary parameters and command to be executed to run this usecase is explained in How to launch section of Export Data using SendToXPDM API[5].
