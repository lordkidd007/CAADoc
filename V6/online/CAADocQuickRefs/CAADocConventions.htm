Typographical Conventions

The Encyclopedia uses the following typographical conventions:

Convention Indicates...
italic type Cross-references to other articles for which a hyperlink is provided using a number between square brackets in the References section at the bottom of the article.

For example, See Object Modeler [1].

Interface and class names. For example: CATBaseUnknown, CATI3DGeoVisu, CATIPLMComponent
interface entity type All interface entities (commands, dialog box options, named specification tree components, keyboard keys, etc.).

Typical examples:

  • Select Tools > Options...
  • Click Reverse Direction
  • Press Ctrl+C to copy the file
> A menu item. For example, Select Tools > Options... means select the Options... item from the Tools menu.
code type Input data, variables and code entered in a Command Prompt window.

Code samples appear as follows:

Mouse Terminology

The 3-button mouse is recommended for usability reasons.

How you use the mouse depends on the type of action you need to perform.

Use this mouse button... Whenever you read...
  • Select (menu commands, geometry in graphics area, dialog box options and tabs, etc.)
  • Click (toolbar commands, dialog box command buttons, selection of a location in the window, etc.)
  • Double-click
  • Shift-click
  • Ctrl-click
  • Drag
  • Drag and drop (icons onto objects, objects onto objects)
  • Move or pan
  • Right-click (to select shortcut menu items)
