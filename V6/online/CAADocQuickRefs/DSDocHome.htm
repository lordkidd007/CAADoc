Welcome to DS Developer Assistance

Designed as a single entry point to all Dassault Systèmes app and brand API documentation, DS Developer Assistance covers all brands. It can be viewed using Internet Explorer or Mozilla Firefox.

DS API allows you to create C++ batch and interactive applications on top of CATIA, DELMIA, and SIMULIA apps accessing the ENOVIA Collaboration Platform through the ENOVIA Collaborative Environment, excluding the ENOVIA 3DLive standalone solution. It also allows you to create Automation macros in VBScript, VBA, C#, and VB.NET, as well as consume web services.

DS Developer Assistance is made up of technical articles and how-to documentation, use cases along with their code, and reference documentation that can be accessed rapidly via the table of contents on the left. You can also search for a topic, a specific term or phrase directly via the full-text search engine.
