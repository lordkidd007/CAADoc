What's New with V6R2009x Automation
Technical Article

Abstract

This article presents Automation objects, methods, and properties that are added since V6R2009 GA to enhance V6 Automation capabilities, support functional evolutions, and correct defects. Some of those modifications already exist in V6RXX service packs. A few properties and methods might have been modified or removed. VB and VBScripts code using those methods may have to be modified. VB projects using the corresponding objects must be recompiled.

Consult About Automation Languages, Debug, and Compatibility for details on manual adjustments that may be required for existing VB/VBA and VSTA projects to be usable on a new release.

Modified Methods and Properties
DrawingDefineGenView::DefineFrontView, DefineIsometricView, DefineProjectionView, DefineSectionView DefineCircularDetailView, DefinePolygonalDetailView, DefineAuxiliaryView, DefineUnfoldedView, DefineStandAloneSection. Those methods were not implemented in V6R2009. Their signatures have been modified in V6R2009x to return the created view. See the DrawingDefineGenView Object for more information.
DrawingGenView::ViewLinks (Property Read/Write) This property was not implemented in V6R2009. It has been replaced by the GetViewLinks and SetViewLinks methods in V6R2009x.
