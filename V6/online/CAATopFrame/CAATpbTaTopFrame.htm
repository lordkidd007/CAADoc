What is the TopFrame?

The TopFrame initialize all common components of the 3DExperience platform. These components are:

  1. Coachmark
  2. Compass
  3. TagNavigator
  4. TopBar

The main purpose of TopFrame is to ease the task of initialization. The different components of the TopFrame usually need some common parameters. Initializing TopFrame can be done with one call of the init with the rights parameters.

Deployment

Dependency

The TopFrame has a dependency on UWA library.

All libraries, modules used are available in runtime view in the folder:
    \webapps

Top frame has also dependency on Coachmark, Compass, TagNavigator and TopBar CSS.

Initialization

Use init method to initialize the TopFrame with its options.
TopFrame.init(options)
Parameters
options {Object} options needed to initialize all shared component

  • userId {String} ID of the user, use as login on the platform
  • envId {String} [Optional] the environment identifier, also called the tenant ID.
  • appId {String} [Optional] ID of your app (e.g. IFWE, SwYm or CSMngt)
  • baseAppletPath {String} Compass Path to the applet directory
  • baseHtmlPath {String} Compass Path to the HTML directory
  • baseImgPath {String} Compass Path to the image directory
  • swymUrl {String} url of a SwYm server where to get the picture
  • passportUrl {String} URL of passport (e.g. https://iam.ppd.3ds.com)
  • proxyTicketUrl {String} URL to call to get a proxy ticket for V6 (e.g. https://{your-domain}/api/passport/ticket?url=V6)
  • startupParams {String} Parameters given by myApps with the services URL needed
  • myAppsBaseURL {String} MyApps server URL
  • userName {String} set the user name for TopBar, default value is : "UserName"
  • brand {String} set the brand for TopBar, default value is : "IFWECompass"
  • application {String} set the application name for TopBar
  • lang {String} language to display the components
  • launchOnSameDomain {Function} [Optional] function for Compass to launch app in an iFrame (Used by Enovia)
  • closeOnLaunch {Function} If set to true, the panel of the Compass closes automatically when launching an app
  • onFilterChange {Function} [Optional] callback function triggered when the filter of the tagger change, when the user refines his tags

Exemple

       require(['TopFrame'], function (TopFrame){
           var options = {
               userId: "axz",
               envId: "BSFR215OUTEU4",
               appId: "IFWE"
               baseAppletPath: "../WebClient/",
               baseImgPath: "../plugins/Compass/html/",
               baseHtmlPath: "../plugins/Compass/images/",
               swymUrl: "https://dev.3dswymers-ppd.dsy.ds/",
               passportUrl: "https://iam.ppd.3ds.com",
               proxyTicketUrl: "http://ve4al419dsy:8090/ematrix/resources/bps/proxyticket",
               startupParams: paramsRetrievedByMyAppsServices,
               myAppsBaseURL: "http://myapps:8090/ematrix",
               userName: "Julien Masson",
               brand: "IFWE Compass",
               lang: 'en',
               application: "Application",
               onFilterChange: function (filters) {
                   alert(filters);
               }
           };
           TopFrame.init(options);
       });
