Standard & Specifications

WSDL

  • Web Services Description Language (WSDL) 1.1
    Currently supported level of WSDL specification for CAA Web services descriptions.

SOAP

  • Simple Object Access Protocol (SOAP) 1.1
    Currently supported level of SOAP specification for the CAA Web services runtime.

UDDI

  • UDDI Version 2.04 API Specification
    "Describes the programming interface and expected behaviors of all instances of the Universal Description, Discovery and Integration (UDDI) registry".
  • UDDI Version 2.03 Data Structure Reference
    "Detailed descriptions of the XML structures associated with inquiry and publish SOAP messages".
  • UDDI Other Core tModels, Version 2.04
    "Contains tModels definitions used to help publish Web services in UDDI using leading industry encoding schemes and standard protocols".

WS-I

  • WS-I
    "Open industry organization chartered to promote Web services interoperability across platforms, operating systems and programming languages".
  • Basic Profile 1.0
    "Provides interoperability guidance for a core set of non-proprietary Web services specifications, such as SOAP, WSDL and UDDI, along with interoperability-promoting clarifications and amendments to those specifications".
  • Basic Profile 1.0 Errata
    "Contains the set of published errata for the Basic Profile 1.0".

Technical Articles

  • Using WSDL in a UDDI Registry, Version 2.0.2
    A Technical Note from OASIS that describes a mapping between WSDL definitions and the data models of a UDDI registry.
  • What is Windows Communication Foundation ?
    A Microsoft site dedicated to Windows Communication Foundation which provides technical articles, tutorials and samples for using Windows Communication Foundation (.NET 3.0) web services.

Resources

Axis 2.0

  • Axis 2.0 Home Page
    The home page of Apache Axis2 /Java which gives a global overview of the project.
  • Axis 2.0 User Guide
    The part of the Axis2.0 user guide, which explains how to write a web service client.

Xerces

  • Xerces-J Downloads
    Download Xerces2 Java Parser 2.6.2 release (or any most recent release).

UDDI4J

  • UDDI4J Home Page
    "UDDI4J is a Java class library that provides an API to interact with a UDDI registry".
