3D Shape Editors
Technical Article

Abstract

This article presents two editors, 3D Shape editor and Logical 3D Shape editor, and lists interfaces to extend them.

3D Shape Editor

This editor edits a 3D Shape which is the main stream of a Product Representation Reference [1],a customization of VPMRepReference.

Fig.1 Product View

The 3D Shape associated with the Product Representation Reference contains the physical view of the design. The 3D Shape editor edits this kind of 3D Shape with several apps: Generative Wireframe and Surface Design, Part Design, Generative Shape Design and so on.

Fig.2 Interactive View

This editor can be extended by at least these four interfaces:

Interface Name Behavior Use Case
CATIPrtWksConfiguration Each implementation of this interface is a new app [4] of the 3D Shape Editor [2]
CATIPrtWksAddin Each implementation of this interface is a new add-in [4] for 3D Shape apps [3]
CATIShapeDesignWorkshopAddin Each implementation of this interface is a new add-in [4] of the Generative Shape Design app [3]
CATIPrtCfgAddin Each implementation of this interface is a new add-in [4] of the Part Design app [3]

Logical 3D Shape Editor

This editor edits a 3D Shape which is the main stream of a Logical Representation Reference, a customization of RFLVPMLogical3DRepReference.

Fig.3 Logical View

The 3D Shape associated with the Logical Representation Reference contains the logical view of the design. The Logical 3D Shape editor edits this kind of 3D Shape with few apps as for example CATIA Live Shape. The design could be expressed with very simple forms.

Fig.4 Interactive View

This editor can be extended by two interfaces:

Interface Name Behavior Use Case
CATICATMmrLgcl3DShpWksConfiguration Each implementation of this interface is a new app [4] of the Logical 3D editor [2]
CATICATMmrLgcl3DShpWksAddin Each implementation of this interface is a new add-in [4] for Logical 3D models [3]

Where to put your CAA Mechanical Commands

Your command must first extend the 3D Shape Editor. If the command can have an utility within the Logical 3D Shape Editor, you can integrate it, but do not overload this editor with useless commands.

In Short

You can extend the:

  • Editor editing a 3D Shape whose PLM Representation Reference is a Product.
  • Editor editing a 3D Shape whose PLM Representation Reference is a Logical.
