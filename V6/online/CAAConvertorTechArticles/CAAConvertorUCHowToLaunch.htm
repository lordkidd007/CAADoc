How to Launch Convertor Services Use-case
Use Case

Abstract

This article describes the steps to perform in order to execute the Convertor Services code after launching CATIA

Steps to Perform

  1. Launch CATIA
  2. Then Select PLM Access | Import | File | <own extension> (as shown in picture below)

To get your own extension file option in CATIA File-based Import , you will need to set up the build time environment, then compile CAAVIConvertors.edu framework along with its prerequisites, set up the run time environment, and then launch 3DEXPERIENCE.

The Use-cases are made up of a DLL files, so they will get loaded after you select a file of your desired extension and click on 'OPEN'.

 

Fig. 1 Steps to Launch the Convertors Use-Case

Here, the extension .CAAcmp and the Usage MyXCAD_CAAVI are customizable through changes in .capa file. Please refer to the article[1] for more information

At last when you click on Open, the Convertors code is called and our DLL gets loaded. The flow of the execution code is explained in Use-case articles.

Note: The file to be selected in the Import step are specifically mentioned in the Use-case articles. Please select those files to run a particular use-case scenario
