Electrical Modeler Overview
Technical Article

Abstract

This article gives you an overview of the Electrical objects and their attributes.

The main API to retrieve and set the electrical object attributes is CATIElecAttrAccess [1].

Electrical Geometry

Describes an electrical feature of the electrical geometry type that you create when you select Insert < Electrical Behavior < Electrical Geometry in the Electrical Assembly Design workbench.
For more information, refer to the Electrical 3D Installation User's Guide.

Attributes

V_Elec_SubType Type: String
Defines the subtype (User defined subtype).

Backshell

Description

Defines the extra length of cable to be added to take into account the wire length inside the backshell.

Conductor Group Reference

Description

Describes an electrical feature of Conductor Group type that you create when you select Insert < Electrical Behavior < Conductor Group Reference in the Electrical Assembly Design workbench.
For more information, refer to the Electrical Library User's Guide.

Attributes

V_Elec_WireGroup_Type Type: String
Defines the type of the conductor group.
V_Elec_SubType Type: String
Defines the subtype of the conductor group.
V_Elec_Bend_Radius Type: LENGTH
Defines the bend radius.
V_Elec_Color Type: String
Defines the color of the conductor.
V_Elec_Diameter Type: LENGTH
Defines the conductor diameter.
V_Elec_Line_Weight Type: LINEMASS
Defines the conductor linear mass.
V_Elec_Wire_Length_Coeff Type: Real
Returns a coefficient for conductors in a conductor group.
When computing the conductor length, this coefficient takes into account the twisting of conductors (conductor length = conductor group length x conductor length coefficient). It can be greater than or equal to 1.

Description

For more information, refer to the Electrical Library and Electrical 3D Installation User's Guides.

Attributes

Elec_Bend_Radius_Delta Type: LENGTH
Defines the bend radius value, which corresponds to the minimum bend radius of the tape curve.
This value takes into account the segment and tape bend radius rule and ends up to an increased rigidity due to the tape.
Elec_Covering_Length Type: LENGTH
Defines the length along the segment center curve covered.
Elec_Number_Layer Type: Integer
Defines the number of layers applied over the segment.
Elec_Tape_Thickness Type: LENGTH

Elec_Tape_Width Type: LENGTH
Defines the width.
Elec_Taping_Angle Type: ANGLE
Defines the taping angle.
Elec_Total_Tape_Length Type: LENGTH
Defines the tape length; depends on the taping angle, the covered length and the number of layers.

Conductor Group

Description

Describes an electrical feature of Conductor Group type that you create in the Electrical Physical System when you click Conductor Group in the Electrical Assembly Design workbench.
For more information, refer to the Electrical Conductor Routing User's Guide.

Attributes

Elec_Name Type: String
Defines the name of the conductor.
Elec_Bend_Radius Type: LENGTH
Defines the bend radius.
Elec_Color Type: String
Defines the color of the conductor group.
Elec_Diameter Type: LENGTH
Defines the conductor group diameter.
Elec_Wire_Length_Coeff Type: Real
Returns a coefficient for conductors in a conductor group.
When computing the conductor length, this coefficient takes into account the twisting of conductors (conductor length = conductor group length x conductor length coefficient). It can be greater than or equal to 1.
Elec_Sep_Code Type: String
Defines the separation code of the conductor group used by the algorithm to find out the conductor group route.

Returns the part number of the conductor group reference.
