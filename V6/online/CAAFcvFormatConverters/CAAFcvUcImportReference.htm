Abstract

This article describes the set of APIs used to convert Native App formats to/from ENOVIA V6.

 

Format Conversion Services CAA APIs Frameworks

  • DataExchangeKernelItf
    • CAA framework contains the interfaces considered as the kernel of Format Conversion Services.
  • DataExchangeBaseRepsItf
    •  CAA framework contains all interfaces related to the 3 fundamental representations of CAD domain:
    - Product structure
    - Boundary representation
    - Tessellation representation
