Launching usecases

To launch CAA usecases follow steps metioned below :

  1. Launch MyX-CAD application. Refer to How to Launch MyX-CAD?[1]
  2. Create CAA Assembly data by using MyX-CAD application. Refer to How to Use MyXCAD Data type?[2]
  3. In order to executes different scenarios of X-CAD, Please refer to X-CAD Addins[3].
