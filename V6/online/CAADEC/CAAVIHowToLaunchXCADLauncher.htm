Opening the X-CAD Workspace inside Eclipse

For launching the X-CAD Application we have to follow the execution procedure and be more prudent of errors, we need to import the application into Eclipse then launch it from Eclipse so that we can supervise the Eclipse console. Now, we will have a quick look at what to do in this case.

How to use X-CAD from Eclipse

Import X-CAD into Eclipse
  • Start Eclipse and then enter the path of Workspace repository as you want
    • Fig.1 Eclipse Workspace Launcher
  • Now go to Windows -> Show View -> Other -> 3DS Tools. Select the All four Addins to explore the project inside the Eclipse Fig.[2]

    • But Before doing this step make sure that you have added the DS Addins to the Eclipse

    Fig.2 Addin Selection
  • Now open the workspace from the 3DS addins.[Fig.4]
    • Fig.4 Open Workspace
  • Select the local Path of workspace[Fig.5].
    • Fig.5 Open Local Workspace
  • After Doing this steps we are able to import the X-CAD workspace in the eclipse.

Launching the X-CAD Use Case

After importing the X-CAD Workspace inside the Eclipse the following steps involved to launch a X-CAD Launcher Use Case.

  • Start the Eclipse with workspace for the X-CAD Application.
  • Open the Project Workspace in this tool.
  • Open Project Explorer, where we get All Framework and their respective Module.
  • Open Command prompt and Buid the Workspace. This Will create the Desired jar file in the JavaServer Folder $workspace$\win_b64\docs\javaserver
  • Now In the Project Explorer (Eclipse) Select The module Name MyXCADLauncher Fig[6] which inside Framework CAAVIMyXCAD.edu.
    • Fig.6 Eclipse project Explorer
  • Select the Module Name and 'Right Click' on it it will Shows the menu, in that Select "Run As | Java Application" fig[7]
    • Fig.7 Run The Application

  • After Executing this command we will get the UI for the XCAD Application Fig[8].
    • Fig.8 X-CAD UI
