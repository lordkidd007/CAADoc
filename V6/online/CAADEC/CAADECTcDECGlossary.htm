Glossary for X-CAD

Abbreviation Description
BOM Bill of Materials
CAD Computer Aided Design
CAD object Includes the CAD objects of type drawing, component, assembly, and model
CSE Client Side Extension 

The enhancements or extensions to the CAA required for the collaborative product
EBOM Engineering Bill of Materials
ECO Engineering Change Order
FCS File Collaboration Server
Server required for Hardlink (rapid file access)
GCO Global Configuration Object
   
IEF Integration Exchange Framework
Contains the components for the schema and user interface common to all collaborative products, such as toolbars, search functions, and menus, and can be customized by Integration Administrators.
LCO Local Configuration Object
Major object Revisions stream, which contains versions of CAD objects
Minor object Version of CAD object
MEP Manufacturer Equivalent Part
RFA Rapid File Access
Revision A revision is a stream that contains versions of CAD objects.
Role Job function, such as Design Engineer or Integration Administrator. When a role is granted access to an object or feature, all users assigned that role also get access.
TNR Type Name Revision
WIP Work-In-Progress
