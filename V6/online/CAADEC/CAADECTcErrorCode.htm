X-CAD Error Code
Use Case

Abstract

To begin with, users to get an overview of Error Code decoding while analyzing errors.

 

Error Code Definition

Error Code will NOT be displayed in the messages displayed to the user, however they will be sent back in the response sent to CSEs. The ‘errorcode’ attribute will be absent / blank incase of a successful processing.

A error code is depicted below. 

Error Code: IEF|XXX|X|X|X|XXXX
Error code Decoding Sequence

The table that follows details the relevance of each section contained in that code.

Error code Decoding Sequence    Description
Feature Code  Code of the Feature on which the error was thrown (Eg: Checkin, Checkout etc)
Location Denotes the Place of Error (Client / Server / JPO / JSP)
Product the product that caused the error (DEC / CSE / CORE)
Severity Denotes the severity of the error (0/1/2/3)
Unique Number Unique number to avoid conflict of error code
