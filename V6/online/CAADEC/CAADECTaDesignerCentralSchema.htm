CAA Schema Used For Basic Scenario
Technical Article 

Abstract

This article describes about the CAA Specific schema for X-CAD Integration. This schema is added by 3DEXPERIENCE Platform X-CAD for basic senario, or more specifically the Integration Exchange Framework (IEF) which is installed with 3DEXPERIENCE Platform products. 

This Types are specific for CAA to run CAA Usecases.

This Section describes the Data model which is used For CAA Application for basic scenario. Generic concepts of X-CAD Integration schema explained in [1].

In Short

This article has detailed the schema installed by IEF in an 3DEXPERIENCE Platform stack which is CAA Specific . Integrations inherit from this schema to run CAA Usecases to define their own integration specific schema created.
