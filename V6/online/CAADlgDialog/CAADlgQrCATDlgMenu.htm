CATDlgMenu
Quick Reference 
CATDialog
  |
  +---CATDlgMenu

CATDlgMenu is the base class for all dialog menus. Its derived classes are:

  • CATDlgBarMenu
  • CATDlgSubMenu
  • CATDlgContextualMenu.
Containment Parent
The parent of a instance of a class that derives from CATDlgMenu must be an instance of a class that derives from CATDlgDocument or CATDlgDialog.

Programmer's Guide

A CATDlgMenu instance can be constructed as the base class of one of its derived class only.
