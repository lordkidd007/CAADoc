Quick Reference

This article describes for each useful PLM Class of the Folder modeler how you can invoke the Qeops, Crud, and lifecycle operations [1] either thanks by a generic API [2], or with a modeler specific APIs.

  • Root Folder, ENOFLD_FolderRootReference
  • Folder, ENOFLD_FolderReference
  • References

Root Folder, ENOFLD_FolderRootReference

On PLM Object in Database

Behavior Supported How Is It Supported
Query Yes CATAdpPLMQueryServices
Open Yes PLMFolderServices::GetFolder
Expand No  
Save Yes For a root folder creation the Save operation is useless (creation in DB & session), but when the folder is modified (attributes) save is mandatory
Lock/Unlock Yes CATAdpLockServices
Versioning No  
Change Maturity No  
Identifier Set Values Yes CATAdpQueryServices::GetIdentificationSet

On PLM Object in Session - Modeler APIs

Behavior Supported How Is It Supported
Edition Yes PLMIFolder
Creation Yes PLMIRootFolderFactory
Deletion Yes PLMIRootFolderFactory
Navigation Yes PLMIFolder

On PLM Object in Session - generic APIs

Behavior Supported How Is It Supported
Navigation No  
Public PLM Attributes Edition (RW) Yes CATICkeObject
Public PLM Attributes Browse (R) No  
PLM class type and PLM object Identifier Browse Yes CATIPLMComponent
Identifier Set Browse ? CATIPLMIdentifierSet

Folder, ENOFLD_FolderReference

On PLM Object in Database

PLM Behavior Supported How Is It Supported
Query Yes CATAdpPLMQueryServices
Open Yes PLMFolderServices::GetFolder
Expand No  
Save Yes For a folder creation the Save operation is useless (creation in DB & session), but when the folder is modified (attributes) save is mandatory
Lock/Unlock Yes CATAdpLockServices
Versioning No  
Change Maturity No  
Identifier Set Values Yes CATAdpQueryServices::GetIdentificationSet

On PLM Object in Session - Modeler APIs

Behavior Supported How Is It Supported
Edition Yes PLMIFolder to modify a folder, PLMISubFolder to add/delete item
Creation Yes PLMIFolder
Deletion Yes PLMIFolder
Navigation Yes PLMIFolder to list sub folder, PLMISubFolder to list item

On PLM Object in Session - Generic APIs

Behavior Supported How Is It Supported
Navigation No  
Public PLM Attributes Access (R/W) Yes CATICkeObject
Public PLM Attributes Access (Read only) No  
Accessing PLM class type and PLM object Identifier Yes CATIPLMComponent
Identifier Set Attributes as String ? CATIPLMIdentifierSet
