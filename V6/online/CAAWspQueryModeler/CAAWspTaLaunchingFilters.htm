Before Launching Filters Usecase
Technical Article

Abstract

This article deals with the steps to create Filter interactively.

  • Import Product Model
  • Creating Filter on Product Root
  • Storing Filter object
  • In Short
  • References

Import Product Model

Import the Product Model by importing a 3DXML file. Follow the steps to Import 3dxml as shown in Fig.1

Fig.1. Importing a 3dxml file

After this select the appropriate 3dxml file. In our case select CAAFiltersRootAssembly.3dxml file. You will see the following Product Model from this 3dxml. See Fig.2

Fig.2. Product Model

Creating Filter on Product Root

Once you get the Product Model, we can now create Filter on the Root Product of the Model. Follow the steps as shown in Fig.3. To get the Filters commands in Action Bar, OPEN the Product Root in Product Finder App.

Fig.3. Create Filter

In the last step after filling Attribute Filter, click on OK. The Filter will be applied to the Root. The Filtered Product Model will be seen as in Fig.4.

Fig.4. Filtered Product Model

Storing Filter object

Then Store the Filter. Follow the Steps as shown in Fig.5.

Fig.5. Store Filter

The Store Filter command is available in Navigation bar of Product Finder App. Store the filter with the name shown in Fig.5

In Short

This Article gives us step by step information regarding creation of Filter on Product Model and Saving it in database. This creation of Filter is important for the Use-case Retrieving Filter object [1]
