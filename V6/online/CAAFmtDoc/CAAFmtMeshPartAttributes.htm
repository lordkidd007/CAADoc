Global Attributes

Global attributes can be managed using CATIFmtMeshPart interface:

  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, int iValue ) to set an attribute with an integer value.
  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, double iValue ) to set an attribute with an double value.
  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, const CATUnicodeString &iValue ) to set an attribute with a character value.
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, int &oValue ) to get an attribute with an integer value.
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, double &oValue ) to get an attribute with an double value.
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, const CATUnicodeString &oValue ) to get an attribute with a character value.

iName is the global attribute's name and iType is the type of specification:

  • iType ="Mesh" for Global Mesh Attributes.
  • iType ="Topology" for Global Topology Attributes.

Local Specifications

Local specifications are created with CATIFmtMeshPart interface:

  • CreateLocalSpecification ( const CATString& iType, const CATUnicodeString & iLateType, const CATUnicodeString & iName , CATIFmtLocalSpecification *& oLocalSpec ) with:
    • iType = "Mesh" for a mesh local specification and iType ="Topology" for a topology local specification.
    • iLateType is the late type of the specification to create.

Local specifications are retrieved with CATIFmtMeshPart interface:

  • GetLocalSpecifications ( const CATString& iType, const CATUnicodeString & iLateType, CATListValCATBaseUnknown_var& oListOfSpecifications ) with:
    • iType = "Mesh" for mesh local specifications and iType ="Topology" for topology local specifications.
    • iLateType is the late type of the specification to retrieve.

iName local specification attributes are set using CATIFmtLocalSpecification interface:

  • SetAttribute ( const CATUnicodeString & iName, int iValue ) to set a local specification with an integer value.

  • SetAttribute ( constCATUnicodeString & iName, double iValue ) to set a local specification with a double value.

  • SetAttribute ( constCATUnicodeString & iName, const CATUnicodeString &iValue ) to set a local specification with a character value.

iName local specification attributes are retrieved using CATIFmtLocalSpecification interface:

  • GetAttribute ( const CATUnicodeString & iName, int &oValue ) to retrieve a local specification with an integer value.

  • GetAttribute ( const CATUnicodeString & iName, double &oValue ) to retrieve a local specification with a double value.

  • GetAttribute ( const CATUnicodeString & iName, const CATUnicodeString & oValue ) to retrieve a local specification with a character value.

iName local specification support attribute is managed using CATIFmtLocalSpecification interface:

  • AddSupports ( constCATUnicodeString & iName, CATFmtListOfPath & iSupports)) to add new supports to the local specification.
  • RemoveSupports ( const CATUnicodeString & iName, CATFmtListOfPath & iSupports)) to remove supports of the local specification.
  • GetSupports ( constCATUnicodeString & iName, CATFmtListOfPath & iSupports)) to retrieve supports of the local specification.

iName local specification external geometry attribute is managed using CATIFmtLocalSpecification interface:

  • SetExternalReferences ( const CATUnicodeString & iName, CATFmtListOfPath & iSupports) to add external references to the local specification.
  • GetExternalReferences ( const CATUnicodeString & iName, CATFmtListOfPath & iSupports) to retrieve external references of the local specification.
  • RemoveExternalReferences ( const CATUnicodeString & iName, CATFmtListOfPath & iSupports) to remove external references from the local specification.
