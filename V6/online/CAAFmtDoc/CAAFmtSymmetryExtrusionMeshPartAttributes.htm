Symmetry Extrusion Mesh Part Creation

A "late type" must be given to indicate which type of mesh part is to be created.
In the case of the Symmetry Extrusion mesh part, the late type is "CATFmtExtrusionSymmetryMesher".
The Symmetry Extrusion mesh part can be created by using CATIFmtMeshManager::CreateMeshPart interface method.

Symmetry Extrusion Mesh Part Support Definition

Symmetry Extrusion Mesh Part support can be initialized and managed using CATIFmtMeshPart interface:

  • SetSupport (CATFmtPath * iSupport , int iMode = 0 , const CATUnicodeString& iType = "") to add a new support to the mesh part
  • RemoveSupport ( CATFmtPath * iSupport = NULL) to remove a support of the mesh part
  • GetSupport ( CATFmtPath *& oSupport , int iNumber = 1 ) to read a support of the mesh part
  • GetNumberOfSupports () to read the total number of the mesh part supports

Geometry

The Symmetry Extrusion direction is defined by a plane, this plane is defined by using the CATIFmtMeshPart interface following methods with iType = "Mesh" and iName = "Geometry":

  • SetExternalReference (const CATString& iType, const CATUnicodeString &iName, CATFmtPath * iSupport , int iMode = 0 , const CATUnicodeString& iLinkType = "") to add external reference to the mesh part
  • GetExternalReference (const CATString& iType, const CATUnicodeString &iName, CATFmtPath *& oReference , int iNumber = 1) to read external references of the mesh part
  • RemoveExternalReference ( const CATString& iType, const CATUnicodeString &iName, CATFmtPath * iSupport ) to remove external reference from the mesh part

  • GetNumberOfExternalReferences (const CATString& iType, const CATUnicodeString &iName ) to read the number of external references

Global Mesh Specifications

 Global mesh specifications can be set or retrieved using CATIFmtMeshPart interface:

  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, int iValue ) to set a mesh attribute with an integer value
  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, double iValue ) to set a mesh attribute with an double value
  • SetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, const CATUnicodeString &iValue ) to set a mesh attribute with a character value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, int &oValue ) to get a mesh attribute with an integer value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, double &oValue ) to get a mesh attribute with an double value
  • GetGlobalSpecification ( const CATString &iType, const CATUnicodeString &iName, const CATUnicodeString &oValue ) to get a mesh attribute with a character value

iType is the type of specification and iType ="Mesh" for Global Mesh Specifications.

Type


type: integer valuated to:

  • 1: to get an uniform distribution
  • 2: to get an arithmetic distribution
  • 3: to get a geometric distribution

default value: 1


This attribute specifies the size distribution to be applied to the mesh.

The picture shows how a uniform distribution (at the bottom) differs from a geometric one (at the top, using a ratio of 3).

RatioValue


type: double valuated to the size ratio between the first and last layer thicknesses.

default value: 1.0


This attribute specifies the size ratio R between the first layer thickness h1 and last layer thickness h2.

R = h2 / h1

WARNING: It is taken into account only if the Type attribute is equal to 2 or 3.

NbElementsValue


type: integer valuated to the expected number of boundary layers.

default value: 1


This attribute specifies the number of layers to be added to the volume mesh.

Symmetry


type: boolean valuated to create an extrusion which is symmetric.

default value: false


Attribute which specifies if we want to create a symmetric extrusion, the symmetrical plane is located ad the middle of the extrusion.

WARNING: It is taken into account only if the Type attribute is equal to 2 or 3.

AutomaticMeshCapture


type: integer valuated to:

  • 1: to ignore automatic mesh capture
  • 2: to compute automatic mesh capture

default value: 1

 

Attribute which specifies if external meshes must be captured. This attribute is linked to the attribute AutomaticMeshCaptureTolValue.

AutomaticMeshCaptureTolValue


type:  double valuated to the maximum distance under which mesh is captured.

default value: 0.


Attribute which specifies the maximum distance for mesh capture: if a mesh (element edges)  is farthest from the geometry than this tolerance, it will not be captured even if the AutomaticMeshCapture attribute is valuated to 2. 

WARNING: to precise exactly the list of mesh parts to capture, the method SetMeshPartsToCapture of CATIFmtMeshPart interface can be used:

  • SetMeshPartsToCapture (  CATFmtListOfPath * iParents)

To retrieve the parents mesh parts another method of CATIFmtMeshPart interface can be used:

  • GetParentMeshParts (CATFmtListOfPath * &oParents)
