What You Will Learn With This Use Case

This use case is intended to help you make your first steps in programming with CATIA structure Interfaces. Its main intent is to allow you to retrieve stiffener on free edge information.

The CAASddTestStiffenerOnFreeEdgeUse Case

CAASddTestStiffenerOnFreeEdge.m is a use case of the CAAStructureDetDesignDoc.edu framework that illustrates the CATIA SDD Interfaces framework capabilities.

What Does CAASddTestStiffenerOnFreeEdge Do

The goal of CAASddTestStiffenerOnFreeEdge use case is to show how to retrieve profile information.

How to Launch CAASddTestStiffenerOnFreeEdge

To launch CAASddTestStiffenerOnFreeEdge, you will need to set up the build time environment, then compile CAASddTestStiffenerOnFreeEdge along with its prerequisites, set up the run time environment, and then execute the sample.

To launch the use case, execute the following command:

mktun -c "CAASddTestStiffenerOnFreeEdge Repository Server User Password SecutityCtx Environment PLM_ExternalID"

where:

Repository
Server The server as Host::port_number/RootURI
User The user name.
Password The user password to authenticate the user.
SecurityCtx A string representing the security context (Role.Organization.Project). Choose Role as Designer.
Environment A PLM environment.
PLM_ExternalID The value of the PLM_ExternalID attribute of the PLM representation reference containing the data to create the user feature reference.

* The input model is retrieved by importing the CAASddTestStiffenerOnFreeEdge.3dxml file from the CAAStructureDetDesign.edu framework.

InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddCGR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSR.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddSteel_A42.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddWT18x179_5.3dxml
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\InputData\CAASddTestStiffenerOnFreeEdge.3dxml

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This 3D XML file contains the root Product Reference with the following attributes:

  • PLM_ExternalID: SddVBCreateSFEAll_cov0
  • Version: ---

Where to Find the CAASddTestStiffenerOnFreeEdge Code

The CAASddTestStiffenerOnFreeEdge sample is made of a single class named CAASddTestStiffenerOnFreeEdge located in the CAASddTestStiffenerOnFreeEdge.m module of the CAAStructureDetDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddTestStiffenerOnFreeEdge.m\

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATCkePLMNavPublicServices Class used to retrieve customized reference of the product.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATISddUseProductPlate Interface used to retrieve the SddProductPlate.
CATISddUsePlate Interfaces used to retrieve SddPlate.
CATIStrUseStiffenerMngt Interfaces dedicated to creation & retrieval of stiffeners.
CATISddUseProductStiffenerOnFreeEdge Interface dedicated to manage SddProductStiffenerOnFreeEdge.
CATISddUseStiffenerOnFreeEdge Interfaces dedicated to manage SddProductStiffenerOnFreeEdge.
CATIStrUseMaterialMngt Interfaces dedicated to manage material of objects.
CATIStrUseCategoryMngt Interfaces dedicated to manage category of objects.
CATIStrUseProfileType Interfaces dedicated to manage profile type.
CATIStrUseSectionMngt Interface dedicated to manage section parameters.
CATIStrUseProfileOnOpening Interface dedicated to manage a profile created on the edge of an opening on plate.
CATIStrUseOpeningMngt Interface dedicated to retrieve openings present on a plate.
