What You Will Learn With This Use Case

On this page you will learn how to use the Creating an Opening - Adaptor to create a component. This component adheres to interface CAAISddCreateOpening and creates an opening on a plate using 3DObject.

Before Starting

You may want to read from the beginning for Creating an Opening in SDD.

The CAASddCreate3DObjectOpeningOnPlate component

This component navigates to the desired plate and creates 3DObject opening on this plate.

What Does CAASddCreate3DObjectOpeningOnPlate Component Do

This component:

  • Navigates to a desired product in the SpecTree
  • Creates an empty opening
  • Navigates & selects a 3DObject
  • Applies the 3DObject as input to opening

How to create & Use CAASddCreate3DObjectOpeningOnPlate Component

The Creating an Opening page describes how to create & use the component.

Where to Find the CAASddCreate3DObjectOpeningOnPlate Code

The component can be found at the following location:
CAASddCreateOpening.m module of the CAAStructureDetDesign.edu framework:

The component is made up of the following two files:

InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddCreateOpening.m\src\CAASddCreate3DObjectOpeningOnPlate.cpp
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddCreateOpening.m\LocalInterfaces\CAASddCreate3DObjectOpeningOnPlate.h

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This component deals with the following CAA APIs:

CATIStrUseOpening3DObject Interface on SDD opening created/to create as 3DObject opening.
CATIStrUseOpeningExtrusionMngt Interface on SDD opening to manage the extrusion direction.

This component uses the following enumerations

CATStrUseOpeningExtrusionMode catStrOpeningExtrusionUndefined
catStrOpeningExtrusionNormal
catStrOpeningExtrusionDirectional
catStrOpeningExtrusionRemoval
catStrOpeningExtrusionClashFree

Step-by-Step

In this use case there are 4 logical steps in Main:

#Step
1Navigating to Product Node & Creating a Basic Opening
2Retrieving the 3DObject
3Setting the 3DObject to Create Opening
4Selecting the Extrusion Mode

Navigating to Product Node & Creating a Basic Opening

Navigation to desired profile or plate product is done with the help of the adaptor. The method Process of the adaptor [3] does the actual navigation.

From this component, we merely call the method with parameters as seen below.

...
// ---------------------------------------------
//  Add a new opening and set its basic properties
// ---------------------------------------------
int rc = -1;
CATIMmiMechanicalFeature_var spOpeningFeat, spPlateFeat;
CATStrOpeningCreationMode OpeningCreationgMode = catStrOpening3DObject;
rc = CAASddCreateStdOpeningAdpt::Process( ispRootOccurrence, 7, spPlateFeat, spOpeningFeat, OpeningCreationgMode );
if( rc )        return rc;

// ---------------------------------------------
//  Check for valid outputs
// ---------------------------------------------
CATIStrUseOpening3DObject_var sp3DObjectOpening = spOpeningFeat;
if( sp3DObjectOpening == NULL_var )                                         { cout << "Unable to set 3DObejct." << endl; return 9408; }
...

The above code navigates to the 7th product(which is a transverse bulkhead) under the root node & adds an basic opening to this plate.

Retrieving the 3DObject

Here, some the APIs from CATMecModLifeUseIts are used to navigate from root product to the 3DObject.

...
// ---------------------------------------------
//  Navigate to the 3DObject
// ---------------------------------------------
// Get SFD Part
CATIMmiMechanicalFeature_var spSFDPart;
CAASddNavigationServices::GetMechanicalPartFromProduct( ispRootOccurrence, spSFDPart );
if( spSFDPart == NULL_var )                                                 { cout << "Unable to navigate to the SFD Part." << endl; return 9409; }
CATIPartRequest_var spPartRequest = spSFDPart;
if( spPartRequest == NULL_var )                                             { cout << "Unable to navigate to the SFD Part." << endl; return 9409; }

// Get PartBody
CATUnicodeString ucPartBodyView = "MfDefault3DView";
CATBaseUnknown_var spPartBody;
spPartRequest->GetMainBody( ucPartBodyView, spPartBody );
CATIMmiUseBodyContent_var spBodyContent = spPartBody;
if( spBodyContent == NULL_var )                                             { cout << "Unable to navigate to the PartBody." << endl; return 9410; }

// Get Features under PartBody
CATListValCATBaseUnknown_var ListTotal;
spBodyContent->GetMechanicalFeatures( ListTotal );
if( ListTotal.Size() < 3 )                                                  { cout << "Unable to obtain list of features in PartBody." << endl; return 9411; }
...

Setting the 3DObject to Create Opening

In the given model, the 3rd object is the Extrude.1. This number may vary with a change in model.

// ---------------------------------------------
//  Retrieve the 3DObject
// ---------------------------------------------
CATIMmiMechanicalFeature_var sp3DObject = ListTotal[ 3 ];
if( sp3DObject == NULL_var )                                                { cout << "Unable to set the 3DObejct for opening creation." << endl; return 9412; }
sp3DObjectOpening->SetIntersectingElement( sp3DObject );

Selecting the Extrusion Mode

The CATIStrUseOpeningExtrusionMngt interface is used to set the extrusion mode.

CATIStrUseOpeningExtrusionMngt_var spOpeningExtrusion = spOpeningFeat;
if( spOpeningExtrusion == NULL_var )                                        { cout << "Unable to set opening extrusion mode." << endl; return 9413; }
CATStrOpeningExtrusionMode OpeningExtrusionMode = catStrOpeningExtrusionNormal;
spOpeningExtrusion->SetExtrusionMode( OpeningExtrusionMode );

In Short

This use case has demonstrated the way to create 3DObject opening on a SDD plate.
