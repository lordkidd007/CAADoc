What You Will Learn With This Use Case

On this page you will learn how to use the adaptor [3] to create a component. This component adheres to interface CAAISddCreateOpening and creates an opening on a stiffener using Output Profile.

Before Starting

You may want to read from the beginning for creating an opening in SDD [2].

The CAASddCreateSketchOpeningOnProfile Component

This component navigates to the desired stiffener and creates Output Profile opening on this stiffener.

What Does CAASddCreateSketchOpeningOnProfile component do?

This component:

  • Navigates to a desired product in the SpecTree
  • Creates an empty opening
  • Navigates & selects a Output Profile
  • Applies the Output Profile as input to opening

How to Create & Use CAASddCreateSketchOpeningOnProfile Component

The Creating an Opening page describes how to create & use the component.

Where to Find the CAASddCreateSketchOpeningOnProfile Code

The component can be found at the following location:

CAASddCreateOpening.m module of the CAAStructureDetDesign.edu framework:

The component is made up of the following two files:

InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddCreateOpening.m\src\CAASddCreateSketchOpeningOnProfile.cpp
InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddCreateOpening.m\LocalInterfaces\CAASddCreateSketchOpeningOnProfile.h

where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This component deals with the following CAA APIs:

CATIStrUseOpeningOutputProfile Interface on SDD opening created/to create as output profile opening.
CATIStrUseOpeningExtrusionMngt Interface on SDD opening to manage the extrusion direction.

This component uses the following enumerations

CATStrUseOpeningExtrusionMode catStrOpeningExtrusionUndefined
catStrOpeningExtrusionNormal
catStrOpeningExtrusionDirectional
catStrOpeningExtrusionRemoval
catStrOpeningExtrusionClashFree

Step-by-Step

In this use case there are four logical steps in Main:

#Step
1Navigating to Product Node & Creating a Basic Opening

Navigating to Product Node & Creating a Basic Opening

Navigation to desired profile or plate product is done with the help of the adaptor. The method Process of the adaptor Creating an Opening - Adaptor does the actual navigation.

From this component, we merely call the method with parameters as seen below.

...
// ---------------------------------------------
//  Add a new opening and set its basic properties
// ---------------------------------------------
int rc = -1;
CATIMmiMechanicalFeature_var spOpeningFeat, spStiffenerFeat;
CATStrOpeningCreationMode OpeningCreationgMode = catStrOpeningOutputProfile;
rc = CAASddCreateStdOpeningAdpt::Process( ispRootOccurrence, 4, spStiffenerFeat, spOpeningFeat, OpeningCreationgMode );
if( rc )        return rc;

// ---------------------------------------------
//  The Output Profile opening interface
// ---------------------------------------------
CATIStrUseOpeningOutputProfile_var spSketchOpening = spOpeningFeat;
if( spSketchOpening == NULL_var )                                         { cout << "Unable to set 3DObejct." << endl; return 4408; }
...

The above code navigates to the 4th product(which is a transverse stiffener) under the root node & adds an basic opening to this stiffener.

Retrieving the Output Profile

Here, some the APIs from CATMecModLifeUseIts are used to navigate from root product to the Output Profile.

...
// ---------------------------------------------
//  Navigate to the Output Profile
// ---------------------------------------------
// Get SFD Part
CATIMmiMechanicalFeature_var spSFDPart;
CAASddNavigationServices::GetMechanicalPartFromProduct( ispRootOccurrence, spSFDPart );
if( spSFDPart == NULL_var )                                                 { cout << "Unable to navigate to the SFD Part." << endl; return 4409; }
CATIPartRequest_var spPartRequest = spSFDPart;
if( spPartRequest == NULL_var )
                                             { cout << "Unable to navigate to the SFD Part." << endl; return 4409; }
// Get PartBody
CATUnicodeString ucPartBodyView = "MfDefault3DView";
CATBaseUnknown_var spPartBody;
spPartRequest->GetMainBody( ucPartBodyView, spPartBody );
CATIMmiUseBodyContent_var spBodyContent = spPartBody;
if( spBodyContent == NULL_var )                                             { cout << "Unable to navigate to the PartBody." << endl; return 4410; }

// Get Features under PartBody
CATListValCATBaseUnknown_var ListTotal;
spBodyContent->GetMechanicalFeatures( ListTotal );
if( ListTotal.Size() < 1 )                                                  { cout << "Unable to obtain list of features in PartBody." << endl; return 4411; }

// ---------------------------------------------
//  Retrieve the Output Profile
// ---------------------------------------------
CATIMmiMechanicalFeature_var spSketch = ListTotal[ 1 ];
if( spSketch == NULL_var )                                                { cout << "Unable to set the output profile for opening creation." << endl; return 4412; }

CATIMmiUseStructureAnalyse_var spStructureAnalyse = spSketch;
if( spStructureAnalyse == NULL_var )                                      { cout << "Unable to set the output profile for opening creation." << endl; return 4412; }

CATListValCATBaseUnknown_var ListOfChildren;
spStructureAnalyse->GetAllStructuralChildren( CATBaseUnknown::ClassName(), ListOfChildren );
if( ListOfChildren.Size() < 1 )                                           { cout << "Unable to set the output profile for opening creation." << endl; return 4412; }
...

Setting the Output Profile to Create Opening

In the given model, the 1st child is Profile.1. This number may vary with a change in model.

CATIMmiMechanicalFeature_var spProfile;
spProfile = ListOfChildren[ 1 ];
if( spStructureAnalyse == NULL_var )                                      { cout << "Unable to set the output profile for opening creation." << endl; return 4412; }
spSketchOpening->SetOutputProfile( spProfile );

Selecting the Extrusion Mode

The CATIStrUseOpeningExtrusionMngt interface is used to set the extrusion mode.

CATIStrUseOpeningExtrusionMngt_var spOpeningExtrusion = spOpeningFeat;
if( spOpeningExtrusion == NULL_var )                                        { cout << "Unable to set opening extrusion mode." << endl; return 4413; }
CATStrOpeningExtrusionMode OpeningExtrusionMode = catStrOpeningExtrusionNormal;
spOpeningExtrusion->SetExtrusionMode( OpeningExtrusionMode );

This use case has demonstrated the way to create Output Profile opening on a SDD stiffener.
