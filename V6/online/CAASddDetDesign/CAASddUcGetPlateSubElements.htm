What You Will Learn With This Use Case

In this use case, you will learn how to extract sub-elements (faces) of a physical plate.

Before Starting

The physical plate is built using the thickening of the delimited molded surface, after which the thickness compensation is applied on lateral edges. Out of the plate's faces, two of them are remarkable: the "top" one and the "bottom" one.

IMPORTANT: A plate's top and bottom faces depend on the geometry orientation of its molded surface (which is the same orientation its support), and is independent from the throw orientation.

The CAASddGetPlateSubElements Use Case

CAASddGetPlateSubElements.m is a use case of the CAAStructureDetDesignDoc.edu framework that illustrates the CATIA SDD Interfaces framework capabilities.

What Does CAASddGetPlateSubElements Do

The goal of CAASddGetPlateSubElements use case is to show how to get plate sub elements.

Where to Find the CAASddGetPlateSubElements Code

The CAASddGetPlateSubElements sample is made of a single class named CAASddGetPlateSubElements located in the CAASddGetPlateSubElements.m module of the CAAStructureDetDesign.edu framework: InstallRootFolder\CAADoc\CAAStructureDetDesign.edu\CAASddGetPlateSubElements.m\ where InstallRootFolder [1] is the folder where the API CD-ROM is installed.

This sample deals with the following classes:

CATPLMSessionServices Class to create and delete a batch session.
PLMPSISessionHelpers Class managing session data for the creation of PLM entities.
CATAdpDictionaryServices Class to provide all basic functions to browse and navigate on PLM dictionaries.
CATAdpQueryServices Class to manage queries on PLM component in database.
CATAdpOpenSaveServices Class to manage Open and Save in session.
CATISddUseProductPlate Class to retrieve the Product Plate.
CATISddUsePlate Class to retrieve the SddPlate.
CATISddUsePlateSubElementMngt Interface dedicated to get plate sub elements.
CATStrUsePlateFaceName Enum for plate face name.

Getting Plate Sub-Elements

...

 {

 }
...
  // - Get CATISddUsePlateSubElementMngt interface
  CATISddUsePlateSubElementMngt_var spSddPlateSubElementMngt;
  spSddPlateSubElementMngt = ispSddPlate;
  if(NULL_var == spSddPlateSubElementMngt)  
     return hr;
  cout << "CATISddUsePlateSubElementMngt retrieved successfully"<<endl;
  
  // - Get Faces
  CATListPtrCATCell oListOfFaces;
  CATStrPlateFaceName iPlateFaceName = catStrPlateFaceTop;
  hr = spSddPlateSubElementMngt->GetFaces(oListOfFaces, iPlateFaceName);
  if(!SUCCEEDED(hr))  
     return hr;
  cout << "Plate faces retrieved successfully"<<endl;
...

GetFaces: This method is used to retrieve the faces based on top and bottom face of plate. iPlateFaceName can take three values: catStrPlateFaceBottom, catStrPlateFaceTop and catStrPlateFaceNameUndefined. In the case of catStrPlateFaceNameUndefined, all the faces will be retrieved.
