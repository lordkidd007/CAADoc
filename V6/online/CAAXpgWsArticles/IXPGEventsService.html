Interface IXPGEventsService

Services supporting this interface: XPGEventsService

Operations

o GenerateNewAttachmentContent()
Role: Generates an attachment content filled with a unique event path.
The outputed AttachmentContentType will only have its element eventPath filled with a unique path. The rest must be filled when processing a local event before sending it with the SendEvent method.
Parameters:
Name Type Direction Nillable Description
oGeneratedAttachmentContent {http://www.3ds.com/xsd/XPG/Events}AttachmentContentType out true The generated attachment content with a unique eventPath.
Faults:
Name Type Description
ErrorType {http://www.3ds.com/wserror}ErrorType Raised when an error is detected in the web service infrastructure.

o GetListOfSites()
Role: Retrieves the list of sites configured on the XPG gateway controller.
The list will contain at least the "local site" and one or more V6 sites. The local site should be used as the source site and a V6 site as the target site in the SendEvent method.
Parameters:
Name Type Direction Nillable Description
oSites {http://www.3ds.com/xsd/XPG/Events}ListOfSitesType out true The list of sites
Faults:
Name Type Description
ErrorType {http://www.3ds.com/wserror}ErrorType Raised when an error is detected in the web service infrastructure.

o GetNextEventsForProcessing()
Role: Retrieves a list of events that need processing.
Parameters:
Name Type Direction Nillable Description
iNumberRequested {http://www.w3.org/2001/XMLSchema}int in false The maximum number of events that will be delegated to the adapter with this current call. This number must be a positive integer. A zero value will output a null list and the number of events waiting to be processed on the gateway controller.
oNumberLeft {http://www.w3.org/2001/XMLSchema}int out true The number of events still waiting to be processed on the controller (not including the events delegated via the current call)
oDelegatedEvents {http://www.3ds.com/xsd/XPG/Events}ListOfEventsForProcessingType out true The list of delegated events for processing on the adapter. The adapter will have a limited time to process each event, this time out is configured via a gateway controler setting which default value is set to 15 minutes.
Faults:
Name Type Description
ErrorType {http://www.3ds.com/wserror}ErrorType Raised when an error is detected in the web service infrastructure.
o SendEvent()
Role: Sends an event for processing.
If the events needs local processing then you can choose to process before sending by linking the resulting files with an AttachmentContentType object that must be generated with the proper method. If you choose not to process the event then it will be retrieved when calling the GetNextEventsForProcessing method.
Parameters:
Name Type Direction Nillable Description
iSupportedEventName {http://www.3ds.com/xsd/XPG/Events}EventNameType in false Name of the supported events you want to send.
iQueueName {http://www.3ds.com/xsd/XPG/Events}QueueNameType in false Name of the queue in which the event will be queued. It's generally a good idea to give the user who trigerred the action as the queue name. Events in a same queue will be processed one at a time in the chronological order.
iSourceSiteId {http://www.3ds.com/xsd/XPG/Events}SiteIDType in false Id of the local site.
iTargetSiteId {http://www.3ds.com/xsd/XPG/Events}SiteIDType in false Id ot the V6 target site.
iLocallyProcessedEventContent {http://www.3ds.com/xsd/XPG/Events}AttachmentContentType in true If the event needs local processing it could be processed before sending to the V6 controller. The AttachmentContentType should be generated using the GenerateNewAttachmentContent method and filled with the process resulting files.
iProcessingArgs {http://www.3ds.com/xsd/XPG/Events}ListOfProcessingArgsType in true A list of name/value arguments sometime necessary for the processing.
iDescription {http://www.w3.org/2001/XMLSchema}string in true A string giving some context information about this particular event
oEventId {http://www.3ds.com/xsd/XPG/Events}EventUIDType out true The unique ID of the sent event. It can allow you to track its states using GetEventState.
Faults:
Name Type Description
ErrorType {http://www.3ds.com/wserror}ErrorType Raised when an error is detected in the web service infrastructure.
