Changed C++ APIs in V6R2013 GA

The classes and interfaces listed in Changed C++ APIs in V6R2013 GA that were in the DraftingUseItf framework are moved from V6R2013 to the new CATAnnotationUseItf framework. If you use some of these classes or interfaces:

  • Add the CATAnnotationUseItf framework in the identity cards of the frameworks using them.
  • Add the CATAnnotationUseItf module in the LINK_WITH section of the Imakefile.mk files of the modules using them.

None.
