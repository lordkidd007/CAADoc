C++ APIs

All CAA methods/classes which deal with finite element nodes (CATFmtNode) and elements (CATFmtElement) are marked as deprecated. As the size of finite element meshes will increase more and more, an approach where the entire mesh is loaded with C++ objects representing each individual nodes and elements will not scale in terms of memory consumption. Therefore a new set of methods/classes will be developed and made available through CAA. You should try to minimize usage of these deprecated methods as it is likely that new APIs will require significant changes in existing code.
