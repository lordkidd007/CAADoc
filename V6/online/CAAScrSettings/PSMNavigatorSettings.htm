PSMNavigatorSettings Setting Repository
Quick Reference

PSMNavigatorSettings

Role: Manage navigation settings

Detailed role:Define priority order to retain to create the navigation representations at simulation root node and at category nodes

The PSMNavigatorSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/CATPSMNavigatorWkb/SLMNavigator
  • Repository File: PSMNavigatorSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

PSMNavigatorSettings setting repository attributes:

  • CategoryActiveNodeList
    Role: Define activation of items in previous list
    Detailed role: This attribute is the ordered list, specifies the activation status to retain to create the navigation representations. Note that this array of 6 values must contain activate or deactivate value
    • Type: string
    • Default value: Activated
    • Default Value Role: CategoryNodeList activation (Activated/Deactivated)
  • CategoryNodeList
    Role: search order for creating the 3D display resources at category level
    Detailed role: This attribute is the ordered list, specifies the priority to retain to create the navigation representations. Note that this array of 6 values must not contain any duplicates
    • Type: string
    • Default value: VPMReference
    • Default Value Role: Default order is Category, SimuRepV5, MeshRep, ShapeRep, SimuRepNotV5
  • SimulationActiveNodeList
    Role: Define activation of items in previous list
    Detailed role: This attribute is the ordered list, specifies the activation status to retain to create the navigation representations. Note that this array of 7 values must contain activate or deactivate value
    • Type: string
    • Default value: Activated
    • Default Value Role: SimulationNodeList activation (Activated/Deactivated)
  • SimulationNodeList
    Role: search order for creating the 3D display resources at simulation level
    Detailed role: This attribute is the ordered list, specifies the priority to retain to create the navigation representations. Note that this array of 7 values must not contain any duplicates
    • Type: string
    • Default value: dsc_Model_Category_Ref
    • Default Value Role: Default order is Simulation, CategoryResult, CategorySpecs, CategoryContext
