ElectricalWiringDesign Setting Repository
Quick Reference

ElectricalWiringDesign

Role: Electrical Wire Routing settings

Detailed role:Electrical Wire Routing settings

The ElectricalWiringDesign repository has the following characteristics:

  • Primary Tab Page: /Equipments/Electrical Wire Routing/General
  • Repository File: ElectricalWiringDesign
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalWiringDesign setting repository attributes:

  • BendRadiusMode
    Role: Bend radius compute mode
    Detailed role: Bend radius compute mode
    • Type: string
    • Default value: Always
  • CheckCreationMode
    Role: Branchable Creation Mode
    Detailed role: Branchable Bend Radius to be computed based on BRR and max. Diameter of segment after wire routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default do not take BRR into account
    • Values:
      0 Do not take BRR into account
      1 Take BRR into account
  • ComputeAlternateRoute
    Role: Compute alternate path based on 'EWR_ValidateSegmentInConductorRoute' result
    Detailed role: If checked, failure of business rule 'EWR_ValidateSegmentInConductorRoute' will trigger computation of alternate route
    • Type: integer
    • Default value: 0
    • Default Value Role: By default an alternate route is not computed
    • Values:
      0 Alternate route is not computed
      1 Alternate route is computed
  • ComputeFromWire
    Role: Wire taken into accound
    Detailed role: Check wether or not wire are taken into account for segment update
    • Type: integer
    • Default value: 1
    • Default Value Role: By default take Wire into account
    • Values:
      0 Take Wire into account
      1 Do not take Wire into account
  • CorrendEnds
    Role: Ensure route path extremities matches the conductor ends(if defined till ports)
    Detailed role: If checked, route will be found via segment connected to the correct port to which the conductor ends
    • Type: integer
    • Default value: 0
    • Default Value Role: By default path found may not be from the segment connected to correct port
    • Values:
      0 Path found may not be from the segment connected to correct port
      1 Path found is via the segment connected to correct port
  • CurrentSegregationIndex
    Role: Index of the selected paths of the Segregation file list
    Detailed role: Paths of the Segregation file containing a table of separation code
    • Type: integer
    • Default value: 0
  • ForbidBRValidation
    Role: Forbid business rule validation in manual routing and copy route commands
    Detailed role: If checked, business rules will not be checked for manual routing and copy route commands
    • Type: integer
    • Default value: 0
    • Default Value Role: By default business rules are activated
    • Values:
      0 Business rules are activated for manual routing and copy route commands
      1 Business rules are deactivated for manual routing and copy route commands
  • HandleRule
    Role: Mode Rule activated
    Detailed role: Allows to activated the routing mode with compatibility table. The compatibility table is described in a txt file containing a table of separation code
    • Type: integer
    • Default value: 1
  • HandleSigSigSepCode
    Role: Compatibility mode
    Detailed role: The compatibility between wires or wires and segment can be checked according to this setting
    • Type: integer
    • Default value: 0
  • HandleSource
    Role: Compatibility table file source
    Detailed role: The source of compatibility table files can be selected according to this setting
    • Type: integer
    • Default value: 0
  • IsPropagationActivated
    Role: allow propagation of conductor attributes from conductor reference to conductor feature (same for Conductor Group)
    Detailed role: If checked during the mapping, attributes from conductor reference are propagated in conductor feature (same for conductor group attributes)
    • Type: integer
    • Default value: 0
    • Default Value Role: By default propagation is not activated
    • Values:
      0 Propagation is not activated
      1 Propagation is activated
  • RatioAdded
    Role: Ratio to add to equivalent area in percentage
    Detailed role: Setting to manage the ratio to add to equivalent area in percentage for diameter computation
    • Type: double
    • Default value: 15
    • Default Value Role: By default the tolerance is set to 20 percent
  • SegregationCount
    Role: Size of the path list of Segregation file
    Detailed role: Size of the path list of Segregation file
    • Type: integer
    • Default value: 0
  • SegregationTable
    Role: Paths of the Segregation file list
    Detailed role: Paths of the Segregation files list containing a table of separation code
    • Type: string
    • Default value:
  • SegregationTableFileName
    Role: List of the names of Segregation files selected from PRM
    Detailed role: List of the names of Segregation files selected from PRM
    • Type: string
    • Default value:
  • SegregationTableFilePath
    Role: List of the paths of Segregation files selected from PRM
    Detailed role: List of the paths of Segregation files selected from PRM
    • Type: string
    • Default value:
  • SpecTreeVisuOption
    Role: Option to customize the spec tree visualization under electrical physical system
    Detailed role: Allows to toggle the specification tree visualization between resolution status and route status
    • Type: integer
    • Default value: 1
