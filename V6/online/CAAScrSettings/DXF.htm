DXF Setting Repository
Quick Reference

DXF

Role: Handling of DXF Data Exchange Import and Export parameters

Detailed role:Handling of DXF Data Exchange Import and Export parameters

The DXF repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/DXF
  • Repository File: DXF
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 2
  • Root tab page /x reused as /x

DXF setting repository attributes:

  • ExportBlockMode
    Role: Export Blocks mode
    • Type: integer
    • Default value: 1
    • Values:
      0 Export Blocks None.
      1 Export Blocks One level.
      2 Export Blocks Full level.
  • ExportDimensionsAsDimensions
    Role: Export dimensions as dimensions
    • Type: boolean
    • Default value: false
  • ExportDxfStandard
    Role: name of the Dxf standard, which define the different mapping used at export
    • Type: string
    • Default value: DXF
  • ExportLayerName
    Role: Export Layer Name
    • Type: boolean
    • Default value: false
  • ExportMode
    Role: Export Mode semantic or graphic
    • Type: integer
    • Default value: 1
    • Values:
      0 Export semantic.
      1 Export graphic.
  • ExportSheets
    Role: Which sheets to export
    • Type: integer
    • Default value: 0
    • Values:
      0 Export All sheets. One File is generated per none empty sheet.
      1 Export only the current sheet in a single file.
  • ExportVersion
    Role: Which Export Version
    • Type: integer
    • Default value: 3
    • Values:
      0 Export a DXF or DWG with a format AutoCAD R12.
      1 Export a DXF or DWG with a format AutoCAD R13.
      2 Export a DXF or DWG with a format AutoCAD R14.
      3 Export a DXF or DWG with a format AutoCAD 2000 (same as 2000i and 2002).
      4 Export a DXF or DWG with a format AutoCAD 2004 (same as 2005 and 2006).
      5 Export a DXF or DWG with a format AutoCAD 2007 (same as 2008).
  • ImportDftStandard
    Role: name of the Drafting standard that defines the standard of the resulting Drawing
    • Type: string
    • Default value: ISO
  • ImportDimensions
    Role: How to import dimensions
    • Type: integer
    • Default value: 2
    • Values:
      0 Convert dimensions as dimensions
      1 The Convert dimensions as details
      2 Convert dimensions as geometry
  • ImportDxfStandard
    Role: name of the Dxf standard, which define the different mapping used at import
    • Type: string
    • Default value: DXF
  • ImportEndPoints
    Role: When create end points
    • Type: integer
    • Default value: 0
    • Values:
      0 Never create end points.
      1 Create end points only for few entities.
      2 Always create end points.
  • ImportKeepModelSpace
    Role: Keep the Model Space even if referenced by Viewports
    • Type: boolean
    • Default value: false
  • ImportMapLayerOn2DL
    Role: Map Layer on 2DLlayout Sheet
    • Type: boolean
    • Default value: true
  • ImportMapViewInDetail
    Role: How to import views
    • Type: integer
    • Default value: 0
    • Values:
      0 Convert vieus as viewport
      1 The Convert views as details
  • ImportPaperSpacesInBackground
    Role: Put the Paper Spaces in Background View
    • Type: boolean
    • Default value: false
  • ImportScaleDenominator
    Role: Get the denominator value of the scale factor. That value is only taken into account when the import unit is set to Scale Factor.
    • Type: double
    • Default value: 1.0
  • ImportScaleNumerator
    Role: Get the numerator value of the scale factor. That value is only taken into account when the import unit is set to Scale Factor.
    • Type: double
    • Default value: 1.0
  • ImportUnit
    Role: the unit of the input file
    • Type: integer
    • Default value: 6
    • Values:
      0 The unit of the input file is Millimeter
      1 The unit of the input file is Centimeter
      2 The unit of the input file is Meter
      3 The unit of the input file is Inch
      4 The unit of the input file is Foot
      5 The unit of the input file is given by a ScaleFactor applied to Millimeter
      6 The unit of the input file is determined Automatically
