3DShapeInfrastructure3DShape Setting Repository
Quick Reference

3DShapeInfrastructure3DShape

Role: Setting to manage some 3D Shape Infrastructure options

The 3DShapeInfrastructure3DShape repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/3D Shape
  • Repository File: PartDocument
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructure3DShape setting repository attributes:

  • GenerateDynamicTesselation
    Role: Determines if generating dynamic tesselation at propagate is active
    • Type: boolean
    • Default value: false
  • GenerateDynamicTesselationOpen
    Role: Determines if generating dynamic tesselation at open is active
    • Type: boolean
    • Default value: false
  • HybridDesignMode
    Role: Enable hybrid design inside part bodies and bodies
    Detailed role: Enable to mix wireframe, surfacic and solid designs in part bodies and bodies.
    • Type: boolean
    • Default value: true
  • KnowledgeInHybridDesignMode
    Role: Enable hybrid design for parameters and relations inside part bodies and bodies
    • Type: boolean
    • Default value: false
  • NewWith3DSupport
    Role: Create a 3D working support
    Detailed role: Determines if a new 3D shape will be created with a 3D working support.
    • Alias: CreateWorkingSupportMode
    • Type: boolean
    • Default value: false
  • NewWithAxisSystem
    Role: Create an axis system
    Detailed role: Determines if a new 3D shape will be created with an axis system.
    • Alias: CreateAxisSystemMode
    • Type: boolean
    • Default value: false
  • NewWithGS
    Role: Create a geometrical set
    Detailed role: Determines if a new 3D shape will be created with a geometrical set.
    • Alias: CreateOpenBodyMode
    • Type: boolean
    • Default value: false
  • NewWithOGS
    Role: Create an ordered geometrical set
    Detailed role: Determines if a new 3D shape will be created with an ordered geometrical set.
    • Alias: CreateOGSMode
    • Type: boolean
    • Default value: false
  • SurfaceElementsLocation
    Role: Determines where wireframe and surface elements are created when hybrid design is active
    • Alias: WireframeComponentCreationMode
    • Type: short integer
    • Default value: 0
    • Values:
      0 In a solid set
      Detailed role: Wireframe and surface elements are created within a solid set.
      1 In a geometrical set
      Detailed role: Wireframe and surface elements are created within a geometrical or ordered geometrical set.
