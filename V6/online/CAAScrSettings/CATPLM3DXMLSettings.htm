CATPLM3DXMLSettings Setting Repository
Quick Reference

CATPLM3DXMLSettings

Role: Setting to handle CAA model parameters

The CATPLM3DXMLSettings repository has the following characteristics:

  • Primary Tab Page: /General/3DXML
  • Repository File: CATPLM3DXMLSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPLM3DXMLSettings setting repository attributes:

  • AuthorizationStatus
    Role: Role Visibility
    • Type: string
    • Default value: Refuse
    • Values:
      Authorize The data degradation is authorized
      Refuse The data degradation is refused
  • ConflictManagementDecision
    Role: Role Visibility
    • Type: string
    • Default value: TakeIncoming
    • Values:
      KeepLocal The local data are kept and the new revision is ignored
      TakeIncoming The local data are overwritten by the new revision
  • ConflictManagementOption
    Role: Manage conflict error at import
    • Type: boolean
    • Default value: false
  • GeometricFormats
    Role: Role Visibility
    • Type: string
    • Default value: StaticTesselation
    • Values:
      XMLTesselation The geometric format used is XML tesselation
      StaticTesselation The geometric format used is Static tesselation
      ThirdTesselation The geometric format used is Third tesselation
  • MLCoption
    Role: Allow data degradation at import
    • Type: boolean
    • Default value: false
