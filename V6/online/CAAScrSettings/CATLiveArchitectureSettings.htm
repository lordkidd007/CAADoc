CATLiveArchitectureSettings

Role: These settings define the user choice for CATIA Live Architecture environment

Detailed role:These settings define the user choice of the display the referent who allows you to get a better idea of the scale factor.

The CATLiveArchitectureSettings repository has the following characteristics:

  • Primary Tab Page: /Shape/CATIA Live Architecture/Display
  • Repository File: CATLiveArchitectureSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
