CATBisBclOptions Setting Repository
Quick Reference

CATBisBclOptions

Role: Options for BioSystems Calibration

Detailed role:These options allow user to customize the behavior of BioSystems Calibration

The CATBisBclOptions repository has the following characteristics:

  • Primary Tab Page: /Simulation/BioSystems Experiment/Calibration
  • Repository File: CATBisBclOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATBisBclOptions setting repository attributes:

  • BclMaxNumberOfSimulations
    Role: Maximum Number of Simulations
    • Type: integer
    • Default value: 10000
  • BclOptimizationMethod
    Role: Optimization Method
    • Type: string
    • Default value: PSO
    • Values:
      PSO PSO algorithm
      GlobalNelderMead GlobalNelderMead algorithm
      PSONelderMead PSONelderMead algorithm
  • BclWantedNumberOfParametersSet
    Role: Wanted Number of Parameters Set
    • Type: integer
    • Default value: 3
