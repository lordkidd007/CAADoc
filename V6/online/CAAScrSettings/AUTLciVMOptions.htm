AUTLciVMOptions Setting Repository
Quick Reference

AUTLciVMOptions

Role: Logic simulation options

Detailed role:Logic simulation options

The AUTLciVMOptions repository has the following characteristics:

  • Primary Tab Page: /Automation/Simulation
  • Repository File: AUTLciVMOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: AUTLciVM
    • Framework_IDL: AUTLciVM
    • Interface_CPP: AUTILciVMSettingAtt
    • Interface_IDL: AUTILciVMSettingAtt
    • LateType: CATSysSettingCtrl

AUTLciVMOptions setting repository attributes:

  • ArrowBackgroundColor
    Role: Arrow Background color in debug tab
    • Type: integer
    • Default value: 16776960
  • ArrowForegroundColor
    Role: Arrow Foreground color in debug tab
    • Type: integer
    • Default value: 0
  • CheckDoubleEmission
    Role: Detect double emissions during simulation
    • Type: integer
    • Default value: 0
  • DebugLineThickness
    Role: Debug line thickness
    • Type: integer
    • Default value: 1
    • Min value: 1
    • Max value: 6
  • GcMajorHeapIncrement
    Role: CAML Garbage collector max heap increment
    • Type: integer
    • Default value: 524288
  • GcMaxOverhead
    Role: CAML Garbage collector max space overhead
    • Type: integer
    • Default value: 500
  • GcMinorHeapSize
    Role: CAML Garbage collector min heap size
    • Type: integer
    • Default value: 32768
  • GcSpaceOverhead
    Role: CAML Garbage collector space overhead
    • Type: integer
    • Default value: 80
  • GcVerbose
    Role: CAML Garbage collector verbosity
    • Type: integer
    • Default value: 0
  • ObjectDbgBackgroundColor
    Role: Object Background Color in debug
    • Type: integer
    • Default value: 526344
  • ObjectDbgForegroundColor
    Role: Object Foreground Color in debug
    • Type: integer
    • Default value: 255
  • ObjectDbgHasBackgroundColor
    Role: Use default background for objects in debug.
    • Type: integer
    • Default value: 0
  • SimulationManagerRefresh
    Role: Refresh rate in ms.
    • Type: integer
    • Default value: 400
