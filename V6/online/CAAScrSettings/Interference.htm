Interference Setting Repository
Quick Reference

Interference

Role: Setting to handle the Interference parameters

The Interference repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/Interference
  • Repository File: Interference
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Interference setting repository attributes:

  • ColorInterferrringPart1
    Role: Select color for first interferring part
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • ColorInterferrringPart2
    Role: Select color for second interferring part
    • Tab Page: /Digital Mockup/Interference
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Yellow in RGB order
    • Min value: 0
    • Max value: 255
  • CustomizeDisplay
    Role: Display Customized instance names in multi list
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
  • DrawInterferenceGeometry
    Role: Dynamically draw interference geometry
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
  • ItfIcon
    Role: Display Interference Icon
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
  • MinDistance
    Role: Dispaly minimum distance clearance
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
  • PenetrationVector
    Role: Display penetration vector
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 0
  • ReportPath
    Role: Save HTML report on disk
    • Tab Page: /Digital Mockup/Interference
    • Type: URN
    • Default value:
  • ReportSave
    Role: Save HTML Report into VPM
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
  • ToolTip
    Role: Display Tooltip
    • Tab Page: /Digital Mockup/Interference
    • Type: integer
    • Default value: 1
