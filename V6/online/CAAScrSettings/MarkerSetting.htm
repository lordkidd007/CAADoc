MarkerSetting Setting Repository
Quick Reference

MarkerSetting

Role: Setting to handle the markers visual properties and parameters

The MarkerSetting repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/General
  • Repository File: Marker
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MarkerSetting setting repository attributes:

  • MarkerAutoCustomiseMode
    Role: To enable or disable the auto customization of markers visual properties
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: true
  • MarkerDefaultColor
    Role: Default color of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • MarkerDefaultDashed
    Role: Default dashed value of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 1
    • Min value: 1
    • Max value: 7
  • MarkerDefaultWeight
    Role: Default weight value of a marker
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 2
    • Default Value Role: 1 mm
    • Min value: 1
    • Max value: 55
  • MarkerDocMode
    Role: Document Management
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • MarkerFillColor
    Role: Default color to fill
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: >White in RGB order
    • Min value: 0
    • Max value: 255
  • MarkerFillTransparency
    Role: Default transparency value for fill
    • Tab Page: /Digital Mockup/Markers
    • Type: unsigned integer
    • Default value: 190
    • Default Value Role: Filling is transparent(75%)
    • Min value: 0
    • Max value: 255
  • MarkersFillMode
    Role: Default flag for filling enabling
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: false
  • MarkerUpdateMode
    Role: Update on product structure modifications
    • Tab Page: /Digital Mockup/Markers
    • Type: boolean
    • Default value: true
