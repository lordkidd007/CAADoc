KPISettings Setting Repository
Quick Reference

KPISettings

Role: Settings to handle the KPIs parameters

The KPISettings repository has the following characteristics:

  • Primary Tab Page: /General/KPI
  • Repository File: KPISettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

KPISettings setting repository attributes:

  • DefaultMarginPercentageOnDeclared
    Role: Default Margin percentage On Declared (%)
    • Tab Page: /General/KPI/General
    • Type: double
    • Default value: 10
  • DefaultPurcentageOnDeclared
    Role: Default Confidence percentage On Declared (%)
    • Tab Page: /General/KPI/General
    • Type: double
    • Default value: 10
  • MaskFile
    Role: Describe the mask attributes for publishing
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value:
  • PolicyName
    Role: Policy name
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value: Universal
  • PublishedDirectory
    Role: Describe the directory hosting the published reports
    • Tab Page: /General/KPI/General
    • Type: string
    • Default value:
  • PurcentageSwitch
    Role: Estimated over Real switch confidence threshold (%)
    • Tab Page: /General/KPI/General
    • Type: double
    • Default value: 70
