CATStatisticsActivity Setting Repository
Quick Reference

CATStatisticsActivity

Role: Settings for Activity Statistics

Detailed role:The Activity statistics controler manages the values of all or only a part of the attributes available for the thematic

The CATStatisticsActivity repository has the following characteristics:

  • Primary Tab Page: /General/General/Statistics
  • Repository File: CATStatistics
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /General/Activity Statistics Management reused as /ProductStructure/Activity Statistics Management

CATStatisticsActivity setting repository attributes:

  • ACTIVITY_Activation
    Role: Activation state of the Activity statistics
    Detailed role: Value of Activity statistics activation
    • Type: integer
    • Default value: 0
  • ACTIVITY_CPUS
    Role: If activated, the cpu duration field is outputed.
    • Type: boolean
    • Default value: true
  • ACTIVITY_Cumulate
    Role: Cumulation mode state of the Activity statistics
    Detailed role: Value of Activity statistics cumulation mode
    if false there is one file per session
    if true all sessions are outputted in one file
    • Type: boolean
    • Default value: true
  • ACTIVITY_DateFormat
    Role: Format for the date and time field.
    • Type: integer
    • Default value: 1
    • Values:
      1 standard : Mon Jan 1 08:00.00 2000
      2 numerical : 2000.001.08.00.00
      3 numerical with millisecond : 2000.001.08.00.00.000
  • ACTIVITY_ELPS
    Role: If activated, the elapsed duration field is outputed.
    • Type: boolean
    • Default value: true
  • ACTIVITY_Formating
    Role: Format of the Activity statistics
    • Type: integer
    • Default value: 1
    • Values:
      1 Standart output format (ie : field="value")
      2 Field is not outputed (ie : "value" )
      3 Common Base Event (CBE) format
      4 WMI EventLog integration
  • ACTIVITY_HOST
    Role: If activated, the host name field is outputed.
    • Type: boolean
    • Default value: false
  • ACTIVITY_OutputDirectory
    Role: Output directory for the statistic
    • Type: string
    • Default value: default_directory
  • ACTIVITY_OutputFile
    Role: Name of the output file
    • Type: string
    • Default value: default_file
  • ACTIVITY_OutputFormat
    Role: Output format of the Activity statistics
    • Type: integer
    • Default value: 1
    • Values:
      1 standard output
      2 the thematic name is not repeated on each line
  • ACTIVITY_OutputType
    Role: Format of the Activity statistics
    • Type: integer
    • Default value: 2
    • Values:
      1 Console : log is outputed in the console (not use)
      2 File : log is outputed in a file
  • ACTIVITY_RTIM
    Role: If activated, the response time field is outputed.
    • Type: boolean
    • Default value: true
  • ACTIVITY_THEM
    Role: If activated, the thematic name field is outputed.
    • Type: boolean
    • Default value: true
  • ACTIVITY_TIME
    Role: If activated, the time field is outputed.
    • Type: boolean
    • Default value: true
  • ACTIVITY_TimeUnit
    Role: Time unit for the cpu, elaped and response time.
    • Type: integer
    • Default value: 1
    • Values:
      1 millisecond
      2 second
  • ACTIVITY_UPID
    Role: If activated, the process id field is outputed.
    • Type: boolean
    • Default value: false
  • ACTIVITY_USER
    Role: If activated, the user name field is outputed.
    • Type: boolean
    • Default value: false
  • Version
    Role: Internal use - statistics settings version level.
    • Type: integer
    • Default value: 1
