Quick Reference

AerospaceSheetmetalDesign

Role: These settings define the user choice of the display of characteristic curves of ASL features

Detailed role:These settings define the user choice of the display of characteristic curves of ASL features : control of the display may be set independantly in each of the folded and flatten 3D views.

The AerospaceSheetmetalDesign repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/AerospaceSheetmetal Design/Display
  • Repository File: AerospaceSheetmetalDesign
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x
