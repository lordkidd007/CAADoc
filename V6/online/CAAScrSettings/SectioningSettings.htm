Quick Reference

SectioningSettings

Role: Setting to handle the Sectioning parameters

The SectioningSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Mockup/DMU Space Analysis/DMU Sectioning
  • Repository File: SectioningRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SectioningSettings setting repository attributes:

  • OnlyPlaneDynMode
    Role: Default mode for only plane dynamic
    • Type: integer
    • Default value: 0
    • Values:
      0 Do NOT use only plane dynamic mode
      1 Use only plane dynamic mode
  • SectionInfoDispMode
    Role: Default mode for Display Section Information
    • Type: integer
    • Default value: 1
    • Values:
      0 Do NOT display section information in 3D
      1 Display Section information in 3D
  • SectionMode
    Role: Default choice for the main viewer of section result
    • Type: integer
    • Default value: 4
    • Values:
      0 Borders
      1 Fill
      2 Clash2D
      3 Cross View
      4 Cut View
