StructureDisplaySetting Setting Repository
Quick Reference

StructureDisplaySetting

Role: Setting to manage Pass through Points and Lines

Detailed role:Pass through Points and Lines for all structural standard Openings

The StructureDisplaySetting repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSldPLMRepConfig/Display
  • Repository File: StructureDisplaySetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

StructureDisplaySetting setting repository attributes:

  • CreateOption
    Role: Option to create Point and Lines
    Detailed role: Option to create or Not Point and Lines
    • Type: short integer
    • Default value: 1
    • Default Value Role: CreateOption
  • GPColor
    Role: The Color of GP
    Detailed role: The GP Color
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red
    • Min value: 0
    • Max value: 255
  • GPLength
    Role: The Length of GP
    Detailed role: The GP Length
    • Type: double
    • Default value: 70
    • Default Value Role: GPLength
  • GPRadius
    Role: The Radius of GP
    Detailed role: The GP Radius
    • Type: double
    • Default value: 15.0
    • Default Value Role: GPRadius
  • LineLength
    Role: The Length of Line
    Detailed role: The Line Length
    • Type: double
    • Default value: 250
    • Default Value Role: LineLength
  • LineType
    Role: The Opening OriginLine Type
    Detailed role: The Originline type
    • Type: integer
    • Default value: 1
    • Default Value Role: LineType
  • PointPattern
    Role: Get the Opening OriginPoint Pattern
    Detailed role: The OriginPoint pattern
    • Type: short integer
    • Default value: 1
    • Default Value Role: PointPattern
