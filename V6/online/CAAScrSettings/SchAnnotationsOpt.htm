SchAnnotationsOpt Setting Repository
Quick Reference

SchAnnotationsOpt

Role: Setting to handle the Annotations options in schematic applications"

Detailed role:This setting allows to customize the appearance and the behavior of the Drafting Annotations User Interface in Schematic Applications

The SchAnnotationsOpt repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/RFLP Editor/RFLP Editor Properties
  • Repository File: SchAnnotationsOpt
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
