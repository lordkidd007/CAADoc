DNBDeviceBase Setting Repository
Quick Reference

DNBDeviceBase

Role: Setting Device visualization parameters

Detailed role:Visualization of Device analysis parameters. They are Travel Limit, Velocity Limit, Acceleration Limit and Caution Zone.

The DNBDeviceBase repository has the following characteristics:

  • Primary Tab Page: /Delmia Infrastructure/Device Analysis
  • Repository File: DNBDeviceBase
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /x reused as /x

DNBDeviceBase setting repository attributes:

  • AccelerationLimit
    Role: show acceleration limit with one of following options:Off(not show), Highlight,Verbose and Interrupt.
    • Alias: AccelLimit
    • Type: integer
    • Default value: 0
    • Default Value Role: Off
    • Values:
      0 Off
      1 Highlight
      2 Verbose
      3 Interrupt
  • AccelerationZone
    Role: Short Qualification of the Attribute
    • Alias: DOF_Accel_Color
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White color
    • Min value: 0
    • Max value: 255
  • AxisSafetyUnsafeColor
    Role: Robot color change when TCP is in an unsafe condition
    • Alias: Axis_SafetyUnsafe_Color
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Blue color
    • Min value: 0
    • Max value: 255
