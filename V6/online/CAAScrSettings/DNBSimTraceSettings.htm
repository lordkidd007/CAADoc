DNBSimTraceSettings Setting Repository
Quick Reference

DNBSimTraceSettings

Role: Settings to handle the parameters of Motion Traces

Detailed role:This setting allows to customize the appearance and some behavior of motion traces

The DNBSimTraceSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/DELMIA Infrastructure/Motion Trace
  • Repository File: DNBSimTraceSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBSimTraceSettings setting repository attributes:

  • SimTraceAxisColor
    Role: Color of the axis system at the points of a Motion Trace
    Detailed role: Color of the axis system at the points of the Motion Trace
    • Type: unsigned integer
    • Default value: 255
  • SimTraceAxisThickness
    Role: Thickness of the axis system at the points of the Motion Trace
    Detailed role: Thickness of the axis system at the points of the Motion Trace
    • Type: unsigned integer
    • Default value: 1
  • SimTraceAxisType
    Role: Symbol used to represent the axis system at the points of a Motion Trace
    Detailed role: Different Symbols can be used to represent the axis system
    • Type: unsigned integer
    • Default value: 1
  • SimTraceAxisVisu
    Role: Visbility of the axis system at the points of Motion Trace
    Detailed role: Whether the axis system at the points of Motion Trace is visibile or not
    • Type: unsigned integer
    • Default value: 0
  • SimTraceDeletePath
    Role: Whether or not to delete the motion trace when product motion starts again
    Detailed role: Whether or not to delete the motion trace when product motion starts again
    • Type: unsigned integer
    • Default value: 1
  • SimTraceLegend
    Role: Whether or not to show a legent next to each motion trace point
    Detailed role: Legend can include any of the following: Name, Time, X, Y, Z, Yaw, Pitch, Roll
    • Type: unsigned integer
    • Default value: 0
  • SimTraceLineColor
    Role: Color of the line connecting the points of a Motion Trace
    Detailed role: Color of the line connecting the points of a Motion Trace
    • Type: unsigned integer
    • Default value: 255
  • SimTraceLineThickness
    Role: Thickness of the line connecting the point of the Motion Trace
    Detailed role: Thickness of the line connecting the point of the Motion Trace
    • Type: unsigned integer
    • Default value: 1
  • SimTraceLineType
    Role: Symbol used to represent the line connecting the points of a Motion Trace
    Detailed role: Different Symbols can be used to represent a line
    • Type: unsigned integer
    • Default value: 1
  • SimTraceLineVisu
    Role: Visbility of the line connecting the points of Motion Trace
    Detailed role: Whether the line connecting the points of Motion Trace is visibile or not
    • Type: unsigned integer
    • Default value: 1
  • SimTracePointColor
    Role: Color of the Motion Trace Points
    Detailed role: Color of the Motion Trace Points
    • Type: unsigned integer
    • Default value: 255
  • SimTracePointType
    Role: Symbol used to represent the point
    Detailed role: Different Symbols can be used to represent a point
    • Type: unsigned integer
    • Default value: 1
  • SimTracePointVisu
    Role: Visbility of the Motion Trace Point
    Detailed role: Whether the point is visibile or not
    • Type: unsigned integer
    • Default value: 0
