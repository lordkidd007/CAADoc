VPMNavigator Setting Repository
Quick Reference

VPMNavigator

Role: Setting to handle the parameters of the VPM Navigator User Interface

Detailed role:This setting allows to customize the appearance and the behavior of the VPM Navigator User Interface

The VPMNavigator repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/VPM Navigator/Reference
  • Repository File: VPMNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
