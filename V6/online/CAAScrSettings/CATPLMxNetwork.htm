CATPLMxNetwork Setting Repository
Quick Reference

CATPLMxNetwork

Role: Network Compresssion activation definition

Detailed role:Controls the compression of the data when sent over the Network, Network compression can be controlled only for one Data Source.

The CATPLMxNetwork repository has the following characteristics:

  • Primary Tab Page: /General/Connection
  • Repository File: CATPLMxNetwork
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPLMxNetwork setting repository attributes:

  • DataCompressionActivated
    Role: Defines whether or not the compression mode is activated.
    • Type: boolean
    • Default value: false
  • FCSDataCompressionActivated
    Role: Defines whether or not the compression mode of FCS is activated for download operations.
    • Type: boolean
    • Default value: false
  • FCSDataCompressionUploadActivated
    Role: Defines whether or not the compression mode of FCS is activated for upload operations.
    • Type: boolean
    • Default value: false
  • FCSOverrideCloud
    Role: Internal Use. Let Override Cloud Settings
    • Type: boolean
    • Default value: false
