3DShapeInfrastructureCompatibility Setting Repository
Quick Reference

3DShapeInfrastructureCompatibility

Role: Settings to manage 3D shape infrastructure compatibility options

The 3DShapeInfrastructureCompatibility repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/Compatibility
  • Repository File: part
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructureCompatibility setting repository attributes:

  • V56R2013Compliant
    Role: Defines if the features created in 3D Shapes should be compliant to V5-6R2013 or should be compliant to V5-6R2012
    • Type: boolean
    • Default value: true
