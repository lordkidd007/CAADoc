ABFDrafting Setting Repository
Quick Reference

ABFDrafting

Role: Defines the ABF spot fastener drafting extraction parameters

Detailed role:Enable the user to specify the Spot Fastener Drafting Extraction parameters

The ABFDrafting repository has the following characteristics:

  • Primary Tab Page: /Shape/AutomotiveBiWFastening/Drafting
  • Repository File: ABFDrafting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ABFDrafting setting repository attributes:

  • AngularTolerance
    Role: Setting the maximum angle between the spot normal direction and the drafting view direction for which the spot fastener drafting extraction is possible
    • Type: double
    • Default value: 90.0
  • FixedSpotSize
    Role: Indicate the fixed drawing symbol size (available only if Fixed spot size Checked is yes)
    • Type: double
    • Default value: 0.0
  • FixedSpotSizeCheck
    Role: : Indicate whether the drawing symbol size is fixed for all the spot fasteners OR varies as per the spot fastener diameter during the fastener drafting extraction process
    • Type: boolean
    • Default value: false
  • ReverseDirection
    Role: Indicate whether to include or exlude the spot fasteners in the opposite direction of view direction during spot fastener drafting extraction process
    • Type: boolean
    • Default value: true
