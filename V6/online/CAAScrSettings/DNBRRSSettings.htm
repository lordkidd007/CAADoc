DNBRRSSettings Setting Repository
Quick Reference

DNBRRSSettings

Role: RRS Options

Detailed role:These options allow customization of RRS behavior

The DNBRRSSettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Robotics/RRS
  • Repository File: DNBRRSSettings
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBRRSSettings setting repository attributes:

  • RCSDATALIBDirectory
    Role: RCSDATA$LIB Directory
    Detailed role: Directory (list) of RCS Module Data File Home locations
    • Type: URN
    • Default value:
  • RRS2FreePlaySimStepSize
    Role: RRS-II Free Play simulation step size
    Detailed role: RRS-II Free Play simulation step size
    • Type: double
    • Default value: 0.1
  • RRS2MotionPlannerActiveDuringTeachOLP
    Role: Causes RRS-II motion planner to be used also during Teach and OLP
    Detailed role: Causes use of RRS-II motion planner also during Teach and OLP
    • Type: integer
    • Default value: 0
  • RRS2StopVRCWhenSimStops
    Role: Stops a real-time managed VRC when simulation is halted
    Detailed role: Stops real-time managed VRC when simulation is halted
    • Type: integer
    • Default value: 0
  • RRS2UseVRCInverseKinematics
    Role: Causes VRC inverse kinematics to be used
    Detailed role: Causes use of VRC inverse kinematics
    • Type: integer
    • Default value: 1
  • RRS2UseVRCTurnLimits
    Role: Causes VRC-based joint turn limits to be used
    Detailed role: Causes use of VRC-based joint turn limits
    • Type: integer
    • Default value: 1
  • RRSLIBDirectory
    Role: RRS$LIB Directory
    Detailed role: Directory containing RRS interface shared library files
    • Type: URN
    • Default value:
  • RRSMsgWindowAutoHideSpeed
    Role: Controls how soon RRS message window will be hidden after a new message is displayed (0%: never hidden, 100%: hidden after 1 sec)
    Detailed role: Controls how soon RRS message window will be hidden after a new message is displayed (0%: never hidden, 100%: hidden after 1 sec)
    • Type: double
    • Default value: 0.5
  • RRSMsgWindowFilterMode
    Role: Filters messages displayed in RRS message window based on severity
    Detailed role: Filters messages displayed in RRS Message Window based on severity
    • Type: integer
    • Default value: 3
  • RRSServersFile
    Role: RRS Servers File
    Detailed role: File listing all available RRS Servers to choose from
    • Type: URN
    • Default value:
