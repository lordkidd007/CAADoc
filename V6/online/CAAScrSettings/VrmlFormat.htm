VrmlFormat Setting Repository
Quick Reference

VrmlFormat

Role: Setting to handle the VRML export/import options

The VrmlFormat repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/VRML
  • Repository File: VrmlFormat
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: VisualizationController
    • Framework_IDL: CATIAApplicationFrame
    • Interface_CPP: CATIVisVrmlSettingAtt
    • Interface_IDL: CATIAVrmlSettingAtt
    • LateType: CATVisVrmlSettingCtrl

VrmlFormat setting repository attributes:

  • ExportBackgroundColor
    Role: Export background color.
    • Type: integer
    • Default value: 0
    • Default Value Role: Black in RGB order
    • Min value: 0
    • Max value: 255
  • ExportEdges
    Role: Permit export of edges.
    • Type: boolean
    • Default value: true
  • ExportNormals
    Role: Permit export of normal informations.
    • Type: boolean
    • Default value: true
  • ExportTexture
    Role: Permit export of textures.
    • Type: boolean
    • Default value: false
  • ExportTextureFile
    Role: Texture export in VRML files or in external files.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default textures are generated in VRML file
    • Values:
      0 VRML file
      1 external files
  • ExportTextureFormat
    Role: Format of exported texture. NOT USED.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default textures are generated in JPEG / PNG depending of transparency management
    • Values:
      0 JPEG / PNG
  • ExportVersion
    Role: Version of exported VRML files.
    • Type: integer
    • Default value: 2
    • Default Value Role: By default VRML 97 files are generated
    • Values:
      1 VRML 1.0 format
      2 VRML 97 format
  • ImportCreaseAngle
    Role: CreaseAngle tuning.
    • Type: float
    • Default value: 0.5
  • ImportUnit
    Role: Units of imported VRML files.
    • Type: integer
    • Default value: 2
    • Default Value Role: By default imported VRML files are considered to be in meters
    • Values:
      0 Millimeters
      1 Centimeters
      2 Meters
