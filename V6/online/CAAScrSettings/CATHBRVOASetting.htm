CATHBRVOASetting

Role: Setting to handle VOA parameters

Detailed role:This setting allows to customize some parameters in VOA

The CATHBRVOASetting repository has the following characteristics:

  • Primary Tab Page: /Ergonomics Design Analysis/Human Builder/Vehicle Occupant Accommodation
  • Repository File: CATHBRVOASetting
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATHBRVOASetting setting repository attributes:

  • DisplayArrows
    Role: Show all dimension arrows in package definition
    Detailed role: The value can be 0 (Do not show) or 1 (Show)
    • Type: integer
    • Default value: 0
  • HandGrasp
    Role: Hand posture uses hand grasp or predefined posture
    Detailed role: The value can be 0 (Predefined posture) or 1 (Hand grasp)
    • Type: integer
    • Default value: 0
  • SeatHPointOutsideRangeManagement
    Role: Setting to manage situations where predicted position of seat h-point is outside track
    Detailed role: The value can be 0 (Ask user), 1 (Force inside track), or 2 (Use predicted position)
