DSESettingsImport Setting Repository
Quick Reference

DSESettingsImport

Role: Settings to manage Digitized Shape Editor Import Options

The DSESettingsImport repository has the following characteristics:

  • Primary Tab Page: /Shape/Digitized Shape Editor/Import
  • Repository File: DSESettingsImport
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DSESettingsImport setting repository attributes:

  • AFEndScan
    Role: End scan for ASCII Free
    • Type: string
    • Default value:
  • AFStartScan
    Role: Start scan for ASCII Free
    • Type: string
    • Default value:
  • AtosMPQuality
    Role: Minimal point quality
    • Type: double
    • Default value: 0.0
    • Min value: 0.00
    • Max value: 100.0
  • AUFFileDescr
    Role: Format Description File
    • Type: string
    • Default value:
  • FileUnit
    Role: Model unit
    • Type: string
    • Default value: Millimeter (mm)
  • GomFacets
    Role: Toggle to create facets
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • GomMPQuality
    Role: Minimal point quality
    • Type: double
    • Default value: 0.0
    • Min value: 0.00
    • Max value: 100.0
  • Grouped
    Role: Toggle to produce single/multiple feature from several imported data
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Sampling
    Role: Percentage of points to import
    • Type: double
    • Default value: 100.0
    • Min value: 0.00
    • Max value: 100.0
    • Step: 10.0
  • ScaleFactor
    Role: Scaling of the model
    • Type: double
    • Default value: 1.0
    • Min value: 0.001
    • Max value: 1000.0
  • STLFacets
    Role: Toggle to create facets
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
