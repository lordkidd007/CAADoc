MeasureSettings Setting Repository
Quick Reference

MeasureSettings

Role: Setting to handle the Measure parameters

The MeasureSettings repository has the following characteristics:

  • Primary Tab Page: /General/Knowledge
  • Repository File: MeasureSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MeasureSettings setting repository attributes:

  • AutoAttachMoveStatus
    Role: Automatically attach the measure result to the mouse
    • Type: integer
    • Default value: 1
    • Values:
      0 Auto attachment Deactivated
      1 Auto attachment Activated
  • BorderColor
    Role: Default color of the border of the text box
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • BorderStatus
    Role: Border Status
    • Type: integer
    • Default value: 1
    • Values:
      0 Border Unchecked
      1 Border Checked
  • FillStatus
    Role: Fill Status
    • Type: integer
    • Default value: 1
    • Values:
      0 Fill Unchecked
      1 Fill Checked
  • FillStyleType
    Role: Fill Style
    • Type: integer
    • Default value: 1
    • Values:
      0 Solid Fill
      1 Gradient Fill L To R
      2 Gradient Fill R To L
      3 Gradient Fill T To B
      4 Gradient Fill B To T
  • LineColor
    Role: Default color of the Line
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • LineWidth
    Role: Default width of line which define the measure
    • Type: unsigned integer
    • Default value: 2
    • Default Value Role: 0.13 mm
    • Min value: 1
    • Max value: 55
  • MeasureOnlyShownElementsStatus
    Role: Measure Only Shown Elements Status
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Only Shown Unchecked
      1 Measure Only Shown Checked
  • MeasureTildeStatus
    Role: Measure Tilde Status
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Tilde Status Unchecked
      1 Measure Tilde Status Checked
  • ProductUpdateStatus
    Role: Activate automatic measure update in product
    • Type: integer
    • Default value: 0
    • Values:
      0 Measure Update Deactivated
      1 Measure Update Activated
  • TextBoxColor
    Role: Default color of the text box
    • Type: unsigned integer
    • Default value: 99
    • Default Value Role: Grey in RGB order
    • Min value: 0
    • Max value: 255
  • TextBoxTransparency
    Role: Default transparency value of the text box
    • Type: unsigned integer
    • Default value: 75
    • Min value: 0
    • Max value: 100
  • TextColor
    Role: Default color of the text
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • TextFont
    Role: Default font of a the text
    • Type: string
    • Default value: "Font"
    • Default Value Role: Default value
  • TextSize
    Role: Default size of the text
    • Type: float
    • Default value: 4
    • Min value: 0
    • Max value: 100
