IGES Setting Repository
Quick Reference

IGES

Role: defines Settings to control conversion of IGES files

The IGES repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/IGES
  • Repository File: IGES
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

IGES setting repository attributes:

  • Angle
    Role: To define the angle (in degree) below which contiguous elements (curves) can be merged.
    • Type: float
    • Default value: 2.0
    • Min value: 0.0
    • Max value: 10
    • Step: 0.5
  • Continuity optimization of curves ans surfaces
    Role: Optimization mode
    • Alias: OptC2Mode
    • Type: integer
    • Default value: 1
    • Values:
      0 No optimization
      1 Automatic optimization
      2 Advanced optimization
      Detailed role: To Define a larger tolerance value which may enhance the optimization impact(resulting in less C2 cutting on faces).
  • Curve and surface type
    Role: To select which curve and surface types you want to be generated
    Detailed role: If you leave the default Standard option selected the curve and surface types created in the Part are kept as is. If you select the
    BSpline option all curves and surfaces are converted into B-splines.
    • Alias: CrvMod
    • Type: integer
    • Default value: 0
    • Values:
      0 Standard
      1 BSpline
  • Curves and surfaces approximation
    Role: To Allow approximation of B-Spline curves and surfaces, controlled by deformation parameter.
    Detailed role: This option usually results in a significant decrease in the number of faces cuttings, but may change the internal structure of BSplines (equations and knots).
    • Alias: OptFittingMode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Deformation
    Role: To define the maximum deformation (in millimeter) allowed in the optimization of curves and surfaces
    • Alias: Tolerance
    • Type: float
    • Default value: 0.001
    • Min value: 0.0005
    • Max value: 0.10
    • Step: 0.0005
  • Detection of invalidity in input geometry
    Role: To detect discrepancies in geometry, controlled by Tolerance value
    • Alias: OptInvalidGeomMode
    • Type: integer
    • Default value: 1
    • Values:
      0 No Detection
      1 Detection
  • Export Units
    Role: To define the unit to be used for export
    • Alias: ExportUnit
    • Type: integer
    • Default value: 0
    • Values:
      0 Keep user unit
      10 Centimeter (cm)
      11 Microinch (uin)
      1 Inch(in)
      2 Millimeter (mm)
      4 Foot (ft)
      5 Mile (mi)
      6 Meter (m)
      7 Kilometer (km)
      8 Mil - i.e. 0.001 inch (mil)
      9 Micron (um)
  • Geometry Quality Enhancement Mode
    Role: Enhancement mode
    • Alias: GeomQualityMode
    • Type: short integer
    • Default value: 1
    • Values:
      1 Default enhancement without deformation
      2 Advanced enhancement with customized allowed deformation
      Detailed role: To define a maximum tolerance value which may enhance the quality.
  • Import Groups
    Role: To Import IGES groups (Entity Type 402, Forms 1-7-14-15: Associativity Instance)as Selection Sets
    • Alias: ImportGroupAsSelSet
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Import Mode
    Role: To generate one single 3D Shape Representation or a Product Structure
    Detailed role: When the Import as a 3D Shape Representation option is selected, the IGES file is translated into one single 3D Shape Representation. For large files containing a large number of 308/408 IGES entities, you can select the option Import as a Product Structure
    • Alias: PartProdMode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • Invalidity Tolerance
    Role: Define tolerance for invalidity detection
    • Alias: TolOptInvalidGeom
    • Type: float
    • Default value: 3.0
    • Min value: 0.5
    • Max value: 99.0
    • Step: 0.5
  • Join surfaces of the model
    Role: To join the surfaces of your IGES model into a shell
    • Alias: ApplyJoin
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Maximum Shape Deformation Factor for Geometry Quality Enhancement
    Role: To define the maximum deformation allowed in the Geometry Quality Enhancement. This is a factor of the Resolution.
    • Alias: GeomQualityTolFact
    • Type: short integer
    • Default value: 5
    • Min value: 5
    • Max value: 100
  • Name of author
    Role: To define the name of author
    • Alias: AuthorName
    • Type: string
    • Default value: UNIX: ${LOGNAME} Windows: ${USERNAME}
  • Organization
    Role: To define the organization into IGES file.
    • Alias: AuthorOrganization
    • Type: string
    • Default value: ${USERDOMAIN}
  • Representation for Boundaries of Faces
    Role: To define the preferred representation between 2D and 3D representations
    • Alias: OptLoop3DMode
    • Type: integer
    • Default value: 0
    • Values:
      0 Keep File Preference
      1 Force 3D representation
  • Representation mode
    Role: To choose the representation mode in the resulting file
    • Alias: RepMod
    • Type: integer
    • Default value: 0
    • Values:
      0 Surface
      1 Wireframe
      2 Solid - Shell
  • Save only shown entities
    Role: To allow to save only the PLM entities which are in the Show mode.
    • Alias: OnlyShow
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Show/NoShow dialog box
    Role: Select this option to display the completion dialog box at the end of the transfer
    Detailed role: By default, the Show Completion Dialog Box option is not selected
    • Alias: ShowCompletionDialogBox
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Tolerance
    Role: Define tolerance for Join operator
    • Alias: TolJoin
    • Type: float
    • Default value: 0.01
    • Min value: 0.00
    • Max value: 0.10
    • Step: 0.001
  • Topological Reduction of Boundaries
    Role: BSpline surfaces and curves continuity is optimized, within deformation and Angle values
    • Alias: OptCleanTopoMode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • XHealing
    Role: Enables XHealing handling, Geometry will be heal
    • Alias: XHealing
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • XHealingMode
    Role: Mode on Geometry Standard, Advanced, Forced
    • Alias: XHealingMode
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 2
  • XHealingTrimmed
    Role: Apply to faces / trimmed surfaces
    • Alias: XHealingTrimmed
    • Type: short integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
