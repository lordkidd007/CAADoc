3DShapeInfrastructureGeneral Setting Repository
Quick Reference

3DShapeInfrastructureGeneral

Role: Setting to manage some 3D Shape Infrastructure options

The 3DShapeInfrastructureGeneral repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/3D Shape Infrastructure/General
  • Repository File: SelectSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

3DShapeInfrastructureGeneral setting repository attributes:

  • AlsoDeleteExclusiveParents
    Role: Delete exclusive parents
    Detailed role: Deletes the exclusive parents of the feature. The parents will be deleted only if they are exclusively used for that feature.
    • Alias: DeleteSketchMode
    • Type: boolean
    • Default value: false
  • DeleteWarningBox
    Role: Display the Delete dialog box
    Detailed role: Displays a dialog box providing options for deleting the parents and children of the element selected for deletion.
    • Type: boolean
    • Default value: true
  • ExternalReferencesAsVisible
    Role: Show newly created external references
    Detailed role: External references are visible in the geometry when they are created.
    • Alias: ImportsVisualizationMode
    • Type: boolean
    • Default value: false
  • LinkedExternalReferences
    Role: Keep link with the selected object
    Detailed role: The object created from the selection will keep the link with it.
    • Alias: SelectLink
    • Type: boolean
    • Default value: false
  • LinkedExternalReferencesOnlyOnPublication
    Role: Restrict external selection with link to published elements
    Detailed role: Only published elements are allowed when selecting or copying from another representation with a link.
    • Alias: SelectPublishLinkMode
    • Type: boolean
    • Default value: false
  • LinkedExternalReferencesWarningAtCreation
    Role: Confirm when creating a link with selected object
    Detailed role: Asks to confirm when the object created from the selection will keep the link with it.
    • Alias: ConfirmSelectLinkMode
    • Type: boolean
    • Default value: false
  • LinkStatusAtRepLevel
    Role: Display link status at rep level.
    Detailed role: Display link status at rep level.
    • Alias: LinkStatusAtRepLevel
    • Type: boolean
    • Default value: false
  • ReplaceOnlyAfterCurrent
    Role: Do replace only for elements situated after the In Work Object
    • Alias: ReplaceCurrent
    • Type: boolean
    • Default value: false
  • StatusOnPubOnly
    Role: Restrict display status to published elements
    Detailed role: Restrict display status to published elements
    • Alias: StatusOnPubOnly
    • Type: boolean
    • Default value: false
  • ViewerOnSynchroWhenNeeded
    Role: Launch Replace Viewer during synchronization process always or when really needed
    • Type: boolean
    • Default value: true
  • ViewerWantedOnSynchro
    Role: Launch Replace Viewer during synchronization process
    Detailed role: Launch Replace Viewer to enable user interaction during an External Reference synchronisation process.
    • Type: boolean
    • Default value: true
