DNBVNCSimulSettings Setting Repository
Quick Reference

DNBVNCSimulSettings

Role: Lists the settings used during the machine simulation in the Machining applications

Detailed role:The machining simulation settings are useful to manages options used during a machine simulation

The DNBVNCSimulSettings repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Simulation
  • Repository File: DNBVNCSimulSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBVNCSimulSettings setting repository attributes:

  • DisplayTPInSimul
    Role: Defines if the tool path is shown during the machine simulation
    Detailed role: Defines if the tool path is shown during the machine simulation
    • Type: boolean
    • Default value: false
  • NCCodeSimulFastMRS
    Role: Enables fast computation for material removal
    Detailed role: Enables fast computation for material removal
    • Type: boolean
    • Default value: false
  • NCCodeSimulFastMRSAccuracy
    Role: Fast Milling Accuracy
    Detailed role: Defines the accuracy of the fast milling simulation
    • Type: float
    • Default value: 0.05
    • Default Value Role: By default, the accuracy of the milling simulation is set to 0.05
    • Min value: 0.01
    • Max value: 1000
  • NCCodeSimulMode
    Role: Defines if material removal path is done during the machine simulation
    Detailed role: Defines if material removal path is done during the machine simulation
    • Type: boolean
    • Default value: false
  • NCCodeSimulStepBackward
    Role: Defines number of steps backward to be managed during material removal path
    Detailed role: Defines number of steps backward to be managed during material removal path
    • Type: integer
    • Default value: 1000
    • Default Value Role: By default, the max number of steps backward managed is 1000
    • Min value: 10
    • Max value: 10000
  • NCCodeSimulSyncMat
    Role: Synchronizes material removal with simulation steps
    Detailed role: Synchronizes material removal with simulation steps
    • Type: boolean
    • Default value: false
  • NCCodeSimulVolumeClash
    Role: Defines if volume is computed when a clash is detected during material removal path
    Detailed role: Defines if volume is computed when a clash is detected during material removal path
    • Type: boolean
    • Default value: false
  • NCCodeToolRepType
    Role: Type of tool for ISO. Available options: Nominal, Maximum Wear
    Detailed role: Defines the type of tool used during ISO code simulation
    • Type: integer
    • Default value: 0
    • Default Value Role: Nominal
    • Values:
      0 Nominal
      1 Maximum Wear
