VisualizationRepository Setting Repository
Quick Reference

VisualizationRepository

Role: Setting to handle the visualization options

The VisualizationRepository repository has the following characteristics:

  • Primary Tab Page: /General/Display/Visualization
  • Repository File: VisualizationRepository
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: VisualizationFoundation
    • Framework_IDL: VisualizationOptions
    • Interface_CPP: CATIVizVisualizationSettingAtt
    • Interface_IDL: CATIAVisualizationSettingAtt
    • LateType: CATVizVisualizationSettingCtrl
