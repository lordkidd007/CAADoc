DmtDocumentGeneral Setting Repository
Quick Reference

DmtDocumentGeneral

Role: External Document Options

Detailed role:This option enables to modify External Document behaviors.

The DmtDocumentGeneral repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/PLM Access
  • Repository File: DmtDocumentGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DmtDocumentGeneral setting repository attributes:

  • AlwaysAskWhileDnDFile
    Role: Should always ask for document type before creation by drag and drop.
    • Type: boolean
    • Default value: true
    • Default Value Role: By default document type is always asked.
  • DocumentExpandMode
    Role: Should Documents be displayed in the navigator tree during an expand
    • Type: boolean
    • Default value: false
    • Default Value Role: By default, Documents are not displayed in the tree
  • PreSelectedDocType
    Role: The type of document to be created from a dragged and dropped file from Windows environment onto a Folder in VMP Navigator.
    • Type: string
    • Default value: Document Customized
