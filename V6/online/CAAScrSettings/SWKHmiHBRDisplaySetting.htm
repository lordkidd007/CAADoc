SWKHmiHBRDisplaySetting Setting Repository
Quick Reference

SWKHmiHBRDisplaySetting

Role: Setting to handle manikin display parameters

Detailed role:This setting allows to customize the display parameters in human builder products

The SWKHmiHBRDisplaySetting repository has the following characteristics:

  • Primary Tab Page: /Ergonomics Design Analysis/Human Builder/Display
  • Repository File: SWKHmiHBRDisplaySetting
  • Version: 2
  • Full Compatibility: 2
  • Upward Compatibility: 2

SWKHmiHBRDisplaySetting setting repository attributes:

  • FKAutoReframeOnMode
    Role: Automatic reframe on selected segment
    • Type: integer
    • Default value: 0
    • Default Value Role: Auto reframe off
  • FKDOF1Color
    Role: Color of first DOF of FK manipulator
    • Type: unsigned integer
    • Default value: 4294901760
    • Default Value Role: Yellow color
  • FKDOF2Color
    Role: Color of second DOF of FK manipulator
    • Type: unsigned integer
    • Default value: 4278190080
    • Default Value Role: Red color
  • FKDOF3Color
    Role: Color of third DOF of FK manipulator
    • Type: unsigned integer
    • Default value: 16711680
    • Default Value Role: Green color
  • FKScaleFactorReframe
    Role: Scale factor for FK reframe
    • Type: double
    • Default value: 2.0
    • Default Value Role: Factor
  • IKPrintAvailability
    Role: Enable the twin segment to be displayed
    • Type: integer
    • Default value: 1
    • Default Value Role: By default enable
    • Default Value Detailed Role The value can be 0(disable) or 1(enable)
  • LoadActivatedColor
    Role: Color of the Load when activate
    • Type: unsigned integer
    • Default value: 16711680
    • Default Value Role: Green color
  • LoadDeactivatedColor
    Role: Color of the Load when deactivate
    • Type: unsigned integer
    • Default value: 4294901760
    • Default Value Role: Yellow color
  • LoadDisplayMode
    Role: Control the display of the Load
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Load Arrow
    • Default Value Detailed Role the value can be 0(load arrow), 1(force line) and 2(none)
  • PrintColor
    Role: Twin segment color when displayed
    • Type: unsigned integer
    • Default value: 4294901760
    • Default Value Role: Yellow color
  • PrintTransparency
    Role: Transparency of the print
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: The value can be between [0, 255] %
    • Min value: 0
    • Max value: 255
  • ReachPrintAvailability
    Role: Enable the twin segment to be displayed in 3D reach mode
    • Type: integer
    • Default value: 1
    • Default Value Role: By default enable
    • Default Value Detailed Role The value can be 0(disable) or 1(enable)
  • SegmentOffsetActivatedColor
    Role: Color of the segment offset when activate
    • Type: unsigned integer
    • Default value: 16711680
    • Default Value Role: Green color
  • SegmentOffsetDeactivatedColor
    Role: Color of the segment offset when deactivate
    • Type: unsigned integer
    • Default value: 4210688000
  • SelectedHandGraspColor
    Role: Selected hand grasp color
    • Type: unsigned integer
    • Default value: 16711680
    • Default Value Role: Green color
  • VisionBlindSpotColor
    Role: Blind Spot color
    • Type: unsigned integer
    • Default value: 65280
    • Default Value Role: Color
  • VisionCentralConeColor
    Role: Central cones and contours color
    • Type: unsigned integer
    • Default value: 4294901760
    • Default Value Role: Yellow color
  • VisionPeripheralConeColor
    Role: Peripheral cones and contours color
    • Type: unsigned integer
    • Default value: 4278190080
    • Default Value Role: Red color
  • VisionPeripheralContourRotation
    Role: Enable the rotation of the peripheral contour of vision window
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: By default enable
    • Default Value Detailed Role The value can be 0(disable) or 1(enable)
