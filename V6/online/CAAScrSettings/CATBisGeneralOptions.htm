CATBisGeneralOptions

Role: General Options for BioSystems Experiment

Detailed role:These options allow user to customize the behavior of BioSystems Experiment

The CATBisGeneralOptions repository has the following characteristics:

  • Primary Tab Page: /Simulation/BioSystems Experiment/General
  • Repository File: CATBisGeneralOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATBisGeneralOptions setting repository attributes:

  • ActiveEntityColor
    Role: Options for Active Entity Color
    Detailed role: This option allows user to customize the color of the Active Entity
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: RGB order
    • Min value: 0
    • Max value: 255
  • InactiveEntityColor
    Role: Options for Inactive Entity Color
    Detailed role: This option allows user to customize the color of the Inactive Entity
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: RGB order
    • Min value: 0
    • Max value: 255
