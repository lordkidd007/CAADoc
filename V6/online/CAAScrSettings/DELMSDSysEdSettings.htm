DELMSDSysEdSettings

Role: Settings for System Editor UI

The DELMSDSysEdSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/System Editor
  • Repository File: DELMSDSysEdSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
