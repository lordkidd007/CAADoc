FbdiSettingsRep Setting Repository
Quick Reference

FbdiSettingsRep

Role: Settings for the FBDI command.

Detailed role:This setting allows define extensions not to process during File-based Design Import. It also allows to define the migration of CAA data.

The FbdiSettingsRep repository has the following characteristics:

  • Primary Tab Page: /General/FBDI
  • Repository File: FbdiSettingsRep
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1

FbdiSettingsRep setting repository attributes:

  • FBDI_ActivatedCAA
    Role: Activate the support of CAA data in FBDI.
    • Type: boolean
    • Default value: 0
  • FBDI_AsynchronousMode
    Role: Activate the asynchronous mode.
    • Type: boolean
    • Default value: 0
  • FBDI_AttrMappingConcat
    Role: Add V5 Id to PLM_ExternalId attribute.
    • Type: boolean
    • Default value: 1
  • FBDI_AttrMappingName
    Role: Attribute name mapped with the phase 1 Part number.
    • Type: string
    • Default value: V_Name
  • FBDI_CATPartMode
    Role: CATPart import mode.
    Detailed role: Type of object created by CATPart import (1 for 3DPart, 2 for once inst Rep, 3 for sharable Rep).
    • Type: integer
    • Default value: 1
  • FBDI_Files_Filter_Size
    Role: Control the File Access mode.
    Detailed role: This option allows to control the numbers of extensions to ignore.
    • Type: integer
    • Default value: 0
  • FBDI_LinkIsolationMode
    Role: Link isolation mode.
    Detailed role: This attribute can take 3 values 1 2 3 corresponding to 3 modes (Always Never User).
    • Type: integer
    • Default value: 3
  • FBDI_MappingTablesDirectory
    Role: Path of the Directory Mapping Table which contains the mapping tables used for asynchronous mode.
    • Type: URN
    • Default value:
  • FBDI_RemoveUnmigrableCAACont
    Role: Remove CAA Containers which can not be imported.
    • Type: boolean
    • Default value: 0
  • FBDI_XMLDirectoryCAA
    Role: Path of the directory for the XML files for CAA support.
    • Type: string
    • Default value:
