FstDrafting Setting Repository
Quick Reference

FstDrafting

Role: Defines the Fst spot assembly fastener drafting extraction parameters

Detailed role:Enable the user to specify the Spot Assembly Fastener Drafting Extraction parameters

The FstDrafting repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/FastenerInfrastructure/Drafting
  • Repository File: FstDrafting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

FstDrafting setting repository attributes:

  • AngularTolerance
    Role: Setting the maximum angle between the spot normal direction and the drafting view direction for which the spot assembly fastener drafting extraction is possible
    • Type: double
    • Default value: 90.0
  • FixedSpotSize
    Role: Indicate the fixed drawing symbol size (available only if Fixed spot size Checked is yes)
    • Type: double
    • Default value: 0.0
  • FixedSpotSizeCheck
    Role: : Indicate whether the drawing symbol size is fixed for all the spot assembly fasteners OR varies as per the spot assembly fastener diameter during the fastener drafting extraction process
    • Type: boolean
    • Default value: false
  • ReverseDirection
    Role: Indicate whether to include or exlude the spot assembly fasteners in the opposite direction of view direction during spot assembly fastener drafting extraction process
    • Type: boolean
    • Default value: true
