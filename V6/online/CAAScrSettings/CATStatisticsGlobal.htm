CATStatisticsGlobal Setting Repository
Quick Reference

CATStatisticsGlobal

Role: Settings for Global Statistics

Detailed role:The global statistics controler manages the values of the attributes common to all the thematics.

The CATStatisticsGlobal repository has the following characteristics:

  • Primary Tab Page: /General/General/Statistics
  • Repository File: CATStatistics
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /General/Global Statistics Management reused as /ProductStructure/Global Statistics Management

CATStatisticsGlobal setting repository attributes:

  • Overall_BufferSize
    Role: Maximum buffer size
    Detailed role: Statistics log are buffered before being send to the output file.
    • Type: unsigned integer
    • Default value: 5
    • Min value: 1
    • Max value: 1024
  • Overall_CopyDirUx
    Role: Copy directory for the statistics on unix
    • Type: string
    • Default value:
  • Overall_CopyDirWin
    Role: Copy directory for the statistics on windows
    • Type: string
    • Default value:
  • Overall_MaxPerFile
    Role: Maximum size per file in kO.
    • Type: unsigned integer
    • Default value: 50
    • Min value: 1
    • Max value: 1024
  • Overall_NbCopies
    Role: Maximum number of copy files
    Detailed role: If the file size is exceeded, copy is made in the defined directory. If the maximum number of copies is reached, the oldest files are replaced by the new ones.
    • Type: unsigned integer
    • Default value: 5
    • Min value: 1
    • Max value: 999
  • Overall_StatsScheduling
    Role: Statistics events shceduling capability
    • Type: unsigned integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
