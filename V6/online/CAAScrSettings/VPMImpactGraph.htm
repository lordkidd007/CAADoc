VPMImpactGraph Setting Repository
Quick Reference

VPMImpactGraph

Role: Setting to handle the parameters of the VPM Imact Graph User Interface

Detailed role:This setting allows to customize the appearance and the behavior of the VPM Impact Graph User Interface

The VPMImpactGraph repository has the following characteristics:

  • Primary Tab Page: /ENOVIAV5VPM/PLM Impact Graph Properties
  • Repository File: VPMImpactGraph
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

VPMImpactGraph setting repository attributes:

  • *_IGColor
    Role: Color of the arc representing a link
    • Type: integer
    • Default value: 0
    • Default Value Role: DON'T MODIFY THESE VALUES
    • Min value: 0
    • Max value: 255
  • *_IGName
    Role: Expand filter on a link
    Detailed role: This option allows to indicate impact results to take a link into account or not
    • Type: integer
    • Default value: 1
  • BackGroundColor
    Role: Background color for the Impact Analysis editor
    • Type: integer
    • Default value: 223
    • Default Value Role: The RGB values of the background color ( Metal grey )
    • Min value: 0
    • Max value: 255
  • EnableVPMnavCoupling
    Role: Automatic coupling with VPM navigation window
    Detailed role: This option allows to ask for an Automatic coupling with VPM navigation window
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the coupling with a VPM Navigation window is disable
  • ExpandRelationName
    Role: Display full name at expand on relation node
    Detailed role: This option allows to ask for full display name at expand on relation node
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the link name is displayed
  • HideRelation
    Role: Manage the display of the relations
    Detailed role: This option allows to ask for hide the display of the relations
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the relation are displayed
  • ImpactModeAtStartup
    Role: Define the default impact direction
    Detailed role: This option allows to ask for efine the default impact direction
    • Type: integer
    • Default value: 2
    • Default Value Role: By default the impact direction is 'Impacted By'
    • Values:
      1 The default impact direction is 'Impacts On'
      2 The default impact direction is 'Impacted By'
  • Navigation3D
    Role: Activate the 3D Navigation
    Detailed role: This option allows to ask for activate the 3D Navigation
    • Type: integer
    • Default value: 1
    • Default Value Role: By default 3D Navigation is enable
  • PublicationLinkDisplay
    Role: Manage publication links display
    Detailed role: This option allows to ask for hide publication links display
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Publication links are displayed
