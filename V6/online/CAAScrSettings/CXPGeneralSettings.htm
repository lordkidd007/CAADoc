CXPGeneralSettings Setting Repository
Quick Reference

CXPGeneralSettings

Role: Setting to handle CXP model parameters

The CXPGeneralSettings repository has the following characteristics:

  • Primary Tab Page: /Interactive 3DEXPERIENCE/Creative Experience/General
  • Repository File: CXPGeneralSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CXPGeneralSettings setting repository attributes:

  • 3DActorRepVisuAuth
    Role:
    • Type: string
    • Default value: Dummy
    • Values:
      Hidden
      Dummy
  • 3DActorRepVisuPlay
    Role:
    • Type: string
    • Default value: Hidden
    • Values:
      Hidden
      Dummy
  • ForceVPManipPlay
    Role:
    • Type: boolean
    • Default value: false
  • OneClickPrimAuth
    Role:
    • Type: boolean
    • Default value: false
  • PresetChooserAuth
    Role: Parameter to set the preset chooser default value
    • Type: boolean
    • Default value: true
  • ScriptEditorFontBold
    Role: Parameter to set whether the font is in Bold or not in the script editor
    • Type: boolean
    • Default value: false
  • ScriptEditorFontItalic
    Role: Parameter to set whether the font is in Italic or not in the script editor
    • Type: boolean
    • Default value: false
  • ScriptEditorFontSize
    Role: Parameter to set the font size of the script editor
    • Type: integer
    • Default value: 10
  • ScriptEditorShowLines
    Role: Parameter to set whether the line numbers are shown or not in the script editor
    • Type: boolean
    • Default value: true
  • SelectionMode
    Role:
    • Type: string
    • Default value: LeafActorSelection
    • Values:
      RootActorSelection
      LeafActorSelection
      ActorLevelSelection
  • ShowBehaviorsInTree
    Role: Parameter to set the display of behaviors in the Experience Tree View
    • Type: boolean
    • Default value: false
  • TriggerZoneRepVisuAuth
    Role:
    • Type: string
    • Default value: Dummy
    • Values:
      Hidden
      Dummy
      Shown
  • TriggerZoneRepVisuPlay
    Role:
    • Type: string
    • Default value: Hidden

      Hidden
