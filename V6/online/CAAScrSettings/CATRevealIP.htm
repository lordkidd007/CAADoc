CATRevealIP Setting Repository
Quick Reference

CATRevealIP

Role: Setting to handle the parameters of the Reveal-IP User Interface

Detailed role:This setting allows to customize the appearance and the behavior of the Reveal-IP User Interface

The CATRevealIP repository has the following characteristics:

  • Primary Tab Page: /ENOVIAV5VPM/Reveal IP
  • Repository File: CATRevealIP
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATRevealIP setting repository attributes:

  • RIP_DisplayResultMode*
    Role: Define the results display mode
    Detailed role: This option allows to define the results display mode in term of container ( Current, New, Same )
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the results are displayed within the current container
    • Min value: 1
    • Max value: 3
  • RIP_RelatedObjectsAfterAddInScene*
    Role: Compute automatically the Related Objects after insert in Scene
    Detailed role: This option allows to compute automatically the Related Objects of an object after insert in Scene
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the Related Objects are not computed
  • RIPFct_*
    Role: Set the availability of the function
    Detailed role: This option allows to take this function into account with the relational navigation
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the function is enabled
  • RIPGrp_*
    Role: Set the availability of the function group
    Detailed role: This option allows to take this function group into account with the relational navigation
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the function group is enabled
