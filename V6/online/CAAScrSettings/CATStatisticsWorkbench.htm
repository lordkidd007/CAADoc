CATStatisticsWorkbench Setting Repository
Quick Reference

CATStatisticsWorkbench

Role: Settings for Workbench Statistics

Detailed role:The Workbench statistics controler manages the values of all or only a part of the attributes available for the thematic

The CATStatisticsWorkbench repository has the following characteristics:

  • Primary Tab Page: /General/General/Statistics
  • Repository File: CATStatistics
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Root tab page /General/Workbench Statistics Management reused as /ProductStructure/Workbench Statistics Management

CATStatisticsWorkbench setting repository attributes:

  • Version
    Role: Internal use - statistics settings version level.
    • Type: integer
    • Default value: 1
  • WORKBENCH_Activation
    Role: Activation state of the Workbench statistics
    Detailed role: Workbench statistics activation
    • Type: integer
    • Default value: 0
  • WORKBENCH_CPUS
    Role: If activated, the cpu duration field is outputed.
    • Type: boolean
    • Default value: true
  • WORKBENCH_Cumulate
    Role: Cumulation mode state of the Workbench statistics
    Detailed role: Value of Workbench statistics cumulation mode
    if false there is one file per session
    if true all sessions are outputted in one file
    • Type: boolean
    • Default value: true
  • WORKBENCH_DateFormat
    Role: Format for the date and time field.
    • Type: integer
    • Default value: 1
    • Values:
      1 standard : Mon Jan 1 08:00.00 2000
      2 numerical : 2000.001.08.00.00
      3 numerical with millisecond : 2000.001.08.00.00.000
  • WORKBENCH_ELPS
    Role: If activated, the elapsed duration field is outputed.
    • Type: boolean
    • Default value: true
  • WORKBENCH_Formating
    Role: Format of the Workbench statistics
    • Type: integer
    • Default value: 1
    • Values:
      1 Standart output format (ie : field="value")
      2 Field is not outputed (ie : "value" )
      3 Common Base Event (CBE) format
      4 WMI EventLog integration
  • WORKBENCH_HOST
    Role: If activated, the host name field is outputed.
    • Type: boolean
    • Default value: false
  • WORKBENCH_OutputDirectory
    Role: Output directory for the statistic
    • Type: string
    • Default value: default_directory
  • WORKBENCH_OutputFile
    Role: Name of the output file
    • Type: string
    • Default value: default_file
  • WORKBENCH_OutputFormat
    Role: Output format of the Workbench statistics
    • Type: integer
    • Default value: 1
    • Values:
      1 standard output
      2 the thematic name is not repeated on each line
  • WORKBENCH_OutputType
    Role: Choice of the output media : file or console
    • Type: integer
    • Default value: 2
    • Values:
      1 Console : log is outputed in the console (not used)
      2 File : log is outputed in a file
  • WORKBENCH_RTIM
    Role: If activated, the response time field is outputed.
    • Type: boolean
    • Default value: true
  • WORKBENCH_THEM
    Role: If activated, the thematic name field is outputed.
    • Type: boolean
    • Default value: true
  • WORKBENCH_TIME
    Role: If activated, the time field is outputed.
    • Type: boolean
    • Default value: true
  • WORKBENCH_TimeUnit
    Role: Time unit for the cpu, elaped and response time.
    • Type: integer
    • Default value: 1
    • Values:
      1 millisecond
      2 second
  • WORKBENCH_UPID
    Role: If activated, the process id field is outputed.
    • Type: boolean
    • Default value: false
  • WORKBENCH_USER
    Role: If activated, the user name field is outputed.
    • Type: boolean
    • Default value: false
