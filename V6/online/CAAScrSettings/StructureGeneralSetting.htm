StructureGeneralSetting Setting Repository
Quick Reference

StructureGeneralSetting

Role: Setting to manage the options of Structure

Detailed role:Structure setting contains options necessary to the commands of Structure

The StructureGeneralSetting repository has the following characteristics:

  • Primary Tab Page: /MechanicalDesign/CATSldPLMRepConfig/General
  • Repository File: StructureGeneralSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

StructureGeneralSetting setting repository attributes:

    • Default value: 0
    • Default Value Role: FormingMode_Settings
  • Member_Settings
    Role: Setting to manage the display of the Member'Limit Type

    • Default value: 0
  • Plate_Settings
    Role: Setting to manage the display of the Plate'Limit Type
    Detailed role: Displays Plate's Limit Type.
    • Type: integer
    • Default value: 2

    • Default value: 1.22173047639603065
  • StandAlonePlate_Settings
    Role: Setting to manage the display of the StandAlonePlate'Limit Type
    Detailed role: Displays StandAlonePlate's Limit Type.
    • Type: integer
    • Default value: 2
  • Stiffener_Settings
    Role: Setting to manage the display of the Stiffener'Limit Type
