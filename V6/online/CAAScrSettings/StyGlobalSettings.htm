StyGlobalSettings Setting Repository
Quick Reference

StyGlobalSettings

Role: Settings to manage Shape Sculptor General Parameters and Options

The StyGlobalSettings repository has the following characteristics:

  • Primary Tab Page: /Shape/Shape Sculptor/Parameters
  • Repository File: StyGlobalSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

StyGlobalSettings setting repository attributes:

  • CustomSettings
    Role: Custom settings to enable/disable the modification of parameters that control the precision of the curves
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • Deviation
    Role: Chordal deviation used to control the precision of the curves
    • Type: double
    • Default value: 0.001
    • Min value: 0.001
    • Max value: 100000.0
    • Step: 0.1
  • FitCurve
    Role: Option to enable/disable the curve fitting
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • Length
    Role: Segment Length used to control the precision of the curves
    • Type: double
    • Default value: 0.001
    • Min value: 0.001
    • Max value: 100000.0
    • Step: 0.1
  • SectionMesh
    Role: Option to enable/disable the section of meshes
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • ShowContinuityTools
    Role: Option show/hide the continuity tools
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • ShowDegreeTools
    Role: Option show/hide the degree tools
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • ShowDistanceTool
    Role: Option show/hide the distance tools
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
