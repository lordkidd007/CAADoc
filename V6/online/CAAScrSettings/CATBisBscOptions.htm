CATBisBscOptions Setting Repository
Quick Reference

CATBisBscOptions

Role: Options for BioSystems Deterministic Simulation

Detailed role:These options allow user to customize the behavior of BioSystems Stochastic Simulation

The CATBisBscOptions repository has the following characteristics:

  • Primary Tab Page: /Simulation/BioSystems Experiment/Stochastic Simulation
  • Repository File: CATBisBscOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATBisBscOptions setting repository attributes:

  • BscComputeStatisticalResults
    Role: Compute Statistical Results
    • Type: integer
    • Default value: 0
  • BscDuration
    Role: Duration
    • Type: double
    • Default value: 60.
  • BscMaxNumberOfSimulations
    Role: Maximum Number of Simulations to Compute the Statistical Results
    • Type: integer
    • Default value: 400
  • BscNumberOfStoredSimulations
    Role: Number of Simulations Stored in the Results
    • Type: integer
    • Default value: 3
  • BscSchema
    Role: Schema
    • Type: string
    • Default value: GillepsieDirectMethod
    • Values:
      GillepsieDirectMethod GillepsieDirectMethod
  • BscStartTime
    Role: Start time
    • Type: double
    • Default value: 0.
  • BscStepSize
    Role: Step size
    • Type: double
    • Default value: 0.1
  • BscStepType
    Role: Step type
    • Type: integer
    • Default value: 0
    • Values:
      0 Step size
      1 Interval number
  • BscTolerance
    Role: Tolerance
    • Type: double
    • Default value: 0.00001
