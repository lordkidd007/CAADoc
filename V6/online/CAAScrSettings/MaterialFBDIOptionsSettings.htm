MaterialFBDIOptionsSettings Setting Repository
Quick Reference

MaterialFBDIOptionsSettings

Role: Setting to modify material beahvior during FBDI

Detailed role:Change the way materials are imported

The MaterialFBDIOptionsSettings repository has the following characteristics:

  • Primary Tab Page: /General/FBDI/Materials
  • Repository File: MaterialFBDIOptionsSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

MaterialFBDIOptionsSettings setting repository attributes:

  • FBDIOptimisationActivation
    Role: Used to activate optimisations on material data imported during FBDI without mapping table
    • Type: boolean
    • Default value: true
  • FBDIOptimSimilarity
    Role: Used to define the similarity function used if optimisation is activated
    • Type: short integer
    • Default value: 1
    • Values:
      1 Full
      2 Name
