FstGeneral Setting Repository
Quick Reference

FstGeneral

Role: These setting Parameters define the general Fastener Report Parameters

Detailed role:Enable the user to specify the Report Directory

The FstGeneral repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/FastenerInfrastructure/General
  • Repository File: FstGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

FstGeneral setting repository attributes:

  • FstAssitantMode
    Role: Indicates if Fastener assistant computation mode for part selection is cylinderical or spherical
    • Type: string
    • Default value: Cylinderical
    • Values:
      "Cylinderical" select part through cylinderical mode
      "Spherical" select part through spherical mode
  • FstCommonNode
    Role: : Indicates that the fastener will be created under common computed father of fastened parts
    • Type: boolean
    • Default value: true
  • FstComputePilot
    Role: Indicates if the pilot part definition is to be calculated automatically or manually
    • Type: string
    • Default value: ComputePilotAutomatic
    • Values:
      "ComputePilotAutomatic" Compute pilot part automatically
      "ComputePilotManual" Compute pilot part manually
  • FstComputeSurface
    Role: Indicates if the closest surface is to be calculated automatically or manually
    • Type: string
    • Default value: ComputeSurfaceAutomatic
    • Values:
      "ComputeSurfaceAutomatic" Compute surface automatically
      "ComputeSurfaceManual" Compute surface manually
  • FstFastenedPartsModification
    Role: : Indicates if the fastened parts linked to the fastener can be modified during fastener connection edition
    • Type: boolean
    • Default value: true
  • FstFastenerDimension
    Role: Indicates if Fastener dimension is Diameter or Length of the fastener. This is used for fastener assistant in spherical computation mode
    • Type: string
    • Default value: Length
    • Values:
      "Diameter" use diameter as fastener dimension
      "Length" use length as fastener dimension
  • FstFilterRetrievedConnections
    Role: : Indicates apply filter for retrieved connections in CATFstPLMNavigationServices::GetFastenConnectionsFromFastenerInstance
    • Type: boolean
    • Default value: false
  • FstKeepGeomAssocMode
    Role: : Indicates that the fastener will be created with associativity with the selected geometry
    • Type: boolean
    • Default value: true
  • FstLengthUnit
    Role: Selects the unit of length that will be used to display the values in the generated report
    • Type: integer
    • Default value: 0
    • Values:
      0 Millimeter: The length unit is specified in Millimeter
      1 Inch: The length unit is specified in Inch
  • FstLiveAdditionalConnection
    Role: Indicates whether to show or hide the additional connection button in Live Fastening
    • Type: boolean
    • Default value: true
  • FstMaxComponent
    Role: Indicates the number of digits after the decimal places that are to be displayed
    • Type: double
    • Default value: 5
  • FstMaxPart
    Role: Indicates the number of digits after the decimal places that are to be displayed
    • Type: double
    • Default value: 5
  • FstNbLinesTextOnThumbnail
    Role: Indicates the maximum number of lines of text to be shown on thumbnail
    • Type: double
    • Default value: 2
  • FstOrientationMode
    Role: : Indicates that the fastener will be orientated around its KVector axis
    • Type: boolean
    • Default value: false
  • FstPrecision
    Role: Indicates the number of digits after the decimal places that are to be displayed
    • Type: double
    • Default value: 1.0
  • FstReportDirectory
    Role: Selects the directory that will contain the generated report
    Detailed role: Enables the user to select the output directory that will contain the generated report
    • Type: URN
    • Default value: UNIX: ${CATTemp} Windows: ${CATTemp}
  • FstRulesCheckAfterCreation
    Role: : Indicates that immersive report command will be launched after creation of fastener
    • Type: boolean
    • Default value: false
  • FstRulesCheckAfterEdition
    Role: : Indicates that immersive report command will be launched after edition of fastener
    • Type: boolean
    • Default value: false
  • FstSctfNotation
    Role: : Indicates if the numbers are to be displayed in scientific notation or not
    • Type: boolean
    • Default value: false
  • FstSphereAppearanceMode
    Role: Indicates the appearance of fastener sphere preview
    • Type: integer
    • Default value: 22
    • Values:
      22 PreDefined: Fastener Preview with Fixed Size of Sphere
      23 Dynamic: Fastener Preview with Dynamic Size of Sphere
  • FstSphereRepRadius
    Role: Indicates the live fasener preview sphere radius to be displayed
    • Type: double
    • Default value: 2.5
  • FstStandardDirectory
    Role: Specifies the directory that contains the fastener standard to use
    • Type: URN
    • Default value:
  • FstStandardSelectionMode
    Role: Indicates if Fastener standard is selected from directory or through project resource management
    • Type: string
    • Default value: Directory
    • Values:
      "Directory" select standard through directory mode
      "ProjectResourceManagment" select standard through PRM mode
  • FstStoreSurface
    Role: : Indicates if the surface information is stored for non parametric location method
    • Type: boolean
    • Default value: true
  • FstValidationMode
    Role: Indicates in which way user need to validate the steps, while creating the fasteners
    • Type: integer
    • Default value: 10
    • Values:
      10 Single Click: Validation is done by single click in empty space
      11 Double Click: Validation is done by double click in empty space
      14 Button: Validation is done by arrow buttons provided on time line
  • FstWeightUnit
    Role: Selects the unit of weight that will be used to display the values in the generated report
    • Type: integer
    • Default value: 0
    • Values:
      0 Kilogram: The weight unit is specified in Kilogram
      1 Pound: The weight unit is specified in Pound
