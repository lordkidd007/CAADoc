DNBTSASpotWeldTrajectorySettings

Role: Settings to handle the parameters of the TSA Spot Weld Trajectory Settings

Detailed role:This setting allows to customize the appearance and the behavior of TSA Spot Weld Trajectory Settings

The DNBTSASpotWeldTrajectorySettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Resource Detailing/Tool Selection Assistant
  • Repository File: DNBTSASpotWeldTrajectorySettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBTSASpotWeldTrajectorySettings setting repository attributes:

  • ApproachAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 1
    • Default Value Role: By default +ve X-Axis is the Approach axis
    • Values:
      0 -veX-Axis
      1 X-Axis
      2 -veY-Axis
      3 Y-Axis
      4 -veZ-Axis
      5 Z-Axis
  • PointColor
    Role: The Trajectory Point Color
    Detailed role: The points in the trajectory that are set
    • Type: integer
    • Default value: 255
    • Default Value Role: White in RGB order
    • Min value: 0
    • Max value: 255
  • RotationAxis
    Role: The rotation axis
    Detailed role: Which is the rotation axis
    • Type: short integer
    • Default value: 0
    • Default Value Role: By default Z is the rotation axis
    • Values:
      0 Z-Axis
      1 Y-Axis
  • SystemStructure
    Role: System Structure Creation
    • Type: short integer
    • Default value: 1
    • Default Value Role: MultiFastenProcess
    • Values:
      1 MultiFastenProcess
      2 SingleFastenProcess
  • XMLStyleSheetPath
    Role: XML Style sheet path
    Detailed role: Stores XML Style sheet path
    • Type: string
    • Default value:
    • Default Value Role: By default, the path will be empty
