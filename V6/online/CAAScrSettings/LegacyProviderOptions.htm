LegacyProviderOptions Setting Repository
Quick Reference

LegacyProviderOptions

Role: Settings for All Legacy Providers

Detailed role:This setting allows setting of different providers

The LegacyProviderOptions repository has the following characteristics:

  • Primary Tab Page: /General/FBDI
  • Repository File: LegacyProviderOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

LegacyProviderOptions setting repository attributes:

  • IsLegacyProviderExpObjActivated
    Role: Activate Exposed objects support.
    • Type: boolean
    • Default value: 0
  • VPM1_URLs_Authentication
    Role: Activate URLs authentication with the web application server
    • Type: string
    • Default value: false
