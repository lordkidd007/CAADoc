CATLiveShapeSettings Setting Repository
Quick Reference

CATLiveShapeSettings

Role: These settings define the user choice for CATIA Live Shape environment

Detailed role:These settings define the user choice of the display the referent who allows you to get a better idea of the scale factor.

The CATLiveShapeSettings repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/CATIA Live Shape/Display
  • Repository File: CATLiveShapeSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATLiveShapeSettings setting repository attributes:

  • CurrentContextLabel
    Role: Industrial context in use
    Detailed role: Display the current used industrial context
    • Type: boolean
    • Default value: true
  • DrawBoolean
    Role: Display draw combine operation options
    Detailed role: Makes available new shapes combination with existing geometry options
    • Type: boolean
    • Default value: false
  • FilterConstraint
    Role: Define the constraints diplay mode of the constraint
    Detailed role: Define wether the constraints must be displayed or not, according to the choosen option.
    • Type: integer
    • Default value: 3
    • Default Value Role: All the constraints are shown
    • Values:
      0 All the constraints are hidden
      1 The constraints related to the selected objects are shown
      2 The constraints related to the network of the selected objects are shown
      3 All the constraints are shown
  • GhostOutOfContext
    Role: Define wether the objects that are on a truntable are oriented towards its center.
    Detailed role: Define wether the objects that are on a truntable are oriented towards its center.
    • Type: boolean
    • Default value: true
  • IndustryContext
    Role: Industrial context
    Detailed role: The current used industrial context
    • Type: integer
    • Default value: 0
  • Interrupt
    Role: Define wether long operations will be interupted to keep live interactions
    Detailed role: Define wether partial results are allowed during manipulations
    • Type: boolean
    • Default value: true
  • LinearizationFactor
    Role: Defines the size linearization factor of the objects on a table.
    Detailed role: Defines the size linearization factor of the objects on a table.
    • Type: float
    • Default value: 0.0
    • Min value: 0.0
    • Max value: 1.0
    • Step: 0.05
  • OrientOnTables
    Role: Defines wether the elements that are ousite of the active object are displayed in ghost
    Detailed role: Defines wether the elements of the product structure that are ousite of the active representation or product are displayed in ghost
    • Type: boolean
    • Default value: true
  • Referent
    Role: Define wether the referent is displayed
    Detailed role: Define wether the referent who allows you to get a better idea of the scale factor is displayed.
    • Type: boolean
    • Default value: true
  • ReframeOnInsert
    Role: When inserting a new element, should viewer automatically reframe on the inserted element ?
    Detailed role: When inserting a new element, should viewer automatically reframe on the inserted element ? Check for YES
    • Type: boolean
    • Default value: true
  • ResetContext
    Role: Reset industrial context
    Detailed role: Reset chosen industrial context
    • Type: boolean
    • Default value: true
  • SpecTree
    Role: Define wether the specification's tree is hidden or not
    Detailed role: Define wether the specification's tree is hidden when entering CATIA Live Shape environment.
    • Type: boolean
    • Default value: true
