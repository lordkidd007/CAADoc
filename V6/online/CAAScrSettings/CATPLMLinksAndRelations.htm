CATPLMLinksAndRelations

Role: Setting to handle the Links and Relations Edition options

Detailed role:Setting to handle the Links and Relations Edition options

The CATPLMLinksAndRelations repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/CATPLMLinksAndRelations
  • Repository File: CATPLMLinksAndRelations
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATPLMLinksAndRelations setting repository attributes:

  • EastCompassInLinksAndRelations
    Role: Attribute to manage East Compass activation when editing Links and Relations
    Detailed role: The East Compass is activated when editing Links and Relations according to this attribute's value
    • Type: integer
    • Default value: 0
    • Default Value Role: By default East Compass is deactivated when editing Links and Relations
    • Values:
      0 The East Compass is deactivated when editing Links and Relations
      1 The East Compass is activated when editing Links and Relations (potential bad performances)
