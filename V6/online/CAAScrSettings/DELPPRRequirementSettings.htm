DELPPRRequirementSettings Setting Repository
Quick Reference

DELPPRRequirementSettings

Role: Settings for Requirements in Manufacturing Planning

The DELPPRRequirementSettings repository has the following characteristics:

  • Primary Tab Page: /Requirements in Manufacturing Planning
  • Repository File: DELPPRRequirementSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELPPRRequirementSettings setting repository attributes:

  • AllowIgnoreUsageCheckButton
    Role: Include an Ignore usage
    Detailed role: Add a usage Ignore in the list of available usages
    • Type: boolean
    • Default value: false
  • ManageRequirementUsagesCheckButton
    Role: Management of usages for the Requirements
    Detailed role: Activate the usages management when managing Requirements
    • Type: boolean
    • Default value: false
