Tree Setting Repository
Quick Reference

Tree

Role: Setting to handle the specification tree options

The Tree repository has the following characteristics:

  • Primary Tab Page: /General/Display/Tree Appearance
  • Repository File: Tree
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CATIAApplicationFrame
    • Framework_IDL: CATIAApplicationFrame
    • Interface_CPP: CATICafTreeVizManipSettingAtt
    • Interface_IDL: CATIATreeVizManipSettingAtt
    • LateType: CATCafTreeVizManipSettingCtrl

Tree setting repository attributes:

  • ArcSelection
    Role: Arc-selection mode applied to the specification tree.
    • Type: boolean
    • Default value: true
  • AutoExpand
    Role: Automatic expand mode applied to the specification tree.
    • Type: boolean
    • Default value: false
  • AutoScroll
    Role: Automatic scrolling mode applied to the specification tree.
    • Type: boolean
    • Default value: true
  • DisplayGeometry
    Role: "Display geometry on scrolling" mode applied to the specification tree.
    • Type: boolean
    • Default value: true
  • LeavesDependant
    Role: Type of size applied to the text of the specification tree.
    • Type: integer
    • Default value: 1
    • Values:
      0 catTreeSizeTypeFixedSize
      1 catTreeSizeTypeTextDependentSize
  • LeavesSize
    Role: Number of characters shown for the text of the specification tree in case where LeavesDependant is equal to catTreeSizeTypeFixedSize.
    • Type: integer
    • Default value: 8
    • Min value: 0
    • Max value: 1000
  • Orientation
    Role: Orientation applied to the specification tree.
    • Type: string
    • Default value: V
    • Values:
      V catTreeOrientationVertical
      H catTreeOrientationHorizontal
  • ShowNoShow
    Role: Visualization Show/NoShow's mode applied to the specification tree.
    • Type: boolean
    • Default value: true
  • StructState
    Role: Type of the specification tree.
    • Type: integer
    • Default value: 1
    • Values:
      1 catTreeTypeClassical
      2 catTreeTypeStructural
      3 catTreeTypeHistorical
      4 catTreeTypeRelational
