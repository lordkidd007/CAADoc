AUTLucidDisplay Setting Repository
Quick Reference

AUTLucidDisplay

Role: State chart editor options

Detailed role:State chart editor options

The AUTLucidDisplay repository has the following characteristics:

  • Primary Tab Page: /Automation/StateChart Editor
  • Repository File: AUTLucidDisplay
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: AUTSceEditor
    • Framework_IDL: AUTSceEditor
    • Interface_CPP: AUTILucidDisplaySettingAtt
    • Interface_IDL: AUTILucidDisplaySettingAtt
    • LateType: CATSysSettingCtrl

AUTLucidDisplay setting repository attributes:

  • SCEnableSustainActions
    Role: Enable sustain actions
    • Type: integer
    • Default value: 0
  • SCLinkColor
    Role: Transitions Color
    • Type: integer
    • Default value: 0
  • SCLinkThick
    Role: Transitions thickness
    • Type: integer
    • Default value: 0
  • SCPreemptionThick
    Role: Preemption draw thickness
    • Type: integer
    • Default value: 0
  • SCStateColor
    Role: State lines color
    • Type: integer
    • Default value: 0
  • SCStateHeaderColor
    Role: State header color
    Detailed role: OxRRGGBB in Hexadecimal translated in decimal
    • Type: integer
    • Default value: 16775890
  • SCTransitionEntry
    Role: Transition entry mode
    • Type: integer
    • Default value: 2
    • Values:
      1 Resume
      2 Reset
  • SCTransitionMode
    Role: Delay show mode
    • Type: integer
    • Default value: 1
    • Values:
      1 With spaces near center
      2 with circles near center
      3 With circles near extremities
  • SCTransitionPreemption
    Role: Transition preemption mode
    • Type: integer
    • Default value: 1
    • Values:
      1 Weak
      2 Strong
