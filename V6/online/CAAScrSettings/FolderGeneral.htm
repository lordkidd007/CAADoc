FolderGeneral Setting Repository
Quick Reference

FolderGeneral

Role: Folder options

Detailed role:This option enables to modify Folder navigator behaviors.

The FolderGeneral repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/Folder
  • Repository File: FolderGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

FolderGeneral setting repository attributes:

  • BackgroundColor
    Role: Background color of the navigator
    • Type: unsigned integer
    • Default value: 223
    • Default Value Role: Metal grey
    • Min value: 0
    • Max value: 255
  • Navigation3D
    Role: Active the 3D Navigation
    Detailed role: Active the 3D Navigation.
    • Type: boolean
    • Default value: true
  • ReferenceExpandMode
    Role: Could references be expanded or not
    • Type: boolean
    • Default value: false
    • Default Value Role: By default, references are not expandable
