ConstraintModeler Setting Repository
Quick Reference

ConstraintModeler

Role: Setting to manage Constraints and Dimensions visualization options

Detailed role:Constraints and Dimensions setting contains all options necessary to manage Constraints and Dimensions visualization

The ConstraintModeler repository has the following characteristics:

  • Primary Tab Page: /General/Parameters and Measures/Constraints and Dimensions
  • Repository File: ConstraintModeler
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ConstraintModeler setting repository attributes:

  • DisplayIconifiedElements
    Role: Attribute to display the iconified constrained elements
    Detailed role: Defines whether the iconified constrained elements are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default iconified constrained elements are displayed
    • Values:
      0 Does not display iconified constrained elements
      1 Displays iconified constrained elements
  • DisplayMode
    Role: Attribute to define the constraint display mode
    Detailed role: Defines the constraint display mode.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default only the constraint value is displayed
    • Values:
      0 The constraint value is displayed
      1 The constraint name is displayed
      2 The constraint value and name are displayed
      3 The constraint valuen name and formula are displayed
  • Filter_Angle
    Role: Attribute to display angles constraints
    Detailed role: Defines whether angles constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the angles constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Chamfer
    Role: Attribute to display chamfer constraints
    Detailed role: Defines whether chamfer constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the chamfer constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Coincidence
    Role: Attribute to display coincidence constraints
    Detailed role: Defines whether coincidence constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the coincidence constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Concentricity
    Role: Attribute to display concentricity constraints

    • Default Value Role: By default the concentricity constraints are displayed

  • Filter_Dimensions
    Role: Attribute to display dimension constraints
    Detailed role: Defines whether dimension constraints are displayed.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the dimension constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • Filter_Driven
    Role: Attribute to display driven constraints
    Detailed role: Defines whether driven constraints are displayed.
    • Type: integer
    • Default value: 1

    • Values:

  • Filter_Driving

    • Type: integer
    • Default value: 1
    • Default Value Role: By default the tangency constraints are displayed
    • Values:
      0 Constraints are not displayed
      1 Constraints are displayed
  • FilterMode
    Role: Attribute to define the general filter options
    Detailed role: Defines the general filter options.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default all constraints are displayed
    • Values:
      0 No constraints are displayed
      1 All constraints are displayed
      2 Constraints are displayed according to the various filter attributes
  • Gap
    Role: Attribute to define the distance between the constrained geometries and constraint extension lines
    Detailed role: Defines the distance between the constrained geometries and constraint extension lines, in mm.
    • Type: double
    • Default value: 2
    • Default Value Role: By default the view angle is 2 mm
  • HighlightElements
    Role: Attribute to highlight the constrained geometries
    Detailed role: Defines whether the constrained geometries are highlighted when you pre-select a constraint.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default the constrained geometries are highlighted
    • Values:
      0 The constrained geometries are not highlighted
      1 The constrained geometries are highlighted
  • InvalidGeometryColor
    Role: Invalid Geometry Constraints color in 3D viewer
    Detailed role: This attribute defines the color status for constraints set on a geometry became invalid in the 3D viewer. The three integer are the Red, Green, Blue coefficients
    • Type: integer
    • Default value: 46
    • Default Value Role: By default, the color is dark orange
    • Min value: 0
    • Max value: 256
  • OverConstraintColor
    Role: Overconstrained color in 3D viewer
    Detailed role: This attribute defines the color status for overconstrained in the 3D viewer. The three integer are the Red, Green, Blue coefficients
    • Type: integer
    • Default value: 152
    • Default Value Role: By default, the color is purple
    • Min value: 0
    • Max value: 256
  • OverRun
    Role: Attribute to define the distance between the constraint dimension lines and constraint extension lines
    Detailed role: Defines the distance between the constraint dimension lines and constraint extension lines, in mm.
    • Type: double
    • Default value: 2
    • Default Value Role: By default the view angle is 2 mm
  • ProductFilterMode
    Role: Attribute to define the product filter options
    Detailed role: Defines whether the constraints of the active product only are displayed.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default constraints of any product are displayed
    • Values:
      0 Constraints of any product are displayed
      1 The constraints of the active product only are displayed
  • ResolvedConstraintColor
    Role: Resolved Constraints color in 3D viewer
    Detailed role: This attribute defines the color status for resolved constraints in the 3D viewer. The three integer are the Red, Green, Blue coefficients
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, the color is black
    • Min value: 0
    • Max value: 256
  • Scale
    Role: Attribute to define the dimension style options
    Detailed role: Defines the size of the graphical symbols for iconic constraint.
    • Type: integer
    • Default value: 1
    • Default Value Role: By default medium symbols are displayed
    • Values:
      0 Displays small symbols
      1 Displays medium symbols
      2 Displays large symbols
  • ShowColorStatus
    Role: Attribute to display the status color
    Detailed role: Defines if the default status color or the graphical user defined color is displayed on constraint.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default the default status color is used
    • Values:
      0 The graphical user defined color is used
      1 The Update command will be automatically launched
  • StatusFilterMode
    Role: Attribute to define the status filter options
    Detailed role: Defines which constraints are displayed, according to their status.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default constraints status are not taken into account
    • Values:
      0 Constraints are displayed without take into account their status
      1 Only verified constraints are displayed
      2 Only unverified constraints are displayed
  • UnresolvedConstraintColor
    Role: Unresolved Constraints color in 3D viewer
    Detailed role: This attribute defines the color status for unresolved constraints in the 3D viewer. The three integer are the Red, Green, Blue coefficients
    • Type: integer
    • Default value: 47
    • Default Value Role: By default, the color is dark red
    • Min value: 0
    • Max value: 256
  • ViewAngle
    Role: Attribute to define the constraint view angle
    Detailed role: Defines the angle in degrees from which the constraint is displayed in the geometry area, when you are rotating the geometry. This angle is measured between the normal to the screen and the constraint direction.
    • Type: double
    • Default value: 1
    • Default Value Role: By default the view angle is 1 deg
    • Min value: 0
    • Max value: 180
