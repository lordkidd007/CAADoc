DNBTrjTrajectorySettings Setting Repository
Quick Reference

DNBTrjTrajectorySettings

Role: Settings to handle the parameters of the Trajectory Settings

Detailed role:This setting allows to customize the appearance and the behavior of Trajectory Settings

The DNBTrjTrajectorySettings repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/Trajectory Infrastructure/Trajectory
  • Repository File: DNBTrjTrajectorySettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBTrjTrajectorySettings setting repository attributes:

  • AutoAnchorPosition
    Role: Position of Anchor Relative to First Moving Object
    Detailed role: How the anchor is positioned whenever the first moving object is set for a trajectory
    • Type: short integer
    • Default value: 0
    • Default Value Role: By default, anchor positioned by PartBoundingCenter
    • Values:
      0 PartBoundingCenter
      1 PartOrigin
  • LineColor
    Role: The Trajectory Line Color
    Detailed role: The trajectory line color set by the user
    • Type: integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • LineType
    Role: The Trajectory Line Type
    Detailed role: The trajectory line type
    • Type: integer
    • Default value: 1
    • Default Value Role: LineType
  • LineWeight
    Role: The Trajectory Line Weight
    Detailed role: The trajectory line weight
    • Type: short integer
    • Default value: 1
    • Default Value Role: LineWeight
  • PointColor
    Role: The Trajectory Point Color
    Detailed role: The points in the trajectory that are set
    • Type: integer
    • Default value: 255
    • Default Value Role: Blue in RGB order
    • Min value: 0
    • Max value: 255
  • PointPattern
    Role: Get the Trajectory Point Pattern
    Detailed role: The trajectory point pattern
    • Type: short integer
    • Default value: 1
    • Default Value Role: PointPattern
  • TrajectoryMode
    Role: Mode of Trajectory
    Detailed role: The trajectory mode that can be run
    • Type: short integer
    • Default value: 0
    • Default Value Role: By default mode is time
    • Values:
      0 TrjTime
      1 TrjSpeed
  • TrajectoryName
    Role: The name of the trajectory
    Detailed role: The trajectory name that is set
    • Type: string
    • Default value: Track
  • TrajectorySpeed
    Role: The Trajectory speed
    Detailed role: The Trajectory speed
    • Type: double
    • Default value: 0.001
    • Default Value Role: TrajectorySpeed
  • TrajectoryTime
    Role: The Trajectory Time
    Detailed role: The Trajectory Time
    • Type: double
    • Default value: 5
    • Default Value Role: TrajectoryTime
