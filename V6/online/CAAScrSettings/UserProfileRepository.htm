UserProfileRepository Setting Repository
Quick Reference

UserProfileRepository

Role: Setting to handle User Profile parameters

The UserProfileRepository repository has the following characteristics:

  • Primary Tab Page: /General
  • Repository File: UserProfileRepository
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

UserProfileRepository setting repository attributes:

  • ProfileCreated
    Role: Parameter know if a profile has been created
    • Type: integer
    • Default value: 0
  • ProfileName
    Role: Parameter to set the User Profile
    • Type: string
    • Default value: 3DEXPERIENCEPROFILE
