CATBisBdcOptions Setting Repository
Quick Reference

CATBisBdcOptions

Role: Options for BioSystems Deterministic Simulation

Detailed role:These options allow user to customize the behavior of BioSystems Deterministic Simulation

The CATBisBdcOptions repository has the following characteristics:

  • Primary Tab Page: /Simulation/BioSystems Experiment/Deterministic Simulation
  • Repository File: CATBisBdcOptions
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATBisBdcOptions setting repository attributes:

  • BdcDuration
    Role: Duration
    • Type: double
    • Default value: 60.
  • BdcForceTimeHorizon
    Role: Force Time Horizon
    • Type: integer
    • Default value: 0
  • BdcMethod
    Role: Algorithm
    • Type: string
    • Default value: DASSL
    • Values:
      DASSL DASSL algorithm
      Esdirk45a Esdirk45a algorithm
      Dopri45 Dopri45 algorithm
      Dopri853 Dopri853 algorithm
      Sdirk34hw Sdirk34hw algorithm
      Cerk23 Cerk23 algorithm
      Cerk34 Cerk34 algorithm
      Cerk45 Cerk45 algorithm
      LSODAR LSODAR algorithm
      EULER EULER algorithm
      RKFIX2 RKFIX2 algorithm
      RKFIX3 RKFIX3 algorithm
      RKFIX4 RKFIX4 algorithm
      Radau IIa Radau IIa algorithm
      Esdirk23a Esdirk23a algorithm
      Esdirk34a Esdirk34a algorithm
  • BdcStartTime
    Role: Start time
    • Type: double
    • Default value: 0.
  • BdcStepSize
    Role: Step size
    • Type: double
    • Default value: 0.1
  • BdcStepType
    Role: Step type
    • Type: integer
    • Default value: 0
    • Values:
      0 Step Size
      1 Interval Number
  • BdcTimeHorizon
    Role: Time Horizon
    • Type: integer
    • Default value: 100
  • BdcTolerance
    Role: Tolerance
    • Type: double
    • Default value: 0.00001
