SpecV4 Setting Repository
Quick Reference

SpecV4

Role: Setting to handle the V4-V5 as spec migration options

The SpecV4 repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/V4 V5 SPEC
  • Repository File: SpecifV4ToV5
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CATIAV4Interfaces
    • Framework_IDL: CATIAV4Interfaces
    • Interface_CPP: CATIV4ISpecV4SettingAtt
    • Interface_IDL: CATIASpecV4SettingAtt
    • LateType: CATV4ISpecV4SettingCtrl

SpecV4 setting repository attributes:

  • DraftConeMode
    Role: The Draft Feature Migration Mode setting parameter manages the mode of draft migration when copying as spec.
    • Type: integer
    • Default value: 0
    • Values:
      0 Square Mode
      1 Cone Mode
  • Step_Update
    Role: The Step By Step Update And Reroute setting parameter manages the mode reroute when copying as spec.
    • Type: integer
    • Default value: 262144
    • Values:
      262144 Step By Step Update and Reroute
      524288 No Step By Step Update and Reroute
