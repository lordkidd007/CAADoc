CATBatchSettings Setting Repository
Quick Reference

CATBatchSettings

Role: Setting to handle PLMBatchs parameters

The CATBatchSettings repository has the following characteristics:

  • Primary Tab Page: /General/PLMBatchs
  • Repository File: CATBatchSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATBatchSettings setting repository attributes:

  • BatchHomeDir
    Role: Root directory for batch logs and traces
    • Type: string
    • Default value:
  • GridActivation
    Role: Enable Batch to be started on Grid middleware
    • Type: boolean
    • Default value: false
  • GridAllowVisu
    Role: Allow batch to use grahical adapter if needed
    • Type: boolean
    • Default value: false
  • GridDirEnv
    Role: Directory for node local Environment for starting Batch on the nodes
    • Type: string
    • Default value:
  • GridDriver
    Role: Parameter to define the grid middleware implementation
    • Type: string
    • Default value: MS CCS
  • GridEnvName
    Role: Environment Name for starting Batch on the nodes
    • Type: string
    • Default value:
  • GridHeadNode
    Role: Head node of the grid
    • Type: string
    • Default value:
  • GridLicFile
    Role: Path for license information for batches
    • Type: string
    • Default value:
