SWKHmiHBRGeneralSetting Setting Repository
Quick Reference

SWKHmiHBRGeneralSetting

Role: Setting to manage Human Builder general parameters

Detailed role:This setting allows to customize the general parameters for the human builder products

The SWKHmiHBRGeneralSetting repository has the following characteristics:

  • Primary Tab Page: /Ergonomics Design Analysis/Human Builder/General
  • Repository File: SWKHmiHBRGeneralSetting
  • Version: 6
  • Full Compatibility: 2
  • Upward Compatibility: 2

SWKHmiHBRGeneralSetting setting repository attributes:

  • CustomReportIdEnableState
    Role: This attribute enable the user to set the Analysis report ID
    • Type: integer
    • Default value: 0
    • Default Value Role: By default disabled
    • Default Value Detailed Role the value can be 0(disable) or 1(enable)
  • CylinderDefaultHumanInterfaceSkill
    Role: Default skill for cylinder-type Human Interface
    Detailed role: Define the default skill of Human Interface which has cylinder type
    • Type: integer
    • Default value: 3
  • DefaultSide
    Role: Default side
    Detailed role: Define the default side
    • Type: integer
    • Default value: 1
  • DiskDefaultHumanInterfaceSkill
    Role: Default skill for disk-type Human Interface
    Detailed role: Define the default skill of Human Interface which has disk type
    • Type: integer
    • Default value: 9
  • DynClashType
    Role: Enable the user to check clash on the manikins
    • Type: integer
    • Default value: 0
  • HandGraspLockSelectedSegment
    Role: Enable the user to lock the selected segment when the Hand Grasp is activate
    • Type: integer
    • Default value: 0
    • Default Value Role: By default disabled
    • Default Value Detailed Role the value can be 0(disable) or 1(enable)
  • HeelHeight
    Role: manikin shoe heel height value
    Detailed role: manikin shoe heel height value
    • Type: double
    • Default value: 0.0
  • IKModeInterDeleteState
    Role: Enable the user to delete the twin segment on second click
    Detailed role: This attribute is used in Reach and IK Mode activities
    • Type: integer
    • Default value: 1
    • Default Value Role: By default available
    • Default Value Detailed Role The value can be 0(unavailable) or 1(available)
  • IntersegmentMode
    Role: Angle of reference mode
    Detailed role: This attribute is used to set the mode of reference for the angle
    • Type: integer
    • Default value: 1
    • Default Value Role: By default anglemode
    • Default Value Detailed Role It can be 1(angle mode) or 0(inter-segment)
  • LookAt
    Role: Enable the manikin to Look At the position reached during IK Mode or Reach Mode
    • Type: integer
    • Default value: 0
    • Default Value Role: By default disabled
    • Default Value Detailed Role the value can be 0(disable) or 1(enable)
  • ReportPath
    Role: Enable the user to set the default location for reports
    • Type: URN
    • Default value: C:\temp
    • Default Value Role: By default a temporary directory
    • Default Value Detailed Role the value can be any valid directory path
  • SoleHeight
    Role: manikin shoe sole height value
    Detailed role: manikin shoe sole height value
    • Type: double
    • Default value: 0.0
  • SphereDefaultHumanInterfaceSkill
    Role: Default skill for sphere-type Human Interface
    Detailed role: Define the default skill of Human Interface which has sphere type
    • Type: integer
    • Default value: 2
  • TextureExport
    Role: Enable the user to export the textures of the manikins
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, textures are not exported
    • Default Value Detailed Role It can be 1(textures are exported) or 0(textures are not exported)
