SWXGeneralSettings Setting Repository
Quick Reference

SWXGeneralSettings

Role: General settings

The SWXGeneralSettings repository has the following characteristics:

  • Primary Tab Page: /Mechanical Design/SolidWorks V6/General
  • Repository File: SWXGeneralSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SWXGeneralSettings setting repository attributes:

  • AutoSwitchToShapeSection
    Role: Auto switch to Shape section
    Detailed role: Auto switch to Shape section when creating/editing a shape feature
    • Type: boolean
    • Default value: true
  • AutoSwitchToSketchSection
    Role: Auto switch to Sketch section
    Detailed role: Auto switch to Sketch section when sketch is active
    • Type: boolean
    • Default value: true
  • AutoSwitchToViewSection
    Role: Auto switch to View section
    Detailed role: Auto switch to View section when View Selector is active
    • Type: boolean
    • Default value: true
  • WarnOnEditWhenLocked
    Role: Warn On Edit when product is locked
    Detailed role: Warn On Edit when product is locked
    • Type: boolean
    • Default value: false
