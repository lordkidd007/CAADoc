CATDesSettingGeneral Setting Repository
Quick Reference

CATDesSettingGeneral

Role: Settings to manage Imagine And Shape General Options

The CATDesSettingGeneral repository has the following characteristics:

  • Primary Tab Page: /Shape/Imagine And Shape/General
  • Repository File: CATDesSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CATDesSettingGeneral setting repository attributes:

  • Attenuation
    Role: Attenuation value
    • Type: double
    • Default value: 0.5
    • Min value: 0.0
    • Max value: 1.0
  • CommandsOptionMenuMode
    Role: Command Option Menu Mode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • DepthUpdateChilds
    Role: Depth for Update Propagation to Children
    • Type: integer
    • Default value: 0
  • IsAttenuationOn
    Role: Attenuation for mouse move
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • IsAutomaticWeightOn
    Role: Automatic Weight On Vertices
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsQuitWithoutApplyOn
    Role: Warning when Quit without Apply
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • IsRepeatLastOn
    Role: Quick Switch with modification command
    • Type: integer
    • Default value: 1
    • Min value: 0
    • Max value: 1
  • PrimitiveCreationCenterMode
    Role: Primitive Creation Center Mode
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • SubdivPreviewLevel
    Role: Level of subdivision for preview
    • Type: integer
    • Default value: 2
    • Min value: 0
    • Max value: 6
