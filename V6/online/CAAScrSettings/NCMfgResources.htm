NCMfgResources Setting Repository
Quick Reference

NCMfgResources

Role: Lists the settings of the resources used in the Machining applications

Detailed role:The resources machining settings are useful to manages options on tools and ressources catalogs

The NCMfgResources repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Resources
  • Repository File: NCMfgResources
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgResources setting repository attributes:

  • FeedAutoUpd
    Role: Automatic compute of feedrate attributes of the operation
    Detailed role: Automatic compute of feedrates attributes of the operation
    • Type: boolean
    • Default value: true
  • FSConfigKey1List
    Role: List of key1 for feeds and speeds configurations
    • Type: string
    • Default value: Other Metal
  • FSConfigKey1Type
    Role: Type of key1 for feeds and speeds configurations
    • Type: string
    • Default value: Machined Material
  • FSConfigKey2List
    Role: List of key2 for feeds and speeds configurations
    • Type: string
    • Default value: Reamer
  • FSConfigKey2Type
    Role: Type of key2 for feeds and speeds configurations
    • Type: string
    • Default value: Machining Type
  • FSConfigKey3List
    Role: List of key3 for feeds and speeds configurations
    • Type: string
    • Default value: No quality
  • FSConfigKey3Type
    Role: Type of key3 for feeds and speeds configurations
    • Type: string
    • Default value: Machining Quality
  • HolderAttachmentNumberOfSubTypes
    Role: Holder Attachment Number Of SubTypes
    Detailed role: List holding number of subtypes for each Holder Attachment type
    • Type: unsigned integer
    • Default value: 4
  • HolderAttachmentSubTypes
    Role: Holder Attachment SubTypes
    Detailed role: List of SubTypes for all Holder Attachment Types
    • Type: string
    • Default value: 32
    • Default Value Role: List of Tool Attachment Types2
  • HolderAttachmentTypes
    Role: List of Holder Attachment Types
    Detailed role: List of Standard ISO Holder Attachment Types
    • Type: string
    • Default value: CUBIC SHANK
    • Default Value Role: List of Tool Attachment Types
  • NCMfgCriteriaNameforReplaceResource
    Role: Criteria for Replace Resource
    Detailed role: Criteria for Replace Resource
    • Type: integer
    • Default value: 1
    • Default Value Role: By default check on Geometry parameters
    • Values:
      0 Name
      1 Geometry Parameters
  • NCMfgCriteriaStdProjectReplaceResource
    Role: Specify the name of Standard Project
    • Type: string
    • Default value:
  • NCMfgCriteriaTechnoforReplaceResource
    Role: Add Technological Parameters in Criteria
    Detailed role: Add Technological Parameters in Criteria
    • Type: boolean
    • Default value: false
  • NCMfgNoToolInMachProcess
    Role: Interactive Tool selection if no tool is found
    Detailed role: Interactive Tool selection if no tool is found
    • Type: boolean
    • Default value: false
  • NCMfgToolQueryInMachProcess
    Role: Tool Query mode in Machining Process instanciation
    Detailed role: Tool Query mode in Machining Process instanciation
    • Type: integer
    • Default value: 0
    • Default Value Role: By default an Automatic Tool Query is done for Machining Process instanciation
    • Values:
      0 Automatic Tool Query for Machining Process instanciation
      1 Interactive Tool selection in case of multiple results forMachining Process instanciation
  • PPToolsLocation
    Role: Defines the directory where to find catalogs and files for tools and PP tables
    Detailed role: Defines the directory where to find catalogs and files for tools and PP tables
    • Type: URN
    • Default value: UNIX: ${CATStartupPath} Windows: ${CATStartupPath}
  • SpindleAutoUpd
    Role: Automatic compute of spindle attributes of the operation
    Detailed role: Automatic compute of spindle attributes of the operation
    • Type: boolean
    • Default value: true
  • SpindleDisplayArrow
    Role: Spindle Display Arrow color
    Detailed role: Spindle Display Arrow color
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: Violet Blue in RGB order
    • Min value: 0
    • Max value: 255
  • SpindleDisplayArrowHead

  • ToolAttachmentSubTypes

  • ToolAttachmentTypes
