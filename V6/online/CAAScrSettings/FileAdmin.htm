FileAdmin Setting Repository
Quick Reference

FileAdmin

Role: Setting to handle the parameters of the File Options user interface for the Open/Save/choose commands

Detailed role:This setting allows to customize the behavior of the File Open/Save/Choos commands through the user interface

The FileAdmin repository has the following characteristics:

  • Primary Tab Page: /General/TEST/File Access Administration
  • Repository File: FileAdmin
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

FileAdmin setting repository attributes:

  • UpperCaseMode
    Role: Force the File names in Upper Case
    Detailed role: This option allows to create only Upper Caes File names.
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Case sensitif mode
    • Values:
      0 Case sensitif mode
      1 Upper Case mode
