FstDisplay Setting Repository
Quick Reference

FstDisplay

Role: These setting Parameters define the default Fastener Display Parameters

Detailed role:These Fst Display setting Parameters are Fastener Technological Parameters, responsible for the symbol support, color etc

The FstDisplay repository has the following characteristics:

  • Primary Tab Page: /Infrastructure/FastenerInfrastructure/Display
  • Repository File: FstDisplay
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

FstDisplay setting repository attributes:

  • FstAnalysisColors
    Role: Indicate whether to display the fasteners, fastened parts and non-fastened parts with colors in Fasteners Analysis
    • Type: boolean
    • Default value: true
  • FstAnalysisEffectivity
    Role: Indicate whether to display the fasteners and fastened parts effectivities inside the labels in Fasteners Analysis
    • Type: boolean
    • Default value: true
  • FstAnalysisLabels
    Role: Indicate whether to display the fasteners and fastened parts labels in Fasteners Analysis
    • Type: boolean
    • Default value: true
  • FstOverrideFstInstAlias
    Role: Indicate whether to display the overriden alias of the fastener instances in the UI
    • Type: boolean
    • Default value: false
  • FstOverridePartInstAlias
    Role: Indicate whether to display the overriden alias of the fastened part instances in the UI
    • Type: boolean
    • Default value: false
  • FstSymbolColor
    Role: Fastener technological parameter assigned to Symbol Color
    • Type: integer
    • Default value: 1
    • Values:
      0 0 (Unspecified) : no specification is assigned to Symbol Color
      1 1: Process Category is assigned to Symbol Color
      2 2: Diameter value is assigned to Symbol Color
      3 3: (None) : no color is assigned to Symbol
  • FstSymbolMark
    Role: Fastener technological parameter assigned to Symbol Mark
    • Type: string
    • Default value: None
    • Default Value Role: Any classification attribute present in Fst_Standard file can be assigned to Symbol Mark or None
  • FstSymbolSupport
    Role: : Fastener Technological parameter assigned to Symbol
    • Type: integer
    • Default value: 1
    • Values:
      0 0 (Unspecified) : no specification is assigned to Symbol
      1 1: Process Type is assigned to Symbol
      2 2: Zone Count is assigned to Symbol
      3 3: (None) : no symbol support is displayed
