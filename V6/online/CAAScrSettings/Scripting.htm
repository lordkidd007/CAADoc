Scripting Setting Repository
Quick Reference

Scripting

Role: Setting to handle the Automation parameters

The Scripting repository has the following characteristics:

  • Primary Tab Page: /General/Macros
  • Repository File: Scripting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

Scripting setting repository attributes:

  • CATScriptEditor
    Role: Editor path for the .CATScript files
    • Type: string
    • Default value:
  • CATVBSEditor
    Role: Editor path for the .catvbs files
    • Type: string
    • Default value:
  • DefaultMacroLibraries
    Role: List of default macro libraries
    Detailed role: Represents the list of macro libraries (directories or .catvba projects) that are opened by default, in addition to the own user's list of macro libraries.
    • Type: string
    • Default value:
  • DefaultPLMMacroLibraries
    Role: List of default PLM macro libraries
    Detailed role: Represents the list of PLM macro libraries that are opened by default, in addition to the own user's list of macro libraries.
    • Type: string
    • Default value:
  • ExternalReferences
    Role: List of external references
    Detailed role: Represents the list of external references paths (Type Library files) used in macros, in addition to the internal V5 Type Library files.
    • Type: string
    • Default value:
