IVDevice Setting Repository
Quick Reference

IVDevice

Role: Setting to handle the devices parameters

Detailed role:Setting to handle the devices parameters

The IVDevice repository has the following characteristics:

  • Primary Tab Page: /General/VirtualReality
  • Repository File: IVDevice
  • Version: 2
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Relation;
    • Role: Number of devices to run automatically at startup
    • Detailed role:
    • src: NumberOfStartupDevices
    • dest: StartupDevices
    • kind: dimensionOfArray

IVDevice setting repository attributes:

  • AutoSpaceTargetMode
    Role: Determines automatically the rotation center when using a SpaceMouse
    • Type: unsigned integer
    • Default value: 1
  • MaxFrequency
    Role: Determines the maximum number of events sent by devices per second (0 means no limit)
    • Type: unsigned integer
    • Default value: 0
  • NumberOfStartupDevices
    Role: Number of devices to run automatically at startup
    • Type: unsigned integer
    • Default value: 0
  • RobotSensitivity
    Role: Determines if robot is sensitive to manipulation through 3D devices
    • Type: boolean
    • Default value: true
  • ShowSpaceTargetOpt
    Role: Options for displaying rotation center when using a SpaceMouse
    • Type: unsigned integer
    • Default value: 1
  • StartupDevices
    Role: Names of devices to run automatically at startup
    • Type: string
