ActionGeneral Setting Repository
Quick Reference

ActionGeneral

Role: Action options

Detailed role:This option enables to modify Action Editor behaviors.

The ActionGeneral repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 VPM/Action
  • Repository File: ActionGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ActionGeneral setting repository attributes:

  • ActionEditorSingleMode
    Role: Is the editor displaying a single Action or a list of Actions
    • Type: boolean
    • Default value: true
    • Default Value Role: By default, Action Editor displays a single Action
  • BackgroundColor
    Role: Background color of the navigator
    • Type: unsigned integer
    • Default value: 223
    • Default Value Role: Metal grey
    • Min value: 0
    • Max value: 255
  • ChgValMode
    Role: Enable the Change Validation Basic Mode
    Detailed role: Enable the Change Validation Basic Mode.
    • Type: boolean
    • Default value: false
  • ImpactedPartsColor
    Role: Color of impacted parts by NE for Show Changes command
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red
    • Min value: 0
    • Max value: 255
  • Navigation3D
    Role: Active the 3D Navigation
    Detailed role: Active the 3D Navigation.
    • Type: boolean
    • Default value: true
  • NonImpactedPartsColor
    Role: Color of non-impacted parts by NE for Show Changes command
    • Type: unsigned integer
    • Default value: 255
    • Default Value Role: White
    • Min value: 0
    • Max value: 255
  • ProgramSteeringMode
    Role: Is the "ECA association as change task deliverable" enabled
    • Type: boolean
    • Default value: false
    • Default Value Role: By default, association to change task is disabled
