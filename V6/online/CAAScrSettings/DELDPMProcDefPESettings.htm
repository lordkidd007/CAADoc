DELDPMProcDefPESettings Setting Repository
Quick Reference

DELDPMProcDefPESettings

Role: Settings for Process Editor UI

The DELDPMProcDefPESettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/Process Editor
  • Repository File: DELDPMProcDefPESettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELDPMProcDefPESettings setting repository attributes:

  • ColorDataReq
    Role: Color of Data Requirement
    • Type: unsigned integer
    • Default value: 744335360
    • Default Value Role: middle Blue in RGB order
  • ColorNodeBG
    Role: Background color for process node
    • Type: unsigned integer
    • Default value: 3368615680
    • Default Value Role: light Violet in RGB order
  • ColorNodeLN
    Role: Edge color for process node
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black in RGB order
  • ColorParentNodeBG
    Role: Background color for parent process node
    • Type: unsigned integer
    • Default value: 4294967040
    • Default Value Role: White in RGB order
  • ColorParentNodeLN
    Role: Edge color for parent process node
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black in RGB order
  • ColorPrecConstr
    Role: Color of Precedence Constraints
    • Type: unsigned integer
    • Default value: 3966828544
    • Default Value Role: middle Orange in RGB order
  • CreateRootAtOpening
    Role: Create root at opening
    • Type: boolean
    • Default value: true
  • DefaultViewOption
    Role: Define default view when entering Live Editor
    Detailed role: Enables the user to choose the view when entering Live Editor
    • Type: integer
    • Default value: 0
  • FillParentNodeCheckButton
    Role: Fill Parent Node
    Detailed role: Toggles filling of reference frame in process editor
    • Type: boolean
    • Default value: false
  • LayoutDirectionOption
    Role: Layout Expansion direction
    Detailed role: Enables the user to choose the layout expansion direction
    • Type: integer
    • Default value: 1
  • RootProcessType
    Role: Root process type 1: create assemble 2 : create material 3: create kit
    • Type: integer
    • Default value: 1
  • ScaleFactor
    Role: Scale factor for 3D representation above the tile
    • Type: integer
    • Default value: 70
  • ShowAnimationCheckButton
    Role: Turns on/off animated effect
    Detailed role: Turns on/off the display of animations while in the PPR Live Process Planning Editor.\n This includes the highlighting of modified objects, the rotation animation of the process pad, and the view point animation during process verification.
    • Type: boolean
    • Default value: true
  • ShowDataReqCheckButton
    Role: Hides/Shows Data Requirements
    Detailed role: Toggles visibility of Data Requirements in process editor
    • Type: boolean
    • Default value: true
  • ShowFastenerCheckButton
    Role: Hides/Show Fastener
    Detailed role: Toggles visibility of Fastener in process editor
    • Type: boolean
    • Default value: true
  • ShowImplLinkCheckButton
    Role: Hides/Shows Implement Links
    Detailed role: Toggles visibility of Implement Links in process editor
    • Type: boolean
    • Default value: true
  • ShowPrecConstrCheckButton
    Role: Hides/Shows Precedence Constraints
    Detailed role: Toggles visibility of precedence constraints in process editor
    • Type: boolean
    • Default value: true
  • SmartZoomCheckButton
    Role: Activate automatic smart zoom
    Detailed role: Allow the user to choose the activation of the Smart zoom when navigating (zooming or double-clicking) through the process structure.\n Smart zoom is still available by pressing F6 key.
    • Type: boolean
    • Default value: false
  • VisualizationRepsMode
    Role: Select the visualization mode for the display of the Products 3D representations
    Detailed role: Enables the user to choose the visualization mode for the display of the Products 3D representations.\n 3D Shape representations are the representations. 3D Nav representations are better regarding performances.
    • Type: integer
    • Default value: 1
