DNBFittingSettings Setting Repository
Quick Reference

DNBFittingSettings

Role: Settings to handle the parameters in the Assembly Path Definition workbench

Detailed role:This setting allows to customize the appearance and the behavior in Assembly Path Definition

The DNBFittingSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/Assembly Path Definition
  • Repository File: DNBFittingSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DNBFittingSettings setting repository attributes:

  • AutomaticTPointInsertionAngleSpinner
    Role: Define the Orientation Angle of T-Point Insertion.
    Detailed role: Define the Orientation Angle of T-Point Insertion.
    • Type: double
    • Default value: 0.087266463
  • AutomaticTPointInsertionDistanceSpinner
    Role: Define the Distance of T-Point Insertion.
    Detailed role: Define the Distance of T-Point Insertion.
    • Type: double
    • Default value: 0.02
  • AutomaticTPointInsertionWhileMoving
    Role: Automatic TPoint Insertion While mouse moving.
    Detailed role: Automatic TPoint Insertion While mouse moving.
    • Type: integer
    • Default value: 0
  • DisplayProductBuildup
    Role: Display Product Buildup On Selecting Operation
    Detailed role: Displays The Product Buildup On Selecting an Operation
    • Type: integer
    • Default value: 1
  • DisplayProductBuildupForProcess
    Role: Display Product Buildup On Selecting Process
    Detailed role: Displays The Product Buildup On Selecting a Process
    • Type: integer
    • Default value: 1
  • PreviewWindowAvailability
    Role: Show the Preview Window in Track create or edit
    Detailed role: Determines whether the Preview Window is shown in Track create or edit
    • Type: integer
    • Default value: 0
  • ProductPositionInOperationContext
    Role: Display The Product According To Operation Context
    Detailed role: Displays The Product According To Operation Context
    • Type: integer
    • Default value: 1
  • TiltRestriction
    Role: Option to restrict object orientation to an angle w.r.to an axis.
    Detailed role: Option to restrict object orientation to an angle w.r.to an axis.
    • Type: integer
    • Default value: 0

    • Default value: 0.087266463

    Role: Define the axis along which tilt is restricted.

    • Type: integer
    • Default value: 0
