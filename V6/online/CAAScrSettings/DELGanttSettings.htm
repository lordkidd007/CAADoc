DELGanttSettings Setting Repository
Quick Reference

DELGanttSettings

Role: Provides persistent settings for the Gantt Chart

The DELGanttSettings repository has the following characteristics:

  • Primary Tab Page: /Digital Process for Manufacturing/Gantt Chart
  • Repository File: DELGanttSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DELGanttSettings setting repository attributes:

  • AddedValueCheckButton
    Role: Specifies if Value Added has to be displayed
    • Type: boolean
    • Default value: false
  • AddedValueColor
    Role: The color of the added value bar representation
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: By default the added value is yellow
    • Min value: 0
    • Max value: 255

    Role: The names of all Columns in the Gantt Window
    • Type: string
