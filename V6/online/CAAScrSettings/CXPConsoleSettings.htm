CXPConsoleSettings Setting Repository
Quick Reference

CXPConsoleSettings

Role: Setting to handle CXP model parameters

The CXPConsoleSettings repository has the following characteristics:

  • Primary Tab Page: /Interactive 3DEXPERIENCE/Creative Experience/Console
  • Repository File: CXPConsoleSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CXPConsoleSettings setting repository attributes:

  • CallStackTracesColor
    Role: Color of CallStack traces shown in Play Console
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black
    • Min value: 0
    • Max value: 255
  • ClearBeforePlay
    Role: Parameter to set it the console is cleared before each Play
    • Type: boolean
    • Default value: false
  • DebugTracesColor
    Role: Color of Debug traces shown in Play Console
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black
    • Min value: 0
    • Max value: 255
  • ErrorTracesColor
    Role: Color of Error traces shown in Play Console
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black
    • Min value: 0
    • Max value: 255
  • InfoTracesColor
    Role: Color of Info traces shown in Play Console
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black
    • Min value: 0
    • Max value: 255
  • LaunchAutoAtPlay
    Role: Parameter to set if the console is launched automatically at play
    • Type: boolean
    • Default value: false
  • TypeTracesCallStack
    Role: Parameter to set if the traces of type CallStack are retrieved
    • Type: boolean
    • Default value: true
  • TypeTracesDebug
    Role: Parameter to set if the traces of type debug are retrieved
    • Type: boolean
    • Default value: true
  • TypeTracesError
    Role: Parameter to set if the traces of type error are retrieved
    • Type: boolean
    • Default value: true
  • TypeTracesInfo
    Role: Parameter to set if the traces of type info are retrieved
    • Type: boolean
    • Default value: true
  • TypeTracesWarning
    Role: Parameter to set if the traces of type warning are retrieved
    • Type: boolean
    • Default value: true
  • WarningTracesColor
    Role: Color of Warning traces shown in Play Console
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Black
    • Min value: 0
    • Max value: 255
