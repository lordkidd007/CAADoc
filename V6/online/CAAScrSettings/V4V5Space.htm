V4V5Space Setting Repository
Quick Reference

V4V5Space

Role: Setting to handle the V4->V5 migration options

The V4V5Space repository has the following characteristics:

  • Primary Tab Page: /General/Compatibility/V4 V5 SPACE
  • Repository File: GeometrybV4ToV5
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1
  • Setting controller:
    • Framework_CPP: CATIAV4Interfaces
    • Framework_IDL: CATIAV4Interfaces
    • Interface_CPP: CATIV4IV4V5SpaceSettingAtt
    • Interface_IDL: CATIAV4V5SpaceSettingAtt
    • LateType: CATV4IV4V5SpaceSettingCtrl

V4V5Space setting repository attributes:

  • CurvImprovSize
    Role: The Internal Maximum Deformation setting parameter manages the maximum authorized for internal deformation.
    • Type: float
    • Default value: 0.001
  • CurvImprovState
    Role: The Internal Type Deformation setting parameter manages the deformation type.
    • Type: integer
    • Default value: 1
    • Values:
      0 User Defined Value
      1 Model Relative Value
  • DetailsModeExplode
    Role: The DetailsModeExplode setting parameter manages the dittos migration.
    • Type: integer
    • Default value: 0
  • DetailsModeUsual
    Role: The DetailsModeUsual setting parameter manages the dittos migration.
    • Type: integer
    • Default value: 1
  • DetailsModeWireframe
    Role: The DetailsModeWireframe setting parameter manages the dittos migration.
    • Type: integer
    • Default value: 0
  • GapSize
    Role: The External Maximum Deformation setting parameter manages the maximum authorized for external deformation if User Defined Value is chosen.
    • Type: float
    • Default value: 0.001
  • GapState
    Role: The External Type Deformation setting parameter manages the deformation type
    • Type: integer
    • Default value: 1
    • Values:
      0 User Defined Value
      1 Model Relative Value
  • SegmState
    Role: The KeepSegmentation setting parameter manages the segmentation.
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not keep segmentation
      1 Keep segmentation
  • SolidMUState
    Role: The SolidMU setting parameter manages the migration for Solids Mock-Up.
    • Type: string
    • Default value: CGR
    • Values:
      CGR Feature Result As CGR
      PartBody Feature Result As PartBody
  • TextMigrState
    Role: The TextMigration setting parameter manages the 3D text migration.
    • Type: integer
    • Default value: 0
    • Values:
      1 Keep 3D Text
      0 Do not keep 3D Text
