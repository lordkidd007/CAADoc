PPRNavigator Setting Repository
Quick Reference

PPRNavigator

Role: Setting to handle the PPR Navigator parameters

Detailed role:This setting allows to customize the appearance and the behavior of the PPR Navigator User Interface

The PPRNavigator repository has the following characteristics:

  • Primary Tab Page: /ENOVIA V5 PPR/PPR Navigator Properties
  • Repository File: PPRNavigator
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

PPRNavigator setting repository attributes:

  • BackgroundColor
    Role: Background color
    Detailed role: Background color of the PPR Navigator window
    • Type: unsigned integer
    • Default value: 223
    • Default Value Role: Metal grey in RGB order
    • Min value: 0
    • Max value: 255
  • ExploreWithinContext
    Role: Explore within context of existing roots
    Detailed role: Only objects children of the roots already explored will be added in the navigator (for a given PPR type). No other root can be explored if this option is activated and at least one root of the same PPR type is already explored.
    • Type: integer
    • Default value: 0
  • ExploreWithParent
    Role: Explore with parent tree
    Detailed role: Explore with parent tree
    • Type: integer
    • Default value: 0
  • LockAtOpen
    Role: Lock at open
    Detailed role: Lock selected element (instance and reference)
    • Type: unsigned integer
    • Default value: 0
  • NoViewWindowListNextTime
    Role: Display the PPR navigator multiple views manager
    Detailed role: Display the PPR navigator multiple views manager to select the PPR Navigator to complete
    • Type: integer
    • Default value: 0
  • OpenWithChildren
    Role: Open with children
    Detailed role: Open with children
    • Type: unsigned integer
    • Default value: 0
  • OpenWithRep
    Role: Open with default representation
    Detailed role: Open with default representation
    • Type: unsigned integer
    • Default value: 1
  • Process3DNavWithPictures
    Role: Process 3D Navigation showing tiles with pictures
    Detailed role: Process 3D Navigation showing tiles with pictures
    • Type: integer
    • Default value: 0
  • Process3DNavWithProduct3D
    Role: Process 3D Navigation showing Product 3D
    Detailed role: Process 3D Navigation showing materialized or implemented Product 3D
    • Type: integer
    • Default value: 0
  • Process3DNavWithTiles
    Role: Process 3D Navigation showing default tiles
    Detailed role: Process 3D Navigation showing default tiles with process icons
    • Type: integer
    • Default value: 1
  • SystemOp3DNavWithPictures
    Role: System and Operation 3D Navigation showing tiles with pictures
    Detailed role: System and Operation 3D Navigation showing tiles with pictures
    • Type: integer
    • Default value: 0
  • SystemOp3DNavWithTiles
    Role: System and Operation 3D Navigation showing default tiles
    Detailed role: System and Operation 3D Navigation showing default tiles with icons
    • Type: integer
    • Default value: 1
