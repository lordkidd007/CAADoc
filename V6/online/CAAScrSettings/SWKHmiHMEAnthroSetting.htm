SWKHmiHMEAnthroSetting Setting Repository
Quick Reference

SWKHmiHMEAnthroSetting

Role: Setting to handle manikin anthropometry parameters

Detailed role:This setting allows to add a user defined population and the related values

The SWKHmiHMEAnthroSetting repository has the following characteristics:

  • Primary Tab Page: /Ergonomics Design Analysis/Human Measurements Editor/Anthropometry
  • Repository File: SWKHmiHMEAnthroSetting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

SWKHmiHMEAnthroSetting setting repository attributes:

  • AnthroCopyPasteBehavior
    Role: Change population anthropometry with respect to percentile or values
    • Type: integer
    • Default value: 0
    • Default Value Role: By default Percentile
    • Default Value Detailed Role the value can be 0(Percentile) or 1(Values)
  • Populations
    Role: List of user defined populations
    • Type: integer
    • Default value: 1
    • Default Value Role: By default 1
    • Default Value Detailed Role the value can be 1 or 0 (lock / Unlock)
