NCMfgSimulation Setting Repository
Quick Reference

NCMfgSimulation

Role: Lists the settings of the simulation used in the Machining applications

Detailed role:The simulation machining settings are useful to manages options on photo and video replay

The NCMfgSimulation repository has the following characteristics:

  • Primary Tab Page: /Manufacturing/Simulation
  • Repository File: NCMfgSimulation
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

NCMfgSimulation setting repository attributes:

  • BoxType
    Role: Sets the box type
    Detailed role: Sets the box type
    • Type: integer
    • Default value: 0
    • Default Value Role: By default, simulation is done at Program level
    • Values:
      0 Wireframe box
      1 Transparent box
      2 No box

    Detailed role: Sets the collisions management during video simulation

    • Default Value Role: By default, collisions are ignored
