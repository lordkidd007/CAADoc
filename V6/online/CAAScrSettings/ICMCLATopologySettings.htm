ICMCLATopologySettings Setting Repository
Quick Reference

ICMCLATopologySettings

Role: G1 to G3 topology tolerances for ISD commands

Detailed role:The G1 to G3 tolerance are used inside the ISD command to take control of the tolerance when new topological data is created.

The ICMCLATopologySettings repository has the following characteristics:

  • Primary Tab Page: /Shape/ICEM Shape Design/Topology
  • Repository File: ICMGlobalUserSettings
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ICMCLATopologySettings setting repository attributes:

  • G1
    Role: G1 tolerance for created topology
    • Alias: G1
    • Type: double
    • Default value: 0.5
  • G2
    Role: G2 tolerance for created topology
    • Alias: G2
    • Type: double
    • Default value: 5
  • G3
    Role: G3 tolerance for created topology
    • Alias: G3
    • Type: double
    • Default value: 15
