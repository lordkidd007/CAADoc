6WTags
 

6WTags reveal and enrich the semantics of your content. They are structured into six categories (the 6Ws): Who / When / What / Where / Why / How. Tags are either automatically extracted or user-defined.

  • Display and Use Tags
  • How to Create 6WTags

Display and Use Tags

To display tags, click the 6WTags icon next to the search box in the top bar. Tags are displayed in 6W categories and groups. They are ordered by number of occurrences. The number of occurrences appears to the right of each group (corresponds to the number of tags) and to the right of each tag (corresponds to the number with matching content).

Tags can be used to refine search results, content lists or tabs.

There are two types of tag:

  • Automatically extracted tags
    • These tags are automatically extracted from the content, e.g. content owner in the Who category or modification date in the When category.
    • They are structured in groups, in each 6W category.
    • They can be used to refine a search result or any content list.
    • It is not possible to edit these tags.
  • User-defined tags
    • You can define your own tags on any content.
    • These tags appear right under each 6W category.
    • They can also be used to refine a search result or any content list.
    • They can be removed at any time.

How to Create 6WTags

  1. Select a content in a search result, content list or tab.
  2. In the top bar, click the 6WTags icon to display the tags.

  3. Click the '+' next to the appropriate 6W category to display an input box.

  4. In the box, type your tag name and press Enter.
