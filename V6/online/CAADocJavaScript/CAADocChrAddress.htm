Bookmarking this Page

Security rules prevent from automatically bookmarking this page, but you can nevertheless bookmark it and save its navigation context as follows:

  1. Click the icon (Bookmark this page) or type Ctrl+D.
    The Bookmarks dialog box opens. The bookmark for this page is added.

  2. In the Bookmarks prompt click Edit.
    The Edit Bookmark dialog box opens.

  3. Copy the name below in the clipboard (Ctrl+C):

  4. Paste the name copied into the Name box to replace the existing one.

  5. Copy the URL below in the clipboard (Ctrl+C):

  6. Paste the URL copied into the URL box to replace the existing one.

  7. Possibly select a folder, or create a new one to store your bookmark.

  8. Click Save.

This bookmark now opens the bookmarked page in its navigation context.
