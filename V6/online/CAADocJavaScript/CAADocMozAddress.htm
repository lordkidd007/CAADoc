Bookmarking this Page

Security rules prevent from automatically bookmarking this page, but you can nevertheless bookmark it and save its navigation context:

  1. Copy the name below in the clipboard (Ctrl+C):

  2. On the Bookmarks menu, click Bookmark This Page.

  3. Paste the name copied into the Name box.

  4. Click Done.

  5. On the Bookmarks menu, click Organize Bookmarks.

  6. In Bookmarks Menu, locate the newly added bookmark and click it.

  7. Copy the URL below in the clipboard (Ctrl+C):

  8. Paste the URL copied to replace the one in the Location box.

  9. Close the Organize Bookmarks dialog box.

This bookmark now opens this page in its context.
