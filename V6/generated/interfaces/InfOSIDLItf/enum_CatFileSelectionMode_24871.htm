CatFileSelectionMode (Enumeration)

enum CatFileSelectionMode {
  CatFileSelectionModeOpen,
  CatFileSelectionModeSave
}
Mode used when displaying a file selection box using CATIA.
FileSelectionBox.
Values:
CatFileSelectionModeOpen
The selection box is used to get the name of a file to open.
CatFileSelectionModeSave
The selection box is used to get the name of a file to save.
