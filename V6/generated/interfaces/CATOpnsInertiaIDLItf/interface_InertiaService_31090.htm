InertiaService (Object)

Object representing the service to retrieve an inertia element.
See also:

Method Index

GetInertiaElement

Methods


o Func GetInertiaElement( iSelectedItem) As
Retrieves the Inertia object.
Example: 
    This example retrieves an Inertia object.
   
     Set theInertiaService = CATIA.ActiveEditor.GetService("InertiaService")
     Dim theInertia As InertiaElement
     Set theInertia = theInertiaService.GetInertiaElement(theSelection)
