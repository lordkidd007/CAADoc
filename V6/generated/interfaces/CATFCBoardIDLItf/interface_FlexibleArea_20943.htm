FlexibleArea (Object)

Object representing a flexible constraint area.

Property Index

AreaType
Returns or sets the type of a flexible constraint area.
Heightmax
Returns or sets the maximum height of the flexible constraint area.
Heightmin
Returns or sets the minimum height of the flexible constraint area.
Identifier
Returns or sets the identifier of the flexible constraint area.
LAYER
Returns or sets the layer of the flexible constraint area.
OWNER
Returns or sets the owner of the flexible constraint area.

Method Index

GetBoard
Returns the Flexible Board on which the flexible constraint area lies.
GetProfile
Returns the profile of the flexible constraint area.
SetProfile
Sets the profile of the flexible constraint area.

Properties


o Property AreaType() As
Returns or sets the type of a flexible constraint area. The available values are:
  • OTHER_OUTLINE
  • ROUTE_OUTLINE
  • PLACE_OUTLINE
  • ROUTE_KEEPOUT
  • PLACE_KEEPOUT
  • PLACE_REGION
  • VIA_KEEPOUT
o Property Heightmax() As
Returns or sets the maximum height of the flexible constraint area. This property is valid for the following constraints areas:
  • OTHER_OUTLINE
  • PLACE_REGION
o Property Heightmin() As
Returns or sets the minimum height of the flexible constraint area. This property is valid for the following constraints areas:
  • PLACE_KEEPOUT
o Property Identifier() As
Returns or sets the identifier of the flexible constraint area. This property is valid for the following constraints areas:
  • OTHER_OUTLINE
  • PLACE_REGION
o Property LAYER() As
Returns or sets the layer of the flexible constraint area. The available values are:
  • STANDART
  • BOTH
  • INNER
  • ALL

It depends on the type of the constraints area according to the IDF format. This property is valid for the following constraints areas:

  • OTHER_OUTLINE
  • ROUTING_OUTLINE
  • PLACE_OUTLINE
  • ROUTE_KEEPOUT
  • PLACE_KEEPOUT
  • PLACE_REGION
o Property OWNER() As
Returns or sets the owner of the flexible constraint area. The available values are:
  • MCAD
  • ECAD
  • UNOWNED

Methods


o Func GetBoard() As
Returns the Flexible Board on which the flexible constraint area lies.
Returns:
The board reference
o Func GetProfile() As
Returns the profile of the flexible constraint area. The profile can be a Face of the flexible board or a sketch.
Returns:
The profile of the face input of the constraint area
o Sub SetProfile( iProfileorFace,
oCreationView)
Sets the profile of the flexible constraint area. The profile can be a Face of the flexible board or a sketch.
Parameters:
iProfileorFace
The profile of the face input of the constraint area
iCreationView
The view of the creation. The available values are MfDefault3DView or MfUnfoldedView
