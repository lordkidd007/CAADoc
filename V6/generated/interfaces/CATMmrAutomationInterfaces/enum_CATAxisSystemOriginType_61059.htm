CATAxisSystemOriginType (Enumeration)

enum CATAxisSystemOriginType {
  catAxisSystemOriginByPoint,
  catAxisSystemOriginByCoordinates
}
Axis System Origin.

Values:
catAxisSystemOriginByPoint
= 0 Specifies that the origin point is put at the position defined by a geometric point. If no point si selected, the origin will be automatically put at the intersection of the lines or planes defining the axes.
catAxisSystemOriginByCoordinates
= 1 Specifies that the origin point must stay at the position defined by three coordinates x,y,z.
