CatScriptCommand (Enumeration)

enum CatScriptCommand {
  CatScriptCommandDefault,
  CatScriptCommandStop,
  CatScriptCommandStart
}
ScriptCommand values.
Used for property, itself used when performing Selection from script.
Values:
CatScriptCommandDefault
The Select interactive command remains active during all the script execution. This is the default value.
CatScriptCommandStop
Ends the previously activated fake command. The fake exclusive command instanciated goes out of its sleeping state. This is also true for the shared command instanciated during the execution of
. The Select command is instanciated and becomes active.
CatScriptCommandStart
Instanciates and activates a fake exclusive command:
  • the active command (the Select command in most cases) is deactivated
is emptied Then, if the user selects an external exclusive command such as the Pad command, the fake exclusive command instanciated will enter a sleeping state. If the user selects an external exclusive command during the execution of an interactive method such as , the shared command enabling the selection will also enter a sleeping state.
Note: Setting this property will fail (being followed by a script error) if one of the following error occurs (CATIA.ScriptCommand current value is... it cannot be set to...):
  • ... CatScriptCommandDefault => CatScriptCommandDefault
  • ... CatScriptCommandDefault => CatScriptCommandStop

  • ... CatScriptCommandStart => CatScriptCommandDefault
  • ... CatScriptCommandStart => CatScriptCommandStart

  • ... CatScriptCommandStop => CatScriptCommandDefault
  • ... CatScriptCommandStop => CatScriptCommandStop
  • ... CatScriptCommandStop => CatScriptCommandStart
