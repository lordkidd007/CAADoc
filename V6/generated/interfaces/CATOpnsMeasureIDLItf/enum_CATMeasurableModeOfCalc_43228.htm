CATMeasurableModeOfCalc (Enumeration)

enum CATMeasurableModeOfCalc {
  MeasExactCalculation,
  MeasApproximateCalculation,
  MeasExactElseApproxCalculation,
  MeasUnknownCalculation
}
Computation mode
Measure computation can be done in three modes : Exact else Approximate, Exact only or Approximate only.

In Exact else Approximate mode, the computation is done in exact if it's possible else in approximate
In Exact only mode, only exact results are given.
In Approximate only mode, all results are approximate.
Values:
MeasExactThenApproxMode
Specifies the computation of the measure is done in exact if it's possible else in approximate.
MeasExactMode
Specifies the computation of the measure is done in exact.
MeasApproxMode
Specifies the computation of the measure is done in approximate.
