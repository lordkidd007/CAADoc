Measure Edge Type
It is used by the
object to specify the Measure Edge Type.
Values:

Specifies the edge type of Measure item as a Line.
catOpnsArcEdge
Specifies the edge type of Measure item as a Arc.
catOpnsCurveEdge
Specifies the edge type of Measure item as a Curve.

catOpnsParabolaEdge

catOpnsHyperbolaEdge
Specifies the edge type of Measure item as a Hyperbola.
catOpnsAxisEdge
Specifies the edge type of Measure item as a Axis.
catOpnsUnknownEdge
Specifies the edge type of Measure item as a Unknown Type.
