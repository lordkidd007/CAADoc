DELOlpMotionType (Enumeration)

enum DELOlpMotionType {
  delOlpJointMotion,
  delOlpLinearMotion,
  delOlpCircularMotion,
  delOlpCircularViaMotion
}
The motion type used by the motion planner to create the robot trajectory for a robot motion activity.

see
delOlpJointMotion
A joint motion.
delOlpLinearMotion
A linear motion.
delOlpCircularMotion
A circular motion.
A motion with this type must be preceded by a circular via motion. The current location, circular via target, and circular target form the 3 points on the arc.
delOlpCircularViaMotion
A circular via motion.
A motion with this type must be followed by a circular motion. The current location, circular via target, and circular target form the 3 points on the arc.
