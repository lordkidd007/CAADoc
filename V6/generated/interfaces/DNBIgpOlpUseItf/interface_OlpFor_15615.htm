OlpFor (Object)

A for loop instruction.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.

Property Index

CountUp
Indicates if the variable is incremented or decremented each iteration.
EndExpr
The final value of the loop variable.
Instructions
The list of instructions in the loop.
LoopVariable
The counter that is incremented or decremented in the loop.
StartExpr
The initial value of the loop variable.
