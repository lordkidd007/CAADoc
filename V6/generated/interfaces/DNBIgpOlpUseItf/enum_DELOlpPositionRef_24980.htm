DELOlpPositionRef (Enumeration)

enum DELOlpPositionRef {
  delOlpWorld,
  delOlpStation,
  delOlpRailOrigin,
  delOlpRobotBase,
  delOlpMount
}
Reference locations that positions are relative to.

All Cartesian positions are relative to some origin location. These are the options for the origin of an OLP upload or download.

delOlpWorld
The origin is the DELMIA world origin. The world is the origin of the root object in the current window.
delOlpStation
The origin is the station origin. The station is the organizational resource that the robot is in. If the robot is not inside an organizational resource or if the robot reference is open then the station origin defaults to the robot product origin.
delOlpRailOrigin
The origin is the robot base with the rail at zero. If the robot is not on a rail, track, or gantry, then the origin is the same as robot base. This is commonly used for an integrated rail axis.
delOlpRobotBase
The origin is the robot base. The robot base location may be different from the robot's product origin. This location was set when the robot model was created. If the robot is mounted on a rail, track, or gantry, then the robot base is not fixed. As a result there are limitations when using this coordinate frame. See , , and .
delOlpMount
The origin is the robot mount plate. This is the origin for fixed TCP target locations, object frames used in fixed TCP and for mobile tool profiles.
