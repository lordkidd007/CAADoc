OlpTeachHelper (Object)

Native Robot Language(NRL) Teach customization.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) NRL Teach command.
This interface is used to customize the commands available in NRL teach and to retrieve information about the state of NRL teach when one of those commands is executed.

Property Index

CurrentCommand
The command currently being executed.
CurrentInstruction
The instruction selected in Teach.
CurrentProcedure
The task displayed in Teach.
IsTeachActive
Returns TRUE when teach is active.
LogicalKeywords
Set the list logical keywords to be highlighted in NRL teach.
SupportsNRLView
Set whether the translator supports the NRL view feature of NRL teach.

Method Index

AddCustomCommand
Add a new command to Teach.

Properties


o Property CurrentCommand() As (Read Only)
The command currently being executed.
This will only return a valid value during the call to MacroRunCommand. Otherwise, a NULL value is returned.
o Property CurrentInstruction() As (Read Only)
The instruction selected in Teach.
This will return NULL if there is no instruction selected.
o Property CurrentProcedure() As (Read Only)
The task displayed in Teach.
o Property IsTeachActive() As (Read Only)
Returns TRUE when teach is active. For example when the current download is performed for display in NRL Teach. Used in MacroDownload.
o Property LogicalKeywords( iKeywords) (Write Only)
Set the list logical keywords to be highlighted in NRL teach.
Each value must be a string. Normally this is keywords like IF, WHILE, FOR, etc. which are colored in blue. These words are only colored if the is delAstKEYWORD. Used in MacroGetTranslatorInfo.
o Property SupportsNRLView( iSupportsNRLView) (Write Only)
Set whether the translator supports the NRL view feature of NRL teach.
Used in MacroGetTranslatorInfo.

Methods


o Func AddCustomCommand( iGroup,
iCommandName) As
Add a new command to Teach.
Parameters:
iGroup
The name of the group to place the command in.
This is the name displayed for the group and also the identifier for the group. The group is created automatically when a new ID is specified here.
iCommandName
The name of the command to add.
The command's Name and
properties are both initialized to this value.
Returns:
The created command.
