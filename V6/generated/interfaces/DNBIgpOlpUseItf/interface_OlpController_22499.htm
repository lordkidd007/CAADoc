OlpController (Object)

Interface for accessing a device (or controller's) properties.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This object can be retrieved from a .

o Func GetBaseLocation( iOrigin) As
The location of the device's base.
The device base location may be different from the device's product origin. This location was set when the device model was created. If the device is mounted on a rail, the base location is taken at the rail origin.
This will fail if this is not the controller for a robot. Use to determine if this is a robot.
Parameters:
iOrigin
The reference coordinate frame for the base location.
delOlpMount and delOlpRobotBase are not a valid origins.
Returns:
The location of the device's base relative to the specified origin.
