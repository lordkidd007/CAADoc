OlpPulse (Object)

An instruction used to asynchronously change the value of an external output.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.

Property Index

Delay
Get/Set the delay value in seconds.
Destination
The variable which is being assigned.
Duration
Get/Set the duration value in seconds.
Source
The expression that evaluates to the value being assigned.
SourceType
Get/Set the type of Pulse value.
UseDuration
Get/Set whether the duration is used.

o Property Source() As
The expression that evaluates to the value being assigned.
When set the of the AST tree is used for the expression. When retrieved the DELMIA expression is parsed and is returned in the format expected by . You can find more information on the OLP Expression AST format in the documentation under Automation | Robotics | Robotics Offline Programming | Offline Programming Expression Translation.
o Property SourceType() As
Get/Set the type of Pulse value.
This is a simple way to access the most common types of pulsed values. If this property is set, the expression is automatically generated.
o Property UseDuration() As
Get/Set whether the duration is used.
If False, then the duration is effectively infinity and the property should not be retrieved. If True, then the Duration value is used.
