OlpTranslatorHelper (Object)

Service used to get access to OLP data objects.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
This interface provides all the input arguments for the translation process, access to related OLP objects needed for translation, and methods to set the outputs of the translation process.
A VB.NET translator is expected to have these macros defined
MacroUpload
Required. Called to upload robot programs into DELMIA.
MacroDownload
Required. Called to generate robot programs from DELMIA procedures.
MacroGetTranslatorInfo
Optional. Called to get information about the translator and translation options supported.
MacroSetConfigs
Optional. Called to update the config for any created robot motions. This is called after MacroUpload.
The properties and methods below indicate in which of these macro's they are intended to be called.
Example: (VB.NET) 
 Dim Helper As OlpTranslatorHelper = CATIA.Application.GetSessionService("OlpTranslatorHelper")

Properties


o Property AstRoot() As (Read Only)
The root node of the abstract syntax tree.
On upload translators should retrieve this element and append all the parsed files to it. On download translators should append your generated AST tree to this node.
Used in MacroUpload and MacroDownload.
o Property Behavior() As (Read Only)
The resource behavior selected for download or upload.
Used in MacroUpload and MacroDownload.
o Property CAAMotionTypes( iInstructionTypes) (Write Only)
Set the list of CAA motion types.
Any of these motion types will be uploaded as instructions customized through CAA. If the type is not in this list, it will be uploaded as a standard instruction. Each item in the list is a . By default the list is empty. Used in MacroGetTranslatorInfo.
o Property CallSucceeded( iSucceeded) (Write Only)
If the operation succeeded, the translator must set this to True.
Used in MacroUpload, MacroDownload, MacroGetTranslatorInfo, and MacroSetConfigs.
o Property Directory() As (Read Only)
The directory which contains the robot programs on upload.
Used in MacroUpload.
o Property DownloadOption() As (Read Only)
The extra option specified by the user (or in tools-options) for download.
The translator can use this option any way it likes. For example, the XML translator uses this to specify the XSLT-file. It could also be used to specify a robot controller version. The translator can specify valid supported options by setting the .
Used in MacroDownload.
o Property ErrorReporter() As (Read Only)
The object used to report errors, warnings, etc.
Used in MacroUpload, MacroDownload, MacroGetTranslatorInfo, and MacroSetConfigs.
o Property ExtensionDescription( iDesciption) (Write Only)
A description of the file type set with .
For example "Kuka robot programs".
Used in MacroGetTranslatorInfo.
o Property FilePaths() As (Read Only)
A list of full paths to all the files the user requested to upload.
Used in MacroUpload.
o Property MotionGroups() As (Read Only)
The motion groups selected for download or upload.
Used in MacroUpload, MacroSetConfigs and MacroDownload.
o Property ResourceControlDevice() As (Read Only)
The resource control device for download or upload.
Used in MacroUpload and MacroDownload.
o Property SupportedExtensions( iExtensions) (Write Only)
The list of supported file extensions, separated by a semicolon (;).
For example "*.src;*.dat".
Used in MacroGetTranslatorInfo.
o Property SupportedMotionTypes( iInstructionTypes) (Write Only)
Set the list of supported motion types.
Each item in the list is a . By default the list contains delOlpSpotOperation, delOlpArcOperation, delOlpSeamSearchOperation, and delOlpRobotMotion. Used in MacroGetTranslatorInfo.
o Property SupportsMulitpleMotionGroups( iSupportsMMG) (Write Only)
Set whether tasks which control multiple motion groups are supported by this translator.
If not supported, the tasks are exposed to the translators as single motion group tasks. Default values is FALSE. Used in MacroGetTranslatorInfo.
o Property SupportsTemplates( iUseTemplates) (Write Only)
Set whether templates are used by this translator.
Used in MacroGetTranslatorInfo.
o Property Tasks() As (Read Only)
All the tasks the user requested to download or all the tasks that have been created so far on upload.
Primarily used in MacroDownload but also available in MacroUpload and MacroSetConfigs.
o Property TemplateDirectory() As (Read Only)
The directory which contains the template files on download.
Used in MacroDownload.
o Property UploadOption() As (Read Only)
The extra option specified by the user (or in tools-options) for upload.
The translator can use this option any way it likes. For example, the XML translator uses this to specify the JAR-file. It could also be used to specify a robot controller version. The translator can specify valid supported options by setting the .
Used in MacroUpload.
o Property UploadPathOperationType() As (Read Only)
The robot's application type for path operations.
Translators which cannot differenciate between delOlpPathAdhesiveOperation and delOlpSealantOperation can call this function to determine which type to create during upload. Used in MacroUpload.
o Property ValidDownloadOptions( iOptions) (Write Only)
Set the list of valid values for DownloadOption.
Used in MacroGetTranslatorInfo.
o Property ValidUploadOptions( iOptions) (Write Only)
Set the list of valid values for UploadOption.
Used in MacroGetTranslatorInfo.

o Func IsSame( iObj1,
iObj2) As
Tests if 2 objects are the same.
This returns true if the underlying objects are the same. The interface pointers need not be of the same type. You cannot use the VB equality tests on interface pointers. That test may say the objects are not equal even if the underlying objects are the same.
Parameters:
iObj1
The 1st object. Can be NULL.
iObj2
The 2nd object. Can be NULL.
Returns:
True if the iObj1 and iObj2 are interfaces of the same underlying object or if both are NULL.
