OlpAssign (Object)

An instruction used to assigning a value to a variable or IO.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.

Property Index

Destination
The variable which is being assigned.
Source
The expression that evaluates to the value being assigned.
