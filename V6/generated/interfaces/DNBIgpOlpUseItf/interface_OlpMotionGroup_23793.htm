OlpMotionGroup (Object)

A motion group.

This interface can only be used by a translator within the Robotics Off-line Programming (OLP) Download or Upload command.
A motion group is a list of devices that have a single Cartesian or joint target in a robot motion.

Property Index

HasRobot
Indicates if this motion group contains a robot.
Index
Get the motion group index.
IndexSet
Get whether an index is set for the motion group.
PrimaryDevice
The main devices in the motion group.

Method Index

GetDevices
Retrieve devices from the motion group.

Properties


o Property HasRobot() As (Read Only)
Indicates if this motion group contains a robot.
o Property Index() As (Read Only)
Get the motion group index.
In some robot languages an index is used to identify the motion group. If the Index property is not set (IndexIsSet returns FALSE), then getting the Index will fail.
o Property IndexSet() As (Read Only)
Get whether an index is set for the motion group.
If not set, then getting the Index property will fail.
o Property PrimaryDevice() As (Read Only)
The main devices in the motion group.
If there is a robot in the motion group, this is the robot.
