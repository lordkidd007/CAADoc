Framework Module package

DS/TagNavigatorProxy/Tagger6WProxyWithFilteringServices. Tagger6WProxyWithFilteringServices

Tagger6WProxyWithFilteringServices is initialized when the filtering mode is 'WithFilteringServices', it holds specific APIs to set tags data, and it inherits from AbstractTagger6WProxy

  • AbstractTagger6WProxy

addSubjectsTags(subjectsTags) → {Object}

Adds 6WTags into the 6WTagger
Parameters:
Name Type Description
subjectsTags Object Object json that represents the association between subjects and their 6WTags.
Properties
Name Type Description
subject-uri Array.<Object> The URI of the subject. For each subject you have to provide an array of {predicate/object}
Properties
Name Type Argument Description
type String <optional>
 Type of the "object". Can be either "integer","string" or "date".
sixw String Hierarchical path of the predicates URIs, from root 6W to used predicate
object String 6WTag object URI or literal. If summaryData.type is "date", the date must be of the following format : "yyyy/mm/dd".
dispValue String <optional>
 The object name displayed in the 6WTagger. If it is not present, summaryData.object value is used instead. (This attribute is not taken into account if the object type is "date") A tag which "type" attribute is not set is considered as a literal, that is to say a string.
Returns:
The current Tagger6WProxyWithFilteringServices instance
Type
Object
Example
 
     var subjectsTags =  {
         'subject-1-uri':[
             {'object':"Alice",'sixw':"ds6w:who/ds6w:responsible", 'dispValue':"Alice"},
             {'object': "2011/05/30", 'sixw': "ds6w:when/ds6w:start", 'dispValue': "WILL NOT BE DISPLAYED", 'type': "date"},
             {'object':"Post",'sixw':"ds6w:what/ds6w:type", 'dispValue':"Post"}
         ],
         'subject-2-uri':[
             {'object':"Thomas",'sixw':"ds6w:who/ds6w:responsible", 'dispValue':"Thomas"},
             {'object':"Post",'sixw':"ds6w:what/ds6w:type", 'dispValue':"Post"}
         ]
     };
     this.taggerProxy.addSubjectsTags(subjectsTags);
