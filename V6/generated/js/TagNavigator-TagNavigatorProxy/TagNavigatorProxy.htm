Methods

<static> createProxy(options) → {object}

Creates and returns a new 6WTaggerProxy to communicate with the 6WTagger
Parameters:
Name Type Description
options object
Properties
Name Type Argument Description
widgetId String Identifier of the widget
filteringMode string Value is 'WithFilteringServices' for a client side filtering, or 'FilteringOnServer' for a server side filtering
tenant string <optional>
 Identifier of the tenant from which subjects come from. Use only if the subjects come from the 3DEXPERIENCE PLATFORM
Returns:
A Tagger6WProxy class instance for a server side filtering, and Tagger6WProxyWithFilteringServices class instance for a client side filtering
Type
object
Example
 
     var options= {
         widgetId : widget.id,
         filteringMode : "FilteringOnServer",
         tenant: 'TENANT_ID'
     };
     var taggerProxy = TagNavigatorProxy.createProxy(options);
