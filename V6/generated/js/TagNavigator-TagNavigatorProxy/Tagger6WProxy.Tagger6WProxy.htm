setTags(tagsdata, summaryData) → {Object}

Sets 6WTags (without the subjects part) to the 6WTagger
Parameters:
Name Type Argument Description
tagsdata Object Useless argument for filtering, set null as value
summaryData Array.<Object> <optional>
 Summary of existing 6WTags
Properties
Name Type Argument Description
type String Type of the "object". Can be either "integer","string" or "date".
object String 6WTag object URI or literal. If summaryData.type is "date", the date must be of the following format : "yyyy/mm/dd".
sixw String Hierarchical path of the predicates URIs, from root 6W to used predicate
dispValue String <optional>
 The object name displayed in the 6WTagger. If it is not present, summaryData.object value is used instead. (This attribute is not taken into account if the object type is "date") (This attribute is not taken into account if the tag's type is "date")
count Integer Number of subjects which hold the 6WTag
Returns:
The current Tagger6WProxy instance
Type
Object
Example
 
 var summaryData = [
     {'object':"Thomas",'sixw':"ds6w:who/ds6w:responsible", 'dispValue':"Thomas", 'type':"string", 'count':1},
     {'object':"2014/11/01",'sixw':"ds6w:when/ds6w:starts", 'type':"date", 'count':1}
     {'object':"Post",'sixw':"ds6w:what/ds6w:type", 'dispValue':"Post", 'type':"string", 'count':2}
 ];
 taggerProxyFilteringOnServerSide.setTags(null ,summaryData);
