activate()

Activates the proxy. It implies to show up the 6WTags of the proxy in the 6WTagger
Example
 this.taggerProxy.activate();

addEvent(eventName, eventCallback, scope)

Sets callback functions, used for setting callbacks for filtering events
Parameters:
Name Type Argument Description
eventName String Name of the event you want to set a callback on. It can be 'onFilterChange' if the proxy filteringMode is "FilteringOnServer", or 'onFilterSubjectsChange' if filteringMode is "WithFilteringServices"
eventCallback function Function called when the event is triggered. 'onFilterChange' and 'onFilterSubjectsChange' callbacks are called with a filter as an argument.
scope scope <optional>
 Scope of the callback
Returns:
null
Example
 
     this.taggerProxy.addEvent('onFilterChange', eventCallback, this);
     this.eventCallback = function (filter) {
         //filter example "OnFilterChange": 
         //{
         //    "allfilters": {
         //        "ds6w:when/ds6w:starts": [
         //            {
         //                "disptext": "2013",
         //                "object": "2013",
         //                "type": "date"
         //            }
         //        ]
         //    }
         //}
         //filter example "OnFilterSubjectsChange": 
         //{
         //   "filteredSubjectList": [
         //        'subject-1-uri',
         //        'subject-2-uri'
         //    ]
         //}
     };

deactivate()

Deactivates the proxy. It implies to hide the 6WTags of the proxy in the 6WTagger
Example
 this.taggerProxy.deactivate();

die()

Deletes the 6WTaggerProxy. First, it removes the 6WTags from the 6WTagger, and then deletes the connection from the 6WTagger. This proxy does not exist anymore for the 6WTagger. Usually this method is called when the widget is destroyed. You cannot use the proxy once the die method is called, all methods will fire an exception.
Returns:
null
Example
 
    this.taggerProxy.die();

getCurrentFilter() → {Object.<String, Array.<Object>>}

Gets the filter of the 6WTagger. The filter is the result of the end-user choice
Returns:
The filter as a json structure
Type
Object.<String, Array.<Object>>
Example
 
    var filter = taggerProxy.getCurrentFilter();
    //example of filter received
    filter = {
        "allfilters": {
              "ds6w:when/ds6w:starts": [
                  {
                      "disptext": "2013",
                      "object": "2013",
                      "type": "date"
                  }
              ]
          }
      }

unsetTags()
