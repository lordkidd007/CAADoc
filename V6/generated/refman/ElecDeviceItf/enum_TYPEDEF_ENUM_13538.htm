ElecDeviceItf Enumeration TYPEDEF_ENUM

enum TYPEDEF_ENUM {
  ALL,
  Cavity,
  CavityCnctPt,
  ConnectorCnctPt,
  BackshellCnctPt,
  ShellCnctPt,
  SegmentCnctPt,
  SegmentPositionPt
}
Type of connection point.
Values:
ALL
All types of connection points.
Cavity
a Cavity.
CavityCnctPt
A cavity connection point.
ConnectorCnctPt
a Connector connection point.
BackshellCnctPt
A Backshell connection point.
ShellCnctPt
a Shell connection point.
SegmentCnctPt
A Segment connection point.
SegmentPositionPt
a Segment Position Point.
