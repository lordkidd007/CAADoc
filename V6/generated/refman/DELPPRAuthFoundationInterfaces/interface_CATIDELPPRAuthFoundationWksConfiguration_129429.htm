interface CATIDELPPRAuthFoundationWksConfiguration

CATIDELPPRAuthFoundationWksConfiguration is implemented by a CAA Workbench.
This allows the wokbench to be used in the context of the DELPPRAuthFoundationWks workshop. Role: This interface has to be implemented on the CAA workbench in order to be used in the workshop DELPPRAuthFoundationWks.

This object is included in the file: CATIDELPPRAuthFoundationWksConfiguration.h
If needed, your Imakefile.mk should include the module: DELPPRAuthFoundationItf
