CATAnnotationUseItf Enumeration CATDrwMirroringDefinition

enum CATDrwMirroringDefinition {
  CATDrwNoFlip,
  CATDrwFlipHorizontal,
  CATDrwFlipVertical,
  CATDrwFlipHorizontalAndVertical,
  CATDrwFlipAutomatic
}
Mirroring of a text block.
Values:
CATDrwNoFlip
No mirroring : the text is not flipped horizontally nor vertically.
CATDrwFlipHorizontal
The text is flipped horizontally.
CATDrwFlipVertical
The text is flipped vertically.
CATDrwFlipHorizontalAndVertical
The text is flipped both horizontally and vertically.
CATDrwFlipAutomatic
The text is automatically flipped.
