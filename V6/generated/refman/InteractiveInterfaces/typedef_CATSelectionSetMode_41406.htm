InteractiveInterfaces Typedef CATSelectionSetMode

typedef unsigned long CATSelectionSetMode
The Selection Set mode.

Role: The Selection Set mode is used in the , , methods to specify if the path of the Selection Set is a relative path in the PLM Representation Reference or a path that contains all the PLM component.
Values:
CATSelSetModeCurrent
To retrieve current PLM Representation Reference Selection Sets.
CATSelSetModeRecursive
To retrieve the Selection Set of all PLM component contained in the input path.
This object is included in the file: CATISelectionSetsFactory.h
