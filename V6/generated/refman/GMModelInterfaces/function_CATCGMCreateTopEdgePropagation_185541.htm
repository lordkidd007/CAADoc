GMModelInterfaces Global Function CATCGMCreateTopEdgePropagation


CATCGMCreateTopEdgePropagation( iFactory,
iTopData,
iBody,
iInitialEdge)
Creates a CATICGMTopEdgePropagation operator.
Parameters:
iFactory
The factory which creates the resulting body.
iTopData
The pointer to the data defining the configuration and the journal. If the journal inside iTopData is NULL, it is not filled.
iBody
The body which contains the initial edge and where the propagation is carried out. This body can be a 1D, 2D or 3D body.
If the body is made of several domains, the propagation is performed only within a single domain ( the domain of the initial edge ).
iInitialEdge
The first edge to be concatenated.
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
This object is included in the file: CATICGMTopEdgePropagation.h
