PrintBase CATPrintImage

Usage: you can derive this class.

public class CATPrintImage

Abstract class to build a print image.

Role: This class has to be derived to build an image. The and methods must be overridden.
The method can be overriden. By default, it initializes the device, calls method and ends the device.
