DELPPRProcessItf Typedef ErrorDiagnostic

	typedef unsigned long ErrorDiagnostic
Definition the available diagnostic values.
Values:
NoError
No Error has been found
UndefinedError
An error occured
NullInput
The provided object is null
NotAGroupOrInstance
The provided object is nor a Group or an Ref. Instance
InvalidObject
The object can't be added to the group
NotDirectChild
The provided object is not a direct child of the group owner
LockedGroup
The object can't be added to the group because the group is locked
AlreadyInGroup
The object can't be added to the group because it already belongs to a group
FatherInGroup
The object is implicitely added to the group (one father already belong to the group)
NotRemovable
The object cannot be removed from the group (one father already belong to the group)
This object is included in the file: DELIPPRProcessGroup.h
