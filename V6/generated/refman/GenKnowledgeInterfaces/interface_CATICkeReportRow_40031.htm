Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATICkeReportRow

Interface dedicated to exposing the Row Properties of the Report data.

Role : This interface object represents each row in the generated report and contains information regarding object/s associated with each row, values of attributes corrosponding to the object associated with the row(in case of grouped rows these values can be ****). User can customize the value of each attribute value using its CATICkeParm value.
