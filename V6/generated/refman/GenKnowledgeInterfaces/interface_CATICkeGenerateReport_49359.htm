Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATICkeGenerateReport

Interface dedicated to generating report from given user data.

Role : Generates report on the user data using report template User can generate reports in different file formats and save it in database. With information generated by this interface, user can customize the file format.

public virtual GenerateReport( const iRootObjects,
const iObjects,
const iReportTemplate,
oRowObject,
oColumnList,
oChildRows,

Generate report using report template and input objects.
Parameters:
iRootObjects
List of root objects.All objects aggregated under these objects are retrieved and appropriate objects specified in report template as type are selected.
iObjects
List of objects whose report directly needs to be generated. No type checking with respect to report template is done. It is a resopsibility of CAA user to provide correct list of objects. No errors will be generated in case correct list is not provided.
iReportTemplate
Path of the specified report template.
oRowObject
List of row objects(CATICkeReportRow)
oColumnList
List of column objects(CATICkeReportColumn) of the report.
oChildRows
List of row objects(CATICkeReportRow) for the list attributes.
oChildColumns
List of column objects(CATICkeReportColumn) of the report for the list attributes.
Returns:
return S_OK or E_FAIL.
WARNING: All existing Row and Column Objects will be removed and fresh Rows and Columns will be appended in the list.

o SaveFileinDatabase
public virtual SaveFileinDatabase( const iAttributeNameList,
const iAttributeValueList,
const iFilePath,
const iExt)
Save the created report in the database.
Parameters:
iAttributeNameList
The list of attributes' internal name. There is an error if this name is not consistent with the entity (no attribute with this name).
iAttributeValueList
The list of attributes' value. There is an error if this value is not consistent with the attribute Name List (list needs to correlate to the iAttributeNameList).
iFilePath
Complete File path for the given file. Legal values xml, html, txt and xls.
iExt
Extension of the file being saved.
Returns:
return S_OK or E_FAIL.
