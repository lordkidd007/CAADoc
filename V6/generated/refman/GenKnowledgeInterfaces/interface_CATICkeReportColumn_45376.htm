GenKnowledgeInterfaces Interface CATICkeReportColumn

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATICkeReportColumn

Interface dedicated to exposing the Column Properties of the Report data.

Role : This interface object represents each column in the generated report and contains information regarding properties of the column.
