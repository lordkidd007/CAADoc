AfrFoundation Global Function CATAfrStartCommand


CATAfrStartCommand( iCommandHeaderId,
oCommand)
Starts the command associated with a command header.
Role: This function starts the associated with a , and returns a pointer to this command.
Parameters:
iCommandHeaderId
The identifier of the CATCommandHeader.
oCommand
The CATCommand.
Returns:
Returns S_OK on success. Returns E_FAIL if the identifier is not an existing CATCommandHeader identifier.
