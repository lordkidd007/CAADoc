PLMLibraryInterfaces Interface PLMICLGCatalogFilter

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface PLMICLGCatalogFilter

Interface representing catalog filter.
Role: The Catalog Filter allows to create a check wich is used to filter the displayed object in the catalog browser. the expression of a filter working with two keywords key1 and key2 can be: "key1 <= 12mm AND key2 >= 15 AND key3 like \"value\""
