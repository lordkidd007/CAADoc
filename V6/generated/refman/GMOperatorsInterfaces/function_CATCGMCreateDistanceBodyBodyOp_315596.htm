GMOperatorsInterfaces Global Function CATCGMCreateDistanceBodyBodyOp


CATCGMCreateDistanceBodyBodyOp( iFactory,
iData,
iBodiesList1,
iBodiesList2)
Creates an operator to compute the distance between two groups of bodies. All the bodies must belong to the same factory.
Parameters:
iFactory
The pointer to the factory of the geometry.
iData
The pointer to the CATTopData defining the software configuration and the journal.
iBodiesList1
The list of the bodies forming the first group.
iBodiesList2
The list of the bodies forming the second group.
Returns:
The pointer to the created operator.
This object is included in the file: CATICGMDistanceBodyBody.h
If needed, your Imakefile.mk should include the module: CATGMOperatorsInterfaces
