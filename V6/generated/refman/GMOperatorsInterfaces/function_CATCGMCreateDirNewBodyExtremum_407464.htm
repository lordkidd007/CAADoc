GMOperatorsInterfaces Global Function CATCGMCreateDirNewBodyExtremum


CATCGMCreateDirNewBodyExtremum( iFactory,
iTopData,
iBody,
iTypeOfExtremum,
const iDirection,
const iPtOrig)
Creates the operator that computes the point(s) the furthest away from a given point in a given direction.
Parameters:
iFactory
The pointer to the geometry factory.
iBody
The pointer to the body.
iTypeOfExtremum
The type of extremum: minimum or maximum.
iDirection
The direction.
iPtOrig
The origin from which the distance returned by GetExtremum is computed. By default, the CATMathPoint is (0,0,0).
iJournal
The pointer to the journal. If NULL, the journal is not filled in.
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
This object is included in the file: CATICGMTopBodyExtremum.h
If needed, your Imakefile.mk should include the module: CATGMOperatorsInterfaces
