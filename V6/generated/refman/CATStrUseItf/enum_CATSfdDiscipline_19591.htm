CATStrUseItf Enumeration CATSfdDiscipline

enum CATSfdDiscipline {
  SfdSetDisciplineOK,
  SfdDisciplineAlreadyStructure,
  SfdWorkbenchNotStructure,
  SfdUnableToFindVPMReference,
  SfdDisciplineAlreadySomethingElse,
  SfdAttemptToSetDisciplineFailed
}
Define the plate's limiting type
Values:
SfdSetDisciplineOK
V_discipline successfully set to Structure.
SfdDisciplineAlreadyStructure
V_discipline is already set to Structure.
SfdWorkbenchNotStructure
The workbench is not SFD.
SfdUnableToFindVPMReference
Unable to find the VPMReference.
SfdDisciplineAlreadySomethingElse
V_discipline already set to something else other than Structure.
SfdAttemptToSetDisciplineFailed
Attempt to set V_discipline is failed.
