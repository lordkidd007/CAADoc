VisualizationFoundation CAT3DCurveRep

Usage: you must use this class as is. You should never derive it.

public class CAT3DCurveRep

Class to create the representation of a curve.
Role: This representation is a collection of CAT3DLineGP, each one representing a Level of Detail (LOD). LOD should be added in the order of increasing tessellation accuracy factor. Example :

  • lod 0 => sag = 0.10
  • lod 1 => sag = 0.25
  • lod 3 => sag = 0.50
