VisualizationFoundation CATEndActivate

Usage: you must use this class as is. You should never derive it.

public class CATEndActivate

Class to notify the end of selection of the representation.
Role: This notification sent by a indicates that the mouse botton 1 is pressed above another representation not controlled by the manipulator or above the background. It can be received by any class by means of the callback through the method.
