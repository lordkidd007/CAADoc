VisualizationFoundation CATEndPreactivate

Usage: you must use this class as is. You should never derive it.

public class CATEndPreactivate

Class to notify the end of preselection of the representation.
Role: This notification sent by a indicates that the mouse leaves the representation controlled by the manipulator. It can be received by any class by means of the callback through the method.
