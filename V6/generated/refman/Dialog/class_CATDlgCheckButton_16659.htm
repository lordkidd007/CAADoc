Dialog CATDlgCheckButton

Usage: you must use this class as is. You should never derive it.

public class CATDlgCheckButton

Base class for check buttons.
Role: The check buttons allow to select several options among all the proposed options, that is several can be checked while the others are unchecked.
Use check buttons whenever you propose to the end user a choice among several inclusive options, that is when several options can be selected at the same time.
