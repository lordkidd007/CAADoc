public class CATFstValidProcessor

The validation processor.

Role: CATFstValidProcessor represents the general class of all the validation processors. For each input filter type, we provide the appropriate class (ex. CATFstValidProcessorText for text input). In order to customize the validation object, one needs to derive from the appropriate CATFstValidProcessor and to implement the validation behavior.
