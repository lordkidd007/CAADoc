Usage: you can derive this class.

public class CATFstFilterProcessorText

The filter processor for a text field input.

Role: CATFstFilterProcessorText represents the general class of all the text filter processors. In order to customize the filtering of a text field, one needs to derive from the CATFstFilterProcessorText and to implement the fitler behavior.
