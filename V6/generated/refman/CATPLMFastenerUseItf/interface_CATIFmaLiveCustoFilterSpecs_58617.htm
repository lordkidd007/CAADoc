interface CATIFmaLiveCustoFilterSpecs

Interface creating the data for the filters.

Role: If the customer wants to add his own text fields (which defines a custo-filter), he must declare them first. To do this, this API must be implemented. It must be implemented on component CAAFmaCustoFilterSpecs
