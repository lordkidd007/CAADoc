CATCompositesUseItf Interface CATICciMaterialCache

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATICciMaterialCache

Interface to CacheMaterial.
Role: Allows to access data of Composites material cache feature, agregated by a Composites parameter object and that exposed a link to a material feature.
