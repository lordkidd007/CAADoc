interface CATIPsiRepresentationReference

Interface dedicated to the Representation Reference management.

The Representation Reference can contain one main stream descriptor which refers to any type of document (3D Shape, V4 model, Drawing, Text file, ...).
A Representation Reference can be instanciated as a Representation Instance which is in turn aggregated under a Product Reference.
This interface is implemented on the Representation Reference.
