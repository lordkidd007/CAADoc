interface CATISktUse2DDatumCurve

Interface representing a Sketcher datum curve.

A CATISktUse2DDatumCurve is a curve which geometry is a CATPSpline, a CATPNurbs or a CATPCurveEquation.
A CATISktUse2DDatumCurve is created by the CreateUnspec method of the CATISktUse2DWFFactory interface.
