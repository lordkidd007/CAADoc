public class DELLiveInfraRefreshNotif

Notification used to refresh immersive browsers content in Live Simulation.
Role: This notification is used to inform browser agents that the content of the immersive browser tabs needs to be refreshed (after an undo/redo event or an user-requested browser refresh).
