interface CATIDftGen3DSpaceClipping3DGeometry

Interface for 3D Space Clipping purpose.
Role: This interface is dedicated to 3D Space Clipping integration in generative Drafting. For compatibility purposes, your implementation must inherit from CATDftGen3DSpaceClipping3DGeometryAdapt.
