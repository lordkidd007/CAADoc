interface CATICatalogEnable

Interface to enable component integration into a catalog document.
Role:In implementing this interface on a component, instances of this component can be inserted into a catalog. It is only useful for the feature component, and not for the document for example.
To integrate a new type of component in a catalog document there are two others interfaces to considere:

  • CATICatalogInstantiation : Interface to provide the interactive command to instantiate in context the component
  • CATICatalogSynchronize : Interface to provide information to fill the description ( CATICatalogDescription ) which contains a link towards a component instance.

This object is included in the file: CATICatalogEnable.h
If needed, your Imakefile.mk should include the module: CATCclInterfaces
