interface CATICatalogBrowser

Interface to manipulate the Catalog Browser.
Role: The Catalog Browser is a Dialog box to browse and instantiate component in context. The interface enables to create and display an instance of this Dialog box.

This catalog browser will be created inside a . After the creation, you must retrieve the window of the Dialob box, thanks to the , and sets as parent for this window the command which has instantiate it. It enables to your command to receive the notifications sent by the Catalog Browser:

and in case of Dialog box closing use the to close it.

On the catalog browser you can impose to the end user a type of behavior:


    • To authorize or not instantiation by contextual menu or double click on descriptions

    • To authorize or not instantiation by drag and drop.

    • To authorize or not catalog change

    • Propose a default Catalog path at the beginning

The last part concerning the catalog browser is the filtering. You can retrieve the end user filter or impose it a filtering. There are two kinds of filter:

  • The internal filter: you impose it to the end user on a chapter, the end user cannot change it.
  • The end user filter: those enter by the end user in the Filter editor, or by the method you can set a end user filter
The filter applied to a chapter is the internal filter and the end user filter.

o AuthorizeChangeCatalog
public virtual AuthorizeChangeCatalog( iEnable)
Authorizes or not user to change of current catalog.
Parameters:
iEnable
0: OK available only for description in end chapters. File Selection Box Icon is Disable.
1: OK available for all description, default mode. File Selection Box Icon is Enable.
o AuthorizeDragDrop
public virtual AuthorizeDragDrop( iEnable)
Authorizes the instantiation with drag&drop.
Parameters:
iEnable
If iEnable is 1 then the functionality drag&drop is enable. Else 0 and the functionality is disable.
Attention, the Drag and Drop is only possible if the current command is the Select command. So if the catalog browser is launched from a command which is exclusive, or shared, the instantiation by drag and drop is unavailable.
