public class CATDlgNotify

Base class for information modal dialog windows.

note: Notify windows are always modal under Windows Operating Systems.

note: These windows are implemented as standard windows provided by the Operating Systems. As so, the button texts ("Cancel", "Apply",...) follow the installation language of the Operatings System and not the language selected for V5.
