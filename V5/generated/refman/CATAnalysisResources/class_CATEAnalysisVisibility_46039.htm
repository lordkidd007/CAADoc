CATAnalysisResources CATEAnalysisVisibility

Usage: you can derive this class.

public class CATEAnalysisVisibility

Adaptor for CATISamVisibility interface.
This Adaptor can be used in BOA DataExtension.

Basically, for implementing this interface, you just need to implement the constructor by initializing the internal data .
