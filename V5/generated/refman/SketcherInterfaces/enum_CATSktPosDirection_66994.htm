SketcherInterfaces Enumeration CATSktPosDirection

enum CATSktPosDirection {
  CATSktDirUndefined,
  CATSktDirHorizontal,
  CATSktDirVertical
}
Direction definition of the sketch axis which is oriented.
Role: The orientation direction lets you specify whether the orientation references will be applied on axis H or V. You can set the direction definition thanks to the CATISketchPositioning.SetOrientationDirection method.

Values:
CATSktDirHorizontal
The oriented axis is H axis.
CATSktDirVertical
The oriented axis is V axis.
CATSktDirUndefined
Do not used.
