TopologicalOperators Enumeration CATDynSewingOption

enum CATDynSewingOption {
  CATDynDefaultSide,
  CATDynReverseSide
}
Defines how the ribbon is seewed on the input skins.

In case of CATDynTrim mode.
Values:
CATDynDefaultSide
The seewed edges are smooth.
CATDynReverseSide
The seewed egdes are sharp.
