TopologicalOperators Enumeration CATDynSegmentationMode

enum CATDynSegmentationMode {
  CATDynTrim,
  CATDynNoTrimWithTraceOnSupport,
  CATDynNoTrim
}
Defines the construction mode of the resulting body.
Values:
CATDynTrim
The GetResult method returns a body inside which the input and ribbon are connected.
CATDynNoTrim
The GetResult method only returns a body containing the ribbon.
This object is included in the file: CATDynFilletRibbon.h
