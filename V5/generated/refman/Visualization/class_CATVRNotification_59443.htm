public class CATVRNotification

This object is the basic event received from specific devices.
Role : After adding a callback method on a given device and events, the CATNotification received on the callback method is of type CATVRNotification. The CATVRNotification behaves as an iterator on the newly arrived event set. This set only contains the events you did not already received in the callback. You access these events through the iterator. It also acts as a filter so that you can only access the events whose types were specifified in the callback declaration.
