Visualization Enumeration CATGenerativeAttribute

enum CATGenerativeAttribute {
  csoOperation,
  pathManipulatorOperation,
  drillOperation
}
Attribute allowing to generate a path.
Role:These attributes are used in the CATVisManager.GenerateRepPathFromPathElement
Values:
csoOperation
Used for highlight and prehighlight operations. When csoOperation is used, the method
CATVisManager.BuildHighLightLook is called. It can be overloaded to highlight a specific rep defined by the user
pathManipulatorOperation
Used for manipulation operations. When pathManipulatorOperation is used, the method
CATVisManager.BuildRepPath is called
This object is included in the file: CATGenerativeAttribute.h
