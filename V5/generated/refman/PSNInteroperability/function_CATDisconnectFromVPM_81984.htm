PSNInteroperability Global Function CATDisconnectFromVPM


HRESULT CATDisconnectFromVPM()
Disconnects the application from ENOVIAvpm V4.
Role:This method first disconnects the user from the ENOVIAvpm V4 database then closes the connection with ENOVIAvpm V4. If an ENOVIAvpm V4 server had been launched, it is stopped.
See also:
CATConnectToVPM.

If needed, your Imakefile.mk should include the module: CATPinServices

Returns:
  • S_OK: The application is successfuly disconnected
  • E_FAIL: The method has failed. The possible returned errors (listed by their identifier) are:
    • ERR_1002: ENOVIAvpm server cannot be detected
This object is included in the file: CATBatchServices.h
