PSNInteroperability Global Function CATTerminateBatch


HRESULT CATTerminateBatch()
Terminates the VPM session.
Role:This method deletes the CATIA V5 session opened using CATInitBatch or CATInitBatchOnHost.
This method must be called before exiting the application.

If needed, your Imakefile.mk should include the module: CATPinServices

Returns:
  • S_OK: The CATIA session is deleted
This object is included in the file: CATBatchServices.h
