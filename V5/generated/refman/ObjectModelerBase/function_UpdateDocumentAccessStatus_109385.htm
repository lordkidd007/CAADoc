ObjectModelerBase Global Function UpdateDocumentAccessStatus


UpdateDocumentAccessStatus( iDocument,
oReadOnly)
Updates the document access status.
Role: Refreshes the current R/W access status of a document opened in session in File doc environment. The document R/W state in session is updated from the current OS access status of the file on disk, if necessary. example : - E:\dir\subdir\Sample.CATPart is set as ReadOnly on OS, and opened in session: the document is seen as ReadOnly in session. - Change the R/W status to ReadWrite on OS: When called, this method allows the document to be seen as ReadWrite in session.
Parameters:
iDocument
[in] The session Document on which the R/W status is to be updated.
oReadOnly
[out] The current access status, as updated on document. (TRUE:ReadOnly / FALSE:ReadWrite)
Returns:
S_OK Access status has been updated successfully S_FALSE Access status cannot be refreshed (wrong doc environment, no file on disk), previous Status is kept and returned. E_* unexpected error
