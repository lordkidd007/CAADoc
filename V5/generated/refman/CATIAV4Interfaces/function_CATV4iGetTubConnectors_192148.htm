CATIAV4Interfaces Global Function CATV4iGetTubConnectors


CATV4iGetTubConnectors( const iV4Element,
iConnectorType,
oConnectorList)
Reads the connectors of a tube
Parameters:
iV4Element
[in] The V4 element
iConnectorType
[in] The desired connector type
Legal values:
  • 0 : all connectors
    • 1 : extremity connectors
    • 2 : branch connectors
oConnectorList
[out] The list of connectors
Returns:
Return code
Legal values:
  • 0 : the function was OK
  • -1 : the function was KO
