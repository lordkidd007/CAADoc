CATIAV4Interfaces Global Function CATV4iGmasln


int CATV4iGmasln( const CATV4iV4Element* iElement,
const CATV4iV4Element* iContainer,
const CATV4iV4Element* iPrevElement,
CATV4iV4Element*& oNextElement,
int& oEnd,
int& ier)
Scans the application elements to which an element is linked.
Parameters:
iElement
[in] The V4 element
iContainer
[in] The container to scan
Legal values: It could be an application set, a workspace or the current workspace
iPrevElement
[in] The previous application element pointing iElement
Legal values: NULL to start the scan
oNextElement
[out] The next application element to which iElement is linked
oEnd
[out] The end of the scan.
Legal values: value is 1 if iPrevElement is the last element
ier
[out] error code
