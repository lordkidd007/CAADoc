public class CATModelIdentificator

Class to create a model identifier.
Role: This class is dedicated to be the only link between the graphical representations and their corresponding objects. Refer to CATIVisu to understand how one object can be visualized. This identifier is designed to make graphical representations (that is the "visualization world") independent from their object (that is the "applicative world") and make the link consistent.

The identifier can be seen either as an integer, or as a pointer to one object, or as a generic pointer .

Such an identifier is set on each graphical representation built from the CATIVisu.BuildRep() method . The enclosed identifier refers to the object that implements a specialized visualization interface so as to find out the "model" associated with the graphical representation.
