public class CATMathSetOfVectors

Class representing a set of CATMathVectors.

The size is defined by a number of CATMathVectors NbVectors. The index of the first CATMathVector is 0.
The SetNumberOfVectors method enlarges the memory if needed, while the Compress inherited method releases the free unused memory.
