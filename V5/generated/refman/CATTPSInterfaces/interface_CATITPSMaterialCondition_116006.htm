interface CATITPSMaterialCondition

Interface for accessing Material Condition modifier on a TPS.
Note that second argument is a default one ; it has only to be valuated, when handled object is a CATITPSDatumSystem instance in order to define the Simple Datum in the Reference Frame concerned by the Material Condition.
