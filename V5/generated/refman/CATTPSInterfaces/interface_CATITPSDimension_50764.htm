interface CATITPSDimension

Interface to dimension tolerance.
Role: Objects that implement this typing interface are dimension tolerances. It is a super type.

The dimension tolerance can be: linear, angular, second linear, chamfer, framed (ISO) or basic (ASME). It defines the range in which the dimension value will be located.

Normative reference: ISO 406, ISO 8015, ISO 1101 and ASME Y14.5M section 1.8 and 1.3.9.
