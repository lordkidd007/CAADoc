interface CATITPS3DGeoVisu

Interface that drives specific visualization behaviour of feature within a tolerancing context.
For instance, this interface is the only way to build a visual representation for TTRS features that do not implement the general visualization behaviour (CATI3DGeoVisu)

This object is included in the file: CATITPS3DGeoVisu.h
If needed, your Imakefile.mk should include the module: CATTPSItf
