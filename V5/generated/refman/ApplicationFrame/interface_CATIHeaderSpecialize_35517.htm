ApplicationFrame Interface CATIHeaderSpecialize

Usage: you can freely reimplement this interface.

interface CATIHeaderSpecialize

Interface for managing callbacks associated to command headers.
Role:Command headers can implement this interface to set callbacks when then command headers get activated, and override the command header availability status.

See also:

Methods


o CleanCallback
public virtual CleanCallback()
Removes callbacks. Remove all callbacks when the command header is deactivated.
o GetStatus
public virtual GetStatus()
Command header availability. Tells whether the command header is available or not.
Returns:
  • 1: The command header is available.
  • 0: The command header is unavailable.
o StartCallback
public virtual StartCallback()
Sets callbacks. Set callback(s) in order to intercept events as soon as the command header is activated.
