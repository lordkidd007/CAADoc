ApplicationFrame Global Function CATApplyCameraToViewer


HRESULT CATApplyCameraToViewer(CATViewer* ioViewer,
CATICamera* ipICamera)
Applies a camera to a viewer.
Parameters:
ioViewer
The viewpoint to which a camera is applied.
ipICamera
The pointer to the interface CATICamera which defines the camera to apply.
Returns:
S_OK
if the query succeeds
E_FAILED
otherwise.
