public class CATCmdAccess

Base class to create an access.
Role: When you want to arrange commands to propose them appropriately to the end user, you need to either create a workshop or a workbench, or create an add-in to an existing workshop or workbench. When you create such objects, you create in fact a containment tree structure to access to the commands. A node of the tree is a , a for the workbench or a for the workshop, and a leaf can be a or a . All these objects are therefore called accesses and their base class is CATCmdAccess.
