ApplicationFrame Global Function CATAfrSetCommandHeaderIconName


HRESULT CATAfrSetCommandHeaderIconName( const CATString& iCommandHeaderId,
const CATString& iIconName)
Changes the icon of a command header.
Parameters:
iCommandHeaderId
The identifier of the command header.
iIconName
The name of the icon.
Returns:
S_OK
the service has succeeded and the icon was changed.
E_FAIL
the service has failed and the icon was not changed. Notice that no check is performed on the icon existence.
