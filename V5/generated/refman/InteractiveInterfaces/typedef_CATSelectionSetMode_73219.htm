InteractiveInterfaces Typedef CATSelectionSetMode

typedef unsigned long CATSelectionSetMode
The Selection Set mode.

Role: The Selection Set mode is used in the CATISelectionSetsFactory.FindSelectionSets , CATISelectionSetsFactory.ListSelectionSets , CATISelectionSetsFactory.FindOwningSets methods to specify if the path of the Selection Set is a relative path in the document or a path that contains the CATProduct.
Values:
CATSelSetModeCurrent
To retrieve current document Selection Sets.
CATSelSetModeRecursive
To retrieve CATProduct structure tree document Selection Sets.
This object is included in the file: CATISelectionSetsFactory.h
