interface CATIFVertexAccess

Interface to filter a selecting object of Vertex types.
Role: This interface allows to filter a selecting object ( CATIBRepAccess ) of Vertex type. All vertices types derive from this interface, whatever the topologic object they belong to may be ( CATBody ).

This object is included in the file: CATIFVertexAccess.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
