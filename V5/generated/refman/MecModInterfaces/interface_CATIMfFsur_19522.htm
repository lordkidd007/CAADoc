interface CATIMfFsur

Interface to filter a selecting feature of Fsur (for functional surface) type.
Role: This interface allows you to filter a selecting feature ( CATIMfBRep ) of Fsur type.
This is the featurization ( CATIFeaturize ) of a CATIRSur object. This featurization from the CATIRSur has to be done in MfFunctionalFeaturization mode to create a CATIMfFsur. See CATMfFeaturizeMode for featurization modes.

This object is included in the file: CATIMfFsur.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
