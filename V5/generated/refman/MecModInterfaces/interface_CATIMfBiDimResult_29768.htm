MecModInterfaces Interface CATIMfBiDimResult

Usage: an implementation of this interface is supplied and you must use it as is. You should not reimplement it.

interface CATIMfBiDimResult

Interface to filter features whose topological result is two dimensional.
Role: Query this interface on the geometrical feature to be checked. The query will succeed if all the domains of the resulting body are two dimensional.

This object is included in the file: CATIMfBiDimResult.h
If needed, your Imakefile.mk should include the module: CATMecModInterfaces
