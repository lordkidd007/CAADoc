interface CATIRib

Class to manage Rib feature.
Role: Rib shape is made up of a profile swept along a center curve. Profile and centercurve are represented by sketches.
Rib shape is a positive shape: it adds material to the mechanical body it belongs to.
The behavior of this interface is defined in the CATISweep interface.

See also:
CATISweep, CATISlot

This object is included in the file: CATIRib.h
If needed, your Imakefile.mk should include the module: CATPartInterfaces
