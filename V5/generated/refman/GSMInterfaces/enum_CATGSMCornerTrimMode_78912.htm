GSMInterfaces Enumeration CATGSMCornerTrimMode

enum CATGSMCornerTrimMode {
  CATGSMCornerNoTrim,
  CATGSMCornerFullTrim,
  CATGSMCornerTrimElement1,
  CATGSMCornerTrimElement2
}
Corner Trimming Mode.
Values:
CATGSMCornerNoTrim
corner without trim.
CATGSMCornerFullTrim
corner with trim of Element 1 and Element 2.
CATGSMCornerTrimElement1
corner with trim of Element 1.
CATGSMCornerTrimElement2
corner with trim of Element 2.
