GSMInterfaces Enumeration CATGSMPositionDirCompMode

enum CATGSMPositionDirCompMode {
  CATGSMPositionDirCompMode_None,
  CATGSMPositionDirCompMode_Tangent,
  CATGSMPositionDirCompMode_User
}
Types for axis computation mode.
Values:
CATGSMPositionDirCompMode_None
No X axis specified
CATGSMPositionDirCompMode_Tangent
The X axis is implicitly the tangent of the profile at the origin
CATGSMPositionDirCompMode_User
User defined X axis

CATIGSMPositionTransfo, CATIGSMSweepUnspec
