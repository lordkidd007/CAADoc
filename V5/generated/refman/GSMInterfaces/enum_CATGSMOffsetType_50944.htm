GSMInterfaces Enumeration CATGSMOffsetType

enum CATGSMOffsetType {
  CATGSMOffset_Constant,
  CATGSMOffset_Rough,
  CATGSMOffset_Smooth,
  CATGSMOffset_SmoothwithDevMax
}
Types for Offset operations.
Values:
CATGSMOffset_Constant
Classic constant Offset.
CATGSMOffset_Rough
Rough Offset.
CATGSMOffset_Smooth
Offset with automatic smoothing.
CATGSMOffset_SmoothwithDevMax
Offset with smoothing limited by a user deviation.
See also:
CATIGSMOffset, CATIGSMFactory.CreateOffset
