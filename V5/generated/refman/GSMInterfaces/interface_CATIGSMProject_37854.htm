interface CATIGSMProject

Interface to project feature.
Role: Allows you to access data of the Project feature created by using an element, a support and a direction. The direction is used if the projection is not normal. Note Solution Type: 0 -> Nearest solution Solution Type: 1 -> All solutions

See also:
CATGSMProjectSolutionType
