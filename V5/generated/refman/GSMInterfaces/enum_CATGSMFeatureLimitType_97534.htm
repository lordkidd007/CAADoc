GSMInterfaces Enumeration CATGSMFeatureLimitType

enum CATGSMFeatureLimitType {
  CATGSMFeatureLimitType_Unknown,
  CATGSMFeatureLimitType_Dimension,
  CATGSMFeatureLimitType_UptoElement
}
Types for Limit types.
Values:
CATGSMFeatureLimitType_Unknown
Unknown limit type.
CATGSMFeatureLimitType_Dimension
If Limit type is Dimension, it implies that limit is defined by length.
CATGSMFeatureLimitType_UptoElement
If Limit type is UptoElement, it implies that limit is defined by a geometrical element.
