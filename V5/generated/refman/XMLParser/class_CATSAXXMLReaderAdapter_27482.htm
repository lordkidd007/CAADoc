Usage: you must use this class as is. You should never derive it.

public class CATSAXXMLReaderAdapter

Class to adapt a SAX2 CATISAXXMLReader as a SAX1 CATISAXParser.

This class wraps a SAX2 and makes it act as a SAX1 . The must support a TRUE value for the http://xml.org/sax/features/namespace-prefixes property or parsing will fail with a ; if the supports a FALSE value for the http://xml.org/sax/features/namespaces property, that will also be used to improve efficiency
