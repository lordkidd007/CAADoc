XMLParser CATSAXParserAdapter

Usage: you must use this class as is. You should never derive it.

public class CATSAXParserAdapter

Class to adapt a SAX1 CATISAXParser as a SAX2 CATISAXXMLReader.

This class wraps a SAX1 and makes it act as a SAX2 , with feature, property, and Namespace support. Note that it is not possible to report events, since SAX1 does not make that information available. This adapter does not test for duplicate Namespace-qualified attribute names.
