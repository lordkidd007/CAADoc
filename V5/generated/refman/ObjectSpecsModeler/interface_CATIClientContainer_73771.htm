o ListMembers
public virtual HRESULT ListMembers( const IID& iIID,
const CATUnicodeString ClientID,
CATListPtrCATBaseUnknown** oLst)const =0
Returns a list of features inside a container, filtered by interface and client id.
Role:On a Container, we can ask for objects adhering to a given interface and belonging to a client identified by a string.
Parameters:
iIID
IID of the interface.
ClientID
String identifying the Client. The names of its StartUps were suffixed at creation by this string.
oLst
List containing the requested features.
