public class CATSurTo2DGlobalMapping

Class defining a consistant global mapping between a set of CATSurface and a virtual 2 dimension space.

This class is the base class for some supplied operators such as CATSurTo2DGlobalPlanarMapping, but you must not derive it yourself.
All global computations are done once when constructing the CATSurTo2DGlobalMapping.
