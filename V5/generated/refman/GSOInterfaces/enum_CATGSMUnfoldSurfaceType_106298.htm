GSOInterfaces Enumeration CATGSMUnfoldSurfaceType

enum CATGSMUnfoldSurfaceType {
  CATGSMUndefinedSurfaceType,
  CATGSMRuledSurfaceType,
  CATGSMAnySurfaceType
}
Define the type of surface to unfold.
Role: Used to know which is the type of the surface to unfold.
Values:
CATGSMUndefinedSurfaceType
The type of surface is not defined
CATGSMRuledSurfaceType
The type of surface is ruled
CATGSMAnySurfaceType
The type of surface is all
See also:
CATIGSMUnfold, CATIGSOFactory
