Usage: you must use this class as is. You should never derive it.

public class CATMSHCustomHighlight

Interface representing a pre-highlight and highlight of a finite element entity.
Role: This interface provides a customization of the highlight behavoiur on : element, node, finite element's edge, finite element's face.
