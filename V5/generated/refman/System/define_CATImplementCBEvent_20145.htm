System Macro CATImplementCBEvent

Macro CATImplementCBEvent (classname,name,debug_chain)
Creates a callback event, given its name.
Role: The event created is assigned to the class that defines it. This class is afterwards able to publish this event, and is therefore called an event publisher. Insert this macro in the source (.cpp) file of the class that should define the event.
Precondition: Use the macro in the class header file to create the event.
Parameters:
classname
The name of the class that creates the event
name
The event name
debug_chain
A character string you can use for debug
This object is included in the file: CATCallbackManager.h
