System Global Function CATDeleteDirectory


CATLibStatus CATDeleteDirectory( const char* iPath)
Deletes a directory.
Role: Deletes a directory, that must exist and be already empty.
Parameters:
iPath
Specifies the name of the directory to be deleted. Legal values
CATLibSuccess
The directory was successfully destroyed
CATLibError
An error has occured.
