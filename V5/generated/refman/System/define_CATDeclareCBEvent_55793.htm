System Macro CATDeclareCBEvent

Macro CATDeclareCBEvent (name)
Declares an event, given its name.
Role: The event created is assigned to the class that defines it. This class is afterwards able to publish this event, and is therefore called an event publisher. Insert this macro in the header file of the class that should define the event.
Postcondition: Use the CATImplementCBEvent macro in the class source (.cpp) file to create the event.
Parameters:
name
The event name
This object is included in the file: CATCallbackManager.h
