interface CATIUExitCrypt

Interface to enable some user specific encryption mechanism.
Role: This interface is a user exit, that allows an user to perform encryption/decryption of data. It is called when transmitting passwords.
You should implement it on the CATUExitCrypt component by creating a data extension.
