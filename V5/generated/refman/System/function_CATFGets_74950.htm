System Global Function CATFGets


HRESULT CATFGets(char* ioBuff,
int iNb,
unsigned int iFileDesc)
Reads a line from a file.
Role: Reads characters from the file into the array pointed to by ioBuff, until iNb-1 characters are read, or a newline character is read and transferred to ioBuff, or an end-of-file condition is encountered. The string is then terminated with a null character.
Parameters:
ioBuff
the buffer where the read characters will be stored.
iNb
The size of the buffer.
iFileDesc
the file descriptor that has been obtained through
CATFOpen
Returns:
Legal values:
S_OK :on Success.
negative : on failure
This object is included in the file: CATStdIO.h
