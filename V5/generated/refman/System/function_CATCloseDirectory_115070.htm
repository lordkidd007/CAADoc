System Global Function CATCloseDirectory


CATLibStatus CATCloseDirectory(CATDirectory* iDir)
Closes a directory.
Role: Closes the directory designated by the iDir handle This handle has been obtained by a previous call to CATOpenDirectory
Parameters:
iDir
the CATDirectory Handle on the directory to be closed
Returns:
Legal values
CATLibSuccess
the directory is closed
CATLibError
An error has occured.
This object is included in the file: CATLib.h
