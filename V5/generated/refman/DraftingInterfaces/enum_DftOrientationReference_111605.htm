DraftingInterfaces Enumeration DftOrientationReference

enum DftOrientationReference {
  DftOrientSheet,
  DftOrientView,
  DftOrientElement
}
Define the possible orientation reference of an annotation.
Sheet : the angle of the annotation is given in the sheet.
View : the angle of the annotation is given in its view.
Element : the angle of the annotation is given associatively to an element.
