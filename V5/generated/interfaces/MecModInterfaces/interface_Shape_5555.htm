Shape (Object)

IUnknown
  |
  +---IDispatch
    |
    +---CATBaseUnknown
      |
      +---CATBaseDispatch
        |
        +---AnyObject
          |
          +---Shape

Represents the Shape object.
It is an abstract object which is not intended to be created as such, from which all objects having a shape representation derive.
