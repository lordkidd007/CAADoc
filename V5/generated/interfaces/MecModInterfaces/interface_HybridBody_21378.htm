HybridBody (Object)

The object is a hybrid body.
The hybrid body manages a set of geometric elements, a set of hybrid shapes, a set of bodies and a set of hybrid bodies.
It belongs to the HybridBodies collection of a Part or @ref CATIABody or HybridBody object.

Property Index

Bodies
Returns the hybrid body's Bodies collection.
GeometricElements
Returns the list of geometrical elements included in the hybrid body.
HybridBodies
Returns the hybrid body's HybridBodies collection.
HybridShapes
Returns the list of hybrid shapes included in the hybrid body.
HybridSketches
Returns the hybrid body's Sketches collection.

Method Index

AppendHybridShape
Appends a hybrid shape to the hybrid body.

Properties


o Property Bodies() As CATIABodies (Read Only)
Returns the hybrid body's Bodies collection.
Example:
The following example returns in bodyColl the collection of bodies of the hybrid body hybridBody : 
 Set bodyColl = hybridBody.Bodies

for more information).

o Property HybridSketches() As CATIASketches (Read Only)
Returns the hybrid body's Sketches collection. These skectches are independent from any shape, since they haven't yet been used to create shapes.
Example:
The following example returns in skColl the collection of independent sketches of a hybrid body : 
 Set skColl = hybridBody.HybridSketches
