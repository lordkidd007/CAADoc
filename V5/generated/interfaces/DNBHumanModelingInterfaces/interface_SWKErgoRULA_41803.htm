SWKErgoRULA (Object)

This interface deals the RULA ergonomic analysis.

Property Index

ArmAbducted
This property is True if the arm of the manikin is abducted, given the manikin's current posture.
ArmOutToSide
This property is True if the arm of the manikin is out to the side, given the manikin's current posture.
NeckSideBending
This property is True if the neck of the manikin is bent, given the manikin's current posture.
NeckTwisted
This property is True if the neck of the manikin is twisted, given the manikin's current posture.
ObjectWeight
Returns or sets the weight of the object being carried.
OverallScore
This field is the overall RULA score, given the manikin's current posture.
ShoulderRaised
This property is True if the shoulder of the manikin is raised, given the manikin's current posture.
TrunkSideBending
This property is True if the trunk of the manikin is bent, given the manikin's current posture.
TrunkTwisted
This property is True if the trunk of the manikin is twisted, given the manikin's current posture.
Verdict
This field gives a text string which represents the interpretaton of the overall RULA score.
WristBent
This property is True if the wrist of the manikin is bent, given the manikin's current posture.
WristTwisted
This property is True if the wrist of the manikin is twisted, given the manikin's current posture.
