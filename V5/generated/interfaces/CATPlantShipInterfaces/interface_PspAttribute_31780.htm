PspAttribute (Object)

IUnknown
  |
  +---IDispatch
    |
    +---CATBaseUnknown
      |
      +---CATBaseDispatch
        |
        +---AnyObject
          |
          +---PspAttribute

Represents Attribute Interface to query Plant Ship objects' attributes.
Role: To query and reset attributes.
