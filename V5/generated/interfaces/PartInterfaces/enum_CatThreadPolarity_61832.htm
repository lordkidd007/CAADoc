CatThreadPolarity (Enumeration)

enum CatThreadPolarity {
  catThread,
  catTap
}
Thread/Tap Polarity Management It describes the thread polarity.
Values:
catThread
The thread/tap polarity is explicitly set to Thread
catTap
The thread/tap polarity is explicitly set to Tap
