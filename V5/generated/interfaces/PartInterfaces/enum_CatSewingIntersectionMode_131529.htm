CatSewingIntersectionMode (Enumeration)

enum CatSewingIntersectionMode {
  catSewingNoIntersect,
  catSewingIntersect
}
Sewing Intersection mode.
The sewing operation parameter for specifying which algorithm is used to perform sewing operation
Values:
catSewingNoIntersect
No intersection is computed to perform the operation.
catSewingIntersect
Intersection is computed to perform the operation
