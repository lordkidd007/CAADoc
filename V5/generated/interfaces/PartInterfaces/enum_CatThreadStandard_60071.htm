CatThreadStandard (Enumeration)

enum CatThreadStandard {
  catMetricThinPitch,
  catMetricThickPitch
}
Thread/Tap Standard It describes the standard to be used for thread/tap.
Values:
catMetricThinPitch
The thread/tap standard is Metric Thin Pitch
catMetricThickPitch
The thread/tap standard is Metric Thick Pitch
