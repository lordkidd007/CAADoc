CatSampledAnalysisMode (Enumeration)

enum CatSampledAnalysisMode {
  CatSampledAnalysisOff,
  CatSampledAnalysisOn,
  CatSampledAnalysisStop,
  CatSampledAnalysisVerbose
}
Possible values of how analysis is to be used when it is added to a sampled based object.
Values:
CatSampledAnalysisOff
Mode to set the analysis to Off
CatSampledAnalysisOn
Mode to set the analysis to On
CatSampledAnalysisStop
Mode to set the analysis to Stop
CatSampledAnalysisVerbose
Mode to set the analysis to Verbose
