Behaviors (Object)

Represents the Behaviors collection currently managed by the application.
These behaviors belong to one objet and can be reached by the GetItem method of CATIABase. For Instance onto a Part using Part.GetItem("CATGetBehaviorExtension") .
Example: 
 Set RootPart = CATIA.ActiveDocument.Part
 Set MyExtension = RootPart.GetItem("CATGetBehaviorExtensions")
 Set listBehaviors = MyExtension.Behaviors

Method Index

Item
Returns a Behavior using its index or its name from the Behaviors collection.

Methods


o Func Item( iIndex) As
Returns a Behavior using its index or its name from the Behaviors collection.
Parameters:
iIndex
The index or the name of the Behavior to retrieve from the collection of Behaviors. As a numerics, this index is the rank of the Behavior in the collection. The index of the first Behavior in the collection is 1, and the index of the last Behavior is Count. As a string, it is the name you assigned to the Behavior using the
property.
Returns:
The retrieved Behavior
Example:
This example retrieves in ThisBeh the fifth Behavior in the collection and in ThatBeh the Behavior named MyBeh. 
 Dim ThisBeh As Behavior
 Set ThisBeh = listBehaviors.Item(5)
 Dim ThatBeh As Behavior
 Set ThatBeh = listBehaviors.Item("MyBeh")
