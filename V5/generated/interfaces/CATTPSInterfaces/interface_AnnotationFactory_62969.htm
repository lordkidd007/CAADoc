AnnotationFactory (Object)

IUnknown
  |
  +---IDispatch
    |
    +---CATBaseUnknown

      +---CATBaseDispatch
        |
        +---AnyObject
          |
          +---Factory
            |
            +---AnnotationFactory

Interface for the TPS Factory.
This factory is implemented on the Set object. All the created specifications are added to the Set from which this interface is retrieved.
