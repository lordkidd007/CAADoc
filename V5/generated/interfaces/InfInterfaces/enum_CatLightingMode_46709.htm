CatLightingMode (Enumeration)

enum CatLightingMode {
  catInfiniteLightSource,
  catNeonLightSource
}
Lighting modes.
They are used by the Viewer3D object.
Values:
catInfiniteLightSource
An infinite light source
catNeonLightSource
A light source simulating a set of parallel neon tubes.
