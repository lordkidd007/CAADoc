HybridShapeMidSurface (Object)

Represents the hybrid shape MidSurface feature object.
Role: To access the data of the hybrid shape MidSurface feature object.
This data includes:
  • Support Body
  • Creation Mode
  • Threshold Thickness

Use the CATIAHybridShapeFactory to create a HybridShapeMidSurface object.
