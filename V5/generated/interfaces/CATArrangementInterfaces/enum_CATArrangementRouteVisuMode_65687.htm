CATArrangementRouteVisuMode (Enumeration)

enum CATArrangementRouteVisuMode {
  CatArrangementRouteVisuModeCurve,
  CatArrangementRouteVisuModeFlat,
  CatArrangementRouteVisuModeSolid
}
The visualization mode for the ArrangementRun, ArrangementPathway, ArrangementBoundary objects.
