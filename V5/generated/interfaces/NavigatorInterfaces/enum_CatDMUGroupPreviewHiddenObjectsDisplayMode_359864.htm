CatDMUGroupPreviewHiddenObjectsDisplayMode (Enumeration)

enum CatDMUGroupPreviewHiddenObjectsDisplayMode {
  CatDMUGroupPreviewShowHidden,
  CatDMUGroupPreviewShowHiddenCustom
}
The different modes for the display of hidden objects in DMU Group Preview.
See also:
N4DNavigatorSettingAtt
Values:
ShowHidden
Hidden objects are visualized as in main 3D viewer.
ShowHiddenCustom
Hidden objects are visualized with customized graphic properties.
