CatTableComputeMode (Enumeration)

enum CatTableComputeMode {
  CatTableComputeOFF,
  CatTableComputeON
}
Compute mode of a table.
Values:
CatTableComputeOFF
Compute Mode is set to OFF: the table is not displayed.
CatTableComputeON
Compute Mode is set to ON: the table is displayed.
