SfmSplitPlates (Object)

Interface to manage the Structure Functional Modeler Split Plates object.
Role: Provides access to split plates of a super plate as a collection.

Method Index

Item
Gets Split Plate.

Methods


o Func Item( iIndex) As
Gets Split Plate.
Parameters:
iIndex
[in] Index of the Split Plate to be retrieved.
oSfmSplitPlate
[out] The retrieved Split Plate.
Returns:
S_OK if everything ran ok
Example
:
This example gets existing Split Plate of a Super Plate. 
 Dim ListSplitPlates As SfmSplitPlates
 Set ListSplitPlates = SuperPlate.SplitPlatesObjects
 Dim Split_Plate As SfmSplitPlate
 Set Split_Plate = ListSplitPlates.Item(i)
