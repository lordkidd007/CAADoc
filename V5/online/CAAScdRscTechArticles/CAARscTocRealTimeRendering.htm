Real Time Rendering objects are obtained from a Product object through three collections:

  • A collection of RenderingEnvironment objects. A RenderingEnvironment object defines a space zone that has an origin and a direction. Its shape may spherical, cylindrical or cubic, depending on its Type property. Limiting faces of the zone can be accessed as RenderingEnvironmentWall objects that may or not be visible and support a texture definition.
  • A collection of UserRenderingLights objects. A RenderingLight object defines a light source that is positionned and oriented in space.
  • A collection of RenderingShooting objects representing screen captures. Such a capture is performed in a rendering environment, using a 3D camera that defines the viewpoint and one or more light sources.

Those objects are accessed through the GetItem method of the root Product object of a CATProduct document as described below:

' Accessing the collection of rendering environments
Dim cEnvs As RenderingEnvironments
Set cEnvs = oRootProduct.GetItem("CATRscRenderingEnvironmentVBExt")

' Accessing the collection of rendering lights
Dim cLights As RenderingLights
Set cLights = oRootProduct.GetItem("CATRscRenderingLightVBExt")

' Accessing the collection of rendering shootings 
Dim cShoots As RenderingShootings
Set cShoots = oRootProduct.GetItem("CATRscRenderingShootingVBExt")
