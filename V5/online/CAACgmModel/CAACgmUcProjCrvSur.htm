Projecting a Curve onto a Surface
Use Case

Abstract

A curve can be projected onto a surface by using the CATICGMProjectionCrvSur operator. The result is a set of geometrical objects (curves and/or points) that you have to scan.

  • Operator to be Used
  • Use Case Description
  • References

Operator to be Used

Use CATICGMProjectionCrvSur. This operator is created by using the CATCGMCreateProjection global function.

Use Case Description

The CAAGMModelProjectionOpe.m module in CAAGMModelInterfaces.edu illustrates how to project a curve onto a surface. This use case constructs its input data. If you are not already familiar with geometric modeler use cases, go to About Geometric Modeler Uses Cases.
