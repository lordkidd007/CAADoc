Topological Objects

This section deals with Topological Objects.
In this section:
  • Topology Concepts introduces the CGM topology objects, and the manifold and non-manifold topologies.
  • The CGM Topological Model describes how the topology concepts are implemented.
  • How to Associate Topology with Geometry defines the rules that exist between topological and geometric objects.
  • Using the Smart Duplicator explains how to modify a touched topology.
  • Using the Topological Objects is an example showing creating and navigating the topological objects using a tetrahedron.
  • Scanning an Edge Curve is an example showing how to use the CATEdgeCurveIterator operator.
