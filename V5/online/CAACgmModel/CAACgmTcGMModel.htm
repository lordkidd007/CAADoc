GM Model

The GM Model component provides you with all the services to create and manage a basic GM model. With GM Model, you can create your own function, create and manage geometric and topological objects, manage their life cycle, browse them by using tessellation API and performs measures and analysis on them. The major part of this component is made up of geometric operators.
In this section:
  • Advanced Mathematics describes how to create your own mathematical functions.
  • Geometric Objects and Operators describes how to create objects, manage their life cycle and perform some basic operations on these objects.
  • Topological Objects describes how to create directly topology. This part is intended to make you understand the principles behind topology with a use case explaining how to create associate topology with geometry. However, to create topology it is recommended to use the services provided by the GM Operators component.
  • Tessellation describes how to discretize GM objects in a basic CAA application and make them browsable.
