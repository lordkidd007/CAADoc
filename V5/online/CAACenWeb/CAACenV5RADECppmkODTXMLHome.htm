C++ Unit Test Manager - mkodt (CUT)
The shell version of the C++ Unit Test Manager - mkodt (CUT) is deprecated starting with V5R24 and available any longer by default. The mkodt command processes XML test objects only. See Adopting XML Test Objects.
If necessary and as a by-pass, to still use shell test objects, set or export the variable MKODT_ENABLE_SH_MODE=1.
