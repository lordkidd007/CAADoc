What's New in V5R10 in CAA V5 Automation Objects

This article presents automation object, methods and properties that have been added since V5R9 GA to enhance CAA V5 automation capabilities, support functional evolutions and correct defects. Some of those modification already exist on V5R9 service packs. Only one method has been modified, VB and VBScripts code using this method must be modified. Due to the addition of methods on existing objects, VB projects must be recompiled.
