Modifying the Face Orientations in a Shell
Use Case

Abstract

The faces in a shell do not have necessarily the same orientation as the shell itself. You can invert the orientation of the faces so that they all point to the same direction, the shell orientation.

  • Operator to be Used
  • Use Case Description
  • References

Operator to be Used

To modify the orientation of faces so that they all have the same orientation as the shell, you must use the CATICGMTopShellOrientation operator which is created by the CATCGMCreateTopShellOrientation global function. This is the only way to modify the orientation of faces in a shell.

Use Case Description

The CAAGMOperatorsShellOrientation.m module in CAAGMOperatorsInterfaces.edu illustrates how to modify face orientations. This use case is to be run with the FaceOrientation.NCGM input file which is delivered in CAAGMOperatorsInterfaces.edu/FunctionTests/InputData. If you are not already familiar with geometric modeler use cases, go to About Geometric Modeler Uses Cases.
