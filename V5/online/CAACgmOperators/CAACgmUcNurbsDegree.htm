Modifying the Degree of a Nurbs
Use Case

Abstract

The CAAGMOperatorsInDegree.m module explains how to modify the degree of a Nurbs by using the CATICGMCrvDegreeModification operator. This piece of code is not documented as from now on, there is a much more straightforward way to do the same operation by using the Nurbs toolkit delivered in CATICGMNurbsCurveTools.h.

From now on, to modify the degree of a NURBS, use directly the SetDegree method of CATICGMNurbsCurveTools.h in GMModelInterfaces.
