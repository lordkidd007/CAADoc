ElectricalModeler

Role: Define electrical wire routing - General settings

Detailed role:Define electrical wire routing - General settings

The ElectricalModeler repository has the following characteristics:

  • Primary Tab Page: /Equipment & Systems/Electrical Wire Routing/General
  • Repository File: ElectricalModeler
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalModeler setting repository attributes:

  • BNSEquipotentialDiaFactor
    Role: Ratio to add to equivalent area in percentage
    Detailed role: Setting to manage the ratio to add to equivalent area in percentage for diameter computation
    • Type: double
    • Default value: 20
    • Default Value Role: By default the tolerance is set to 20 percent
  • BendRadiusMode
    Role: Bend radius compute mode
    Detailed role: Bend radius compute mode
    • Type: string
    • Default value: Always
  • CheckCreationMode
    Role: Branchable Creation Mode
    Detailed role: Branchable Bend Radius to be computed based on bend radius ratio and max. diameter of segment after wire routing
    • Type: integer
    • Default value: 0
    • Default Value Role: By default do not take BRR into account
    • Values:
      0 Do not take bend radius ratio into account
      1 Take bend radius ratio into account
  • ClientId
    Role: Client ID
    Detailed role: Client ID for user dictionary
    • Type: string
    • Default value:
  • ComputeBNSDiaMode
    Role: Equivalent area method option
    Detailed role: If checked, bundle segments diameter will be computed using equivalent area method
    • Type: integer
    • Default value: -1
    • Default Value Role: By default do not use equivalent area method to compute bundle segment diameter
    • Values:
      -1 Do not use equivalent area method to compute bundle segment diameter
      0 Use equivalent area method to compute bundle segment diameter
  • ComputeFromWire
    Role: Wire taken into accound
    Detailed role: Bundle segments diameter and bend radius will be computed from its contents, i.e. wires, wire groups
    • Type: integer
    • Default value: -1
    • Default Value Role: By default take Wire into account
    • Values:
      -1 Take Wire into account
      0 Do not take Wire into account
  • UserDictionary
    Role: User dictionary
    Detailed role: User dictionary for user
    • Type: string
    • Default value:
