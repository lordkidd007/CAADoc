CircuitBoardDesign2 Setting Repository
Quick Reference

CircuitBoardDesign2

Role: Define circuit board design settings

Detailed role:Define circuit board design settings

The CircuitBoardDesign2 repository has the following characteristics:

  • Primary Tab Page: /Equipment & Systems/Circuit Board Design/Components
  • Repository File: CircuitBoardDesign
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

CircuitBoardDesign2 setting repository attributes:

  • CatalogPath
    Role: Catalog file path
    • Type: string
    • Default value: No File
  • ComponentHeigth
    Role: Default Heigth of the component
    Detailed role: Default Heigth of the component coming from library file
    • Type: double
    • Default value: 0.1
  • CopperThickness
    Role: Describes the copper thickness inside hole
    • Type: double
    • Default value: 0.001
  • ReportFolderPath
    Role: Report Folder Path of the CAA partner
    • Type: string
    • Default value: c:\
  • WithConstraint
    Role: Lock the position of the component with constraints
    Detailed role: Option to lock the position of the component with constraints
    • Type: integer
    • Default value: 0
    • Values:
      0 Do not lock the position of the component
      1 Lock the position of the component
  • WithIDF3Feature
    Role: Import the IDF filer with the IDF3.0 + features
    Detailed role: Import the IDF filer with the IDF3.0 + features using the CAA partner implementation
    • Type: integer
    • Default value: 1
    • Values:
      0 No IDF3.0 + feature
      1 With IDF3.0 + feature
  • WithRandomColorForComponent
    Role: Apply a random color to the representation of the component
    Detailed role: Option for applying a random color to the representation of the component when importing it from a library
    • Type: integer
    • Default value: 1
    • Values:
      0 No color apply
      1 A color is applied
