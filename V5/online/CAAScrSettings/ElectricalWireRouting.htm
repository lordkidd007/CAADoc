ElectricalWireRouting

Role: Define electrical wire routing settings

Detailed role:Define electrical wire routing settings

The ElectricalWireRouting repository has the following characteristics:

  • Primary Tab Page: /Equipment & Systems/Electrical Wire Routing/General
  • Repository File: ElectricalWireRouting
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

ElectricalWireRouting setting repository attributes:

  • CurrentExtremityColor
    Role: Define extremity color
    • Type: unsigned integer
    • Default value: 0
    • Default Value Role: Red in RGB order
    • Min value: 0
    • Max value: 255
  • CurrentSegregationIndex
    Role: Index of the selected paths of the Segregation file list
    Detailed role: Paths of the Segregation file containing a table of separation code
    • Type: integer
    • Default value: -1
  • HandleCKESepCode
    Role: Specify the seperation code file/rule
    • Type: integer
    • Default value: 0
    • Values:
      0 Specify the seperation file path using file browser
      1 No rule
      2 Use of a rule for seperation code
  • HandleSigSigSepCode
    Role: Compatibility mode
    Detailed role: The compatibility between wires or wires and segment can be checked according to this setting
    • Type: integer
    • Default value: 0
    • Values:
      0 Compatibility between signal-arc is checked
      1 Compatibility between signal is checked
  • NumberAttribute
    Role: Specify the type of the element in favorites list
    • Type: integer
    • Default value: 3
  • RuleText
    Role: Rule used for seperation code

    Role: Size of the path list of Segregation file
    Detailed role: Size of the path list of Segregation file
    • Type: integer
    • Default value: 0
  • SegregationTable
    Role: Paths of the Segregation file list
    Detailed role: Paths of the Segregation files list containing a table of separation code
    • Type: string
