Quick Reference

DSESettingsGeneral

Role: Settings to manage Digitized Shape Editor General Options

The DSESettingsGeneral repository has the following characteristics:

  • Primary Tab Page: /Shape/Digitized Shape Editor/General
  • Repository File: DSESettingsGeneral
  • Version: 1
  • Full Compatibility: 1
  • Upward Compatibility: 1

DSESettingsGeneral setting repository attributes:

  • UndoActivated
    Role: Toggle to activate/deactivate Undo/Redo for in-place cloud edition commands
    • Type: integer
    • Default value: 0
    • Min value: 0
    • Max value: 1
  • WarmStartNbOfPoints
    Role: Maximum number of points to let the warmstart work properly
    • Type: integer
    • Default value: 1000000
    • Min value: 0
    • Max value: 1000000000
