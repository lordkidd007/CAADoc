Creating a Framework

This task shows you how to create a framework in your workspace.
Before you begin: Note that:
  • Open the appropriate workspace.
  • See Component Rules to know about the different types of frameworks, and their relationship.

  1. In the 3DS Workspace Explorer, right click the workspace root node Workspace, and click New Framework.

    The 3DS - New Framework dialog box opens.

  2. In Framework name, type the name you want to assign to the framework. Refer to the C++ Naming Rules.

    Important: Type a name without spaces, and without any extension. If the framework function requires an extension, it will be added by the wizard.

  3. In Function, check the function appropriate to the framework you want to create.

    Function Description
    Test framework Creates a test framework for the test objects of your development or education frameworks.
    Education framework Creates an education framework for the use cases that demonstrate how to use your exposed classes, interfaces, etc.
    Development framework Creates a framework for your applications, commands, modelers, etc.

  4. If you have selected Development framework, in Type, check the framework type.

    Type Description
    Implementation Creates a framework containing the implementation part of your modeler.
    Interface Creates a framework containing the interface part of your modeler.

  5. If you intend to create IDL interfaces for Automation in your interface framework, check IDL support.

  6. Click Finish.

    The framework is created in the workspace along with the folders and files matching the function and type you declared.

Notes:
  • To see the created framework in the 3DS Workspace Explorer, right click the workspace name, and click Refresh.
  • You can refresh the Solution Explorer with the new framework: in the 3DS Workspace Explorer, right click the framework name, and click Refresh Solution.
