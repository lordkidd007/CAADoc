Installing C++ Interactive Dashboard Help

This task explains you how to install the C++ Interactive Dashboard help.

You can access this help using the View Help command of the Help menu, or open it contextually by clicking the question mark (?) command of the 3DEXPERIENCE Open Extensions for Native Apps dialog boxes. See 3DS Contextual Help. The API documentation is still available using the 3DS Help Viewer of the Help menu, or using the Alt+F1 shortcut from a class name selected in a source editor. See 3DS Help Viewer.

This task also explains you how to uninstall the C++ Interactive Dashboard help.

  • Installing C++ Interactive Dashboard Help
  • Uninstalling C++ Interactive Dashboard Help
Before you begin: Note that:
  • You should have first installed the RADE CD ROM.
  • You should also have set up RADE licenses.
  • You should also have installed C++ Interactive Dashboard from its VSIX. See Installing C++ Interactive Dashboard.
Warning You should be part of the HelpLibraryUpdaters group in the Local Users and Groups. To check this, in the Control Panel, click User Accounts, then click Change account type. In the User Accounts dialog box, click the Advanced tab, and click Advanced. In the lusrmgr dialog box, click Groups, double click HelpLibraryUpdaters, and check whether you are part of the group. If not, click Add to add you to this group.

Installing C++ Interactive Dashboard Help

To install C++ Interactive Dashboard help:

  1. Launch Visual Studio with administrator privileges.

  2. On the HELP menu, click Add and Remove Help Content.

    The Microsoft Help Viewer 2.0 window opens.

  3. In Installation source, check Disk.

  4. Either type or select from an explorer the path of the help file.

    This path is by default C:\Program Files (x86)\Dassault Systemes\B25\intel_a\resources\VisualStudio\HelpContent2012\wxpug.msha.

  5. Click Add.

  6. Click Update.

  7. Click Yes in the User Account Control prompt.

    The following prompt opens.

  8. Click Yes.

    The C++ Interactive Dashboard help is now installed.

The help is available for display:

  • To display the C++ Interactive Dashboard help:

    1. Launch Visual Studio.

    2. On the Help menu, click View Help.

      The Microsoft Help Viewer window opens.

    3. Click the Contents tab in the lower left corner.

    4. Click the arrow in front of Native Apps Development Toolset.

    5. Click Native Apps IDE.

      The Native Apps IDE help opens.

  • To contextually open the help, click ? from the 3DS plug-in dialog boxes which opens the help on the page describing the current dialog box.

Uninstalling C++ Interactive Dashboard Help

To uninstall the C++ Interactive Dashboard help:

  1. Launch Visual Studio.

  2. On the HELP menu, click Add and Remove Help Content.

    The Microsoft Viewer 2.0 window opens.

  3. Click the Manage Content tab.

    The available helps are displayed.

  4. Click Remove beside Native Apps Development Toolset.

  5. Click Update.

    The C++ Interactive Dashboard help is uninstalled.
