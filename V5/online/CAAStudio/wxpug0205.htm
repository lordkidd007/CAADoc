Creating Data

This task shows you create a data folder in a framework.

Before you begin:

Open a workspace containing at least a framework.

  1. In the 3DS Workspace Explorer, right click the name of framework for which you want to create a module, and click New Data.

    The 3DS - New Data dialog box opens.

  2. In Scope, check the scope of the data you want to create.

    Scope Description
    Generic Data valid for all operating systems and intended to be part of all the workspace run time views. Creates a CNext folder in you framework.
    Specific Data specific to one or several operating systems and intended to be part of the workspace run time views for these operating systems.
    Internal Data specific to one or several operating systems and not intended to be part of the workspace run time views for these operating systems.

  3. If you selected Specific or Internal, check the operating systems to which the data applies.

    Operating System Description Folder
    Specific Internal
    AIX IBM AIX operating system. CNext.specifics_AIX CNext.internal_AIX
    HP-UX Hewlett-Packard HP-UX operating system. CNext.specifics_HP-UX CNext.internal_HP-UX
    IRIX Silicon Graphic IRIX operating system. CNext.specifics_IRIX CNext.internal_IRIX
    Linux Linux operating system. CNext.specifics_Linux CNext.internal_Linux
    SunOS Sun Microsystems Solaris operating system. CNext.specifics_SunOS CNext.internal_SunOS
    Windows All Windows operating systems. CNext.specifics_Windows CNext.internal_Windows
    intel_a Windows 32-bit operating system. CNext.specifics_intel_a CNext.internal_intel_a
    UNIX All UNIX operating systems. CNext.specifics_UNIX CNext.internal_UNIX
    win_b Windows operating system. CNext.specifics_win_b CNext.internal_win_b
    win_b64 Windows 64-bit operating system. CNext.specifics_win_b64 CNext.internal_win_b64

  4. Click Finish.

    The folder corresponding to your selection is created.
