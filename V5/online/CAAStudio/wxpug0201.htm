Copying a Framework

This task shows you how to copy a framework from another workspace into the current one.
Before you begin::

Open the appropriate workspace.

  1. In the 3DS Workspace Explorer, right click the workspace root node Workspace, and click Copy Framework.

    The 3DS - Copy Framework dialog box opens.

  2. In Framework to copy, do one of the following:

    • Click Browse to select the framework you want to copy from its location.
    • Type the path name of this framework.
    • Type the framework name if this framework is located among the workspace prerequisites. If the framework is located in several prerequisite workspaces, a prompt opens to ask you to select from which of them you want to copy it.

  3. Click Finish.

    The framework is copied in your workspace. The 3DS Workspace Explorer is refreshed with this framework.

Note: You can refresh the Solution Explorer with the new framework: in the 3DS Workspace Explorer, right click the framework name, and click Refresh Solution Project.
