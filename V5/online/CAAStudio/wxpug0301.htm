Source Generation Overview

The source generation wizards enable you to create the source files of objects matching patterns such as C++ classes, Object Modeler interfaces and their implementations in components.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.

The source generation wizards are available from the Solution Explorer contextual menu.

  1. In the Solution Explorer, right click a module.

  2. In the contextual menu, point to Add, and click New Item.

    The Add New Item dialog box opens.

  3. In Categories, click the + sign in front of 3DS Templates to expand its contents.

  4. Click C++.

    The C++ installed templates are displayed in the Templates pane.

Eight templates are available:

  • In C++:
    1. A C++ class deriving from the CATCommand class (3DS Command)
    2. A C++ state command deriving from the CATStateCommand class (CATStateCommand).
    3. A C++ class (Class).
    4. An Object Modeler interface deriving from CATBaseUnknown (Interface).
    5. An Object Modeler component deriving from CATBaseUnknown to implement an interface (Component).
    6. An Object Modeler adapter (3DS Adapter).
  • In Tests:
    1. A shell test case (Test case).
    2. An XML test case (Xml test case).

The template in Dassault Systèmes Code Gen Creation Wizard is available if you have installed and have a license of Interactive Dialog Designer .

Notes: When creating source files from these templates, bear in mind that:
  • All the characters are allowed except those listed below: 
    * ? \ / < > " ' | :
  • With Windows, full path names must not exceed 260 characters.
