3DS Contextual Help

The 3DS contextual help gives access to the C++ Interactive Dashboard help.
Before you begin:
  • You should have first installed the C++ Interactive Dashboard help.
  • This help is dedicated to the Visual Studio customization. The full CAA Encyclopedia can be accessed from the installation folder, which is by default: C:\Program Files\Dassault Systemes\B25\CAAV5HomePage.htm.

The C++ Interactive Dashboard help is included in the Microsoft help, and displayed in the Microsoft Help Viewer.

To open this viewer, do one of the following:

  • On the Help menu, click View Help, then click Contents at the bottom left, and open a help section.
  • Click CTRL+F1 and open a help section.

To open it contextually, click in a 3DS plug-in dialog box to contextually open the Microsoft Help Viewer on the documentation of this dialog box.

You can navigate the tree, or use the hyperlinks within a page. You can also:

  • Use the index to find topics:

    1. On the bottom of the left pane, click Index.

      The index replaces the navigation tree.

    2. Type a topic in the Index box, or scroll the index.

    3. Click an index topic to open it in the right pane.

  • Search the documentation:

    1. In the Search Documentation box at the top right of the Microsoft Help Viewer, type a character string.

    2. Either click or press Enter.

      The found topics are displayed in the Results tab.

    3. Click an found topic to open it in the right pane.
