3DS - Workspace Finder

The Workspace Finder view shows all the workspace source files.

Before you begin:

Open the appropriate workspace. Note that the workspace must be built.

  1. On the View menu, point to 3DS Views, and click 3DS Workspace Finder.

    The 3DS - Workspace Finder view opens.

This view alphabetically lists the pathnames of all the source files included in the workspace, starting just below the workspace root folder. It includes a file filter, and shows the number of files listed with respect to the total workspace file count, such as 100/100 above.

The file filtering pattern can apply to the whole displayed pathnames, or to the file names only, thanks to a switch command displayed as a couple of icons.

Icon Filtering Pattern
Click this icon to apply the filtering pattern to the whole displayed file pathnames.
Click this icon to apply the filtering pattern to the file names only.

You can use the file filter as follows:

  • You can filter this list using a filtering pattern:

    1. In Filter, type a filtering pattern, for example *.h. The '*' character is the wild card character.

    2. Click or press Enter.

      Only the files matching the filtering pattern are displayed.

      All the .h files of the workspace are listed. They are 38 files out of 100.

  • To apply the filtering pattern to the whole pathnames:

    1. In Filter, type a filtering pattern, for example CAADialog.

    2. Click to apply the filtering pattern to the whole pathnames.

    3. Click or press Enter.

      All the files matching the CAADialog pattern in their whole pathnames are displayed, that is, in this case, all the files in the workspace. Note that the wild card character is not required.

  • To apply the filtering pattern to the file names only:

    1. In Filter, type a filtering pattern, for example CAADialog.

    2. Click to apply the filtering pattern to the file names only.

    3. Click or press Enter.

      All the files matching the CAADialog pattern in their file names only are displayed, that is, in this case, two files out of 100.
