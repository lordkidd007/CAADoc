Build Time Prompt

The Build Time Prompt command opens a command prompt sets up with the workspace build time environment.
Before you begin:

Open the appropriate workspace.

  1. On the Window menu, point to 3DS Windows, and click Build Time Prompt.

    A command prompt opens with the workspace root folder as the current one.

    The workspace build time environment is set up, as if you had typed the tck_init and tck_profile commands with appropriate options to match your workspace build time environment.

    This command prompt is dedicated to build time commands only, such as mkmk.

    Warning: Do not attempt to run applications from this command prompt, unless you have run the mkrun command with appropriate options to set up your workspace run time environment. Use the run time command prompt instead. See Run Time Prompt.
