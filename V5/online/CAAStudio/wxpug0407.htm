Copying the Path Name of a File

The Copy relative path command enables you to get the path name of the currently opened file relatively to the workspace root, or to the prerequisite workspace root. This relative path name is stored in the clipboard.
Before you begin:
  • Open a workspace with at least a framework and a module.
  • Display the Solution Explorer.
  • Open a header or a source file from the Solution Explorer.

  1. With a header or source file currently opened, on the Edit menu, point to 3DS Navigate, and click Copy relative path.

    The path name of this file relatively to its workspace root; either your workspace, or one of your workspace prerequisite workspace, is copied in the clipboard.

  2. Paste this path name, for example in the command line of a prompt, or in a file.

    For example, with the file CATBaseUnknown.h file opened, the relative path name retrieved is System\PublicInterfaces\CATBaseUnknown.h.
