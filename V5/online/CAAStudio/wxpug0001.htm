Installing C++ Interactive Dashboard

This task explains you how to install the C++ Interactive Dashboard customization from the VSIX installed with RADE.

C++ Interactive Dashboard enables you to manage your development workspaces and to access from within Visual Studio the RADE products such as the Multi-Workspace Application Builder (mkmk), the C++ Unit Test Manager (mkodt), or the C++ Source Checker (mkscc).

This task also explains you how to uninstall C++ Interactive Dashboard.

  • Installing C++ Interactive Dashboard
  • Uninstalling C++ Interactive Dashboard
Before you begin: Note that:
  • You should have first installed the RADE CD ROM. See Installing RADE.
  • You should also have set up the RADE licenses. See Setting up the RADE Licenses.

Installing C++ Interactive Dashboard

C++ Interactive Dashboard is installed from the 3DSOpenNativeAppsExt2012.vsix installation program installed with RADE. This VSIX is located in the intel_a\code\bin folder just below the folder where you have installed RADE (C:\Program Files (x86)\Dassault Systemes\B25 by default). This VSIX is valid for Microsoft Visual Studio 2012 only.

To install C++ Interactive Dashboard:

  1. Using an explorer, locate the 3DSOpenNativeAppsExt2012.vsix installation program in your RADE installation.

    Default location: C:\Program Files (x86)\Dassault Systemes\B25\intel_a\code\bin\3DSOpenNativeAppsExt2012.vsix

  2. Double click 3DSOpenNativeAppsExt2012.vsix.

    The VSIX dialog box opens. The Visual Studio version detected is displayed and checked.

  3. Click Install.

    The dialog displays a progress bar that then turns to Installation Complete.

  4. Click Close.

    C++ Interactive Dashboard is now installed.

  5. Start Visual Studio.

    The Options dialog box opens on 3DS.

  6. In Tools Initialization Command Path, set the path to the tck_init command. This command is located in the intel_a\code\command\intel_a\code\command folder just below the folder where you have installed RADE (C:\Program Files (x86)\Dassault Systemes\B25\intel_a\code\command\tck_init.bat by default). Set this path up to and including tck_init.bat.

  7. Click OK.

    A prompt asks you to restart Visual Studio.

    8. Click OK in this prompt.

  8. Exit Visual Studio.

    The C++ Interactive dashboard installation is complete.

    You can now install the documentation. See Installing C++ Interactive Dashboard Documentation.

Warning: If you have uninstalled a previous version of this vsix, check that the tools initialization command path (tck_init) is consistent with the current installation. Since the setting files are not deleted when uninstalling to let your workspaces available with the new version, this path might be wrong.

To check it, and possibly change it, see Accessing and Customizing Settings.

Uninstalling C++ Interactive Dashboard

To uninstall the C++ Interactive Dashboard:

  1. Launch Visual Studio.

  2. On the Tools menu, click Extension and Updates.

    The Extension and Updates dialog box opens.

  3. In the central pane, click 3DS Rich Apps Dev Studio.

  4. Click Uninstall.

    A prompt opens to ask you whether you want to go on uninstalling 3DS Rich Apps Dev Studio.

  5. Click Yes.

    The 3DS Rich Apps Dev Studio extension is removed from your computer.

Important:
  • The C++ Interactive Dashboard customization is no longer available from Visual Studio, but the vsix file from which you have installed the C++ Interactive Dashboard is not removed from your computer, and you can re-install it later on using the Visual Studio Extension Installer, as explained above.
  • The other tools, such as the Multi-Workspace Application Builder (mkmk), the C++ Unit Test Manager (mkodt), etc., are not embedded in this package, and are thus not uninstalled.
