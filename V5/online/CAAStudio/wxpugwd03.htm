Accessing and Customizing Environment

The 3DS environment is described using environment variables. You can edit, create, and delete environment variables.

  • Displaying the Environment Variables Dialog Box
  • Editing an Environment Variable
  • Creating an Environment Variable
  • Deleting an Environment Variable
Before you begin:

Open a workspace containing at least a framework.

Displaying the Environment Variables Dialog Box

  1. On the Tools menu, click 3DS Environment.

    The 3DS - Create or overwrite build time environment variables dialog box opens.

    The dialog box includes three columns showing the possible user icon, and the environment variable name and value respectively.

  2. To display user environment variables only, click at the top right of the dialog box.

    The user environment variables are denoted using in the first column.

Editing an Environment Variable

  1. In the 3DS - Create or overwrite build time environment variables dialog box, click the row of the environment variable to edit.

  2. Click Edit.

    The 3DS - Environment Variable dialog box opens.

  3. In Value, type the new value for the environment variable.

  4. Click OK.

    The environment variable value is modified.

Warning: If you find that a variable value you have changed is not taken into account, it may come from a variable name case sensitivity problem. To overcome this problem, create a new variable with the appropriate case. The value set to this variable will overwrite the value of the retrieved one.

Creating an Environment Variable

  1. In the 3DS - Create or overwrite build time environment variables dialog box, click New.

    The 3DS - Environment Variable dialog box opens.

  2. In Variable, type the name of the environment variable.

  3. In Value, type the value for the environment variable.

  4. Click OK.

    The environment variable value is created as a user environment variable and assigned the typed value.

Deleting an Environment Variable

  1. In the 3DS - Create or overwrite build time environment variables dialog box, click the row of the environment variable to delete.

  2. Click Delete.

    The environment variable is deleted.

Warning: You can delete user environment variables only.
