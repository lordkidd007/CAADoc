Abstract

This article discusses the implementation of the CATIContextualSubMenu interface that enables to insert commands in the contextual menu when selecting an instance of V5V6ExtCombinedCurve feature.
It is a copy of the implementation of CATIContextualSubMenu interface for  the CombinedCurve feature, described in the article "Adding a Contextual Sub-Menu"[4], rewritten to be  V5/V6 compatible. Some specific parts are coded inside #ifdef/#endif macros, we won't detail in this article any of the V6 specific code.

What You Will Learn With This Use Case

This use case is intended to show you how to insert commands in the contextual menu associated with the V5V6ExtCombinedCurve [1], by implementing  CATIContextualSubMenu  interface. A prerequisite is that this mechanical feature implements the CATIEdit interface [2].

[Top]

The CAAV5V6ExtMmrCombinedCurve Use Case

CAAV5V6ExtMmrCombinedCurve is a use case of the CAAV5V6MechanicalModeler.edu framework that illustrates Mechanical Modeler interactive capabilities.

[Top]

What Does CAAV5V6ExtMmrCombinedCurve Do

CAAV5V6ExtMmrCombinedCurve enables you to add commands in the sub-menu of the contextual menu associated with the V5V6ExtCombinedCurve feature. The picture below shows the sub-menu of the V5V6ExtCombinedCurve. There are two pictures because, the last command can be either Deactivate or Activate. You will see how the switch can be done.

Once the V5V6ExtCombinedCurve implements CATIEdit, its contextual menu contains at the end the "V5V6ExtCombinedCurve object" item. This item contains a sub-menu which by default contains only the Definition item. In implementing the CATIContextualSubMenu interface on the V5V6ExtCombinedCurve, you can insert other commands after the Definition item.

The commands that will will insert are Dassault Systèmes commands : Parent/Children, Reset Properties, Deactivate or Activate

The identifiers of the command header instances are:

  • "CATParentChildrenHdr" for Parent/Children command
  • "CATMmrPrtResetPropHdr" for Reset Properties command
  • "CATPrtActiveHdr" for Deactivate and Activate commands

These identifiers will be associated with the starters of the menu.

Where to Find the CAAV5V6ExtMmrCombinedCurve Code

The CAAV5V6ExtMmrCombinedCurve use case is made of one single class, the CAAEV5V6ExtMmrCombinedCurveContSubMenu class, located in the CAAV5V6ExtMmrCombinedCurveUI.m module of the CAAV5V6MechanicalModeler.edu framework:

Windows InstallRootDirectory\CAAV5V6MechanicalModeler.edu\CAAV5V6ExtMmrCombinedCurveUI.m\
Unix InstallRootDirectory/CAAV5V6MechanicalModeler.edu/CAAV5V6ExtMmrCombinedCurveUI.m/

where InstallRootDirectory is the directory where the CAA CD-ROM is installed.
