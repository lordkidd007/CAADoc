Abstract

This use case completes the Shape Design Workbench by adding a new toolbar containing a new command corresponding to V5V6ExtCombinedCurve Creation.
This use case is the same as the original CombinedCurve use case, "Creating an Add-in in the Shape Design Workbench " [3] rewritten for the new V5V6ExtCombinedCurve feature.

What You Will Learn With This Use Case

This use case creates a new toolbar in the Shape Design workbench to present a V5V6ExtCombinedCurve creation icon.

Thanks to this Use Case, the newborn V5V6ExtCombined Curve Mechanical Feature is now fully integrated with other Shape Design Features.

[Top]

The CAAV5V6ExtMmrCombinedCurve- AddIn Use Case

CAAV5V6ExtMmrCombinedCurve - AddIn is a use case of the CAAV5V6MechanicalModeler.edu framework that illustrates MechanicalModeler capabilities.

[Top]

What Does the CAAV5V6ExtMmrCombinedCurve - AddIn Use Case Do

This use case creates an add-in implementing the CATIShapeDesignWorkshopAddin interface. This interface is precisely defined to add toolbars or commands to the Shape Design workbench.

Another use case describes in detail how to create an add-in [1].

[Top]

How to Launch CAAV5V6ExtMmrCombinedCurve - AddIn

See the section entitled "How to Launch the V5V6ExtCombinedCurve Use Case" in the "Creating a New Geometrical Feature: The V5V6ExtCombinedCurve" use case for a detailed description of how this use case should be launched.

Where to Find the CAAV5V6ExtMmrCombinedCurve - AddIn Code

The CAAV5V6ExtMmrCombinedCurve - AddIn use case is made of a several classes located in the CAAV5V6ExtMmrCombinedCurveAddIn.m module of the CAAV5V6MechanicalModeler.edu framework:

Windows InstallRootDirectory\CAAV5V6MechanicalModeler.edu\CAAV5V6ExtMmrCombinedCurveAddIn.m\
Unix InstallRootDirectory/CAAV5V6MechanicalModeler.edu/CAAV5V6ExtMmrCombinedCurveAddIn.m/

where InstallRootDirectory is the directory where the CAA CD-ROM is installed.

[Top]

Step-by-Step

To create the add-in, you should first create a separate module directory to store the add-in code. This is CAAV5V6ExtMmrCombinedCurveAddIn.m for this use case.

You can refer to the original CombinedCurve Use Case [3] for the detail of the implementation, it is exactly the same implementation as for V5V6ExtCombinedCurve.
If you look into the source file CAAV5V6ExtMmrCombinedCurveAdn.cpp, you will see that some parts are in between #ifdef/#endif macros. Indeed, there are some evolutions in V6 that have not been backported to V5.

In Short

Thanks to the Add-in technology, this last use case of the Combined Curve Use Case makes it possible to create a V5V6ExtCombinedCurve by Clicking on an icon of the Shape Design Workbench. V5V6ExtCombinedCurves are now perfectly integrated in V5 Mechanical Design Applications.
