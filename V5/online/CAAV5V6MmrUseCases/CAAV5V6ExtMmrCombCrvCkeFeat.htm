Implementing the CATICkeFeature Interface

Declaring CATICkeFeature Implementation

CATImplementClass (CAAEV5V6ExtMmrCombCrvCkeFeature,
                   DataExtension ,
                   CATBaseUnknown,
                   V5V6ExtCombinedCurve);

TIEchain_CATICkeFeature(CAAEV5V6ExtMmrCombCrvCkeFeature);

The CAAEV5V6ExtMmrCombCrvCkeFeature class states that it implements the CATICkeFeature interface thanks to the TIE_CATICkeFeature macro. The CATImplementClass macro declares that the CAAEV5V6ExtMmrCombCrvCkeFeature class is a data extension, thanks to the DataExtension keyword, that extends V5V6ExtCombinedCurve. The third argument must always be set as CATBaseUnknown or CATNull for any kind of extension.

Do not forget to update your dictionary to declare V5V6ExtCombinedCurve has been extended to implement CATICkeFeature.

V5V6ExtCombinedCurve	   CATICkeFeature             libCAAV5V6ExtMmrCombinedCurve

In Short

This Use case has shown how to integrate a new Feature in Knowledgeware by implementing the CATICkeInterface interface. This interface enables a feature to be usable in Knowledgeware rules.
The implementation should derive from CATKweInstanceAdapter.
